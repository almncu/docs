### 什么是 FreeSWITCH™

FreeSWITCH™ 是一个开源通信平台，根据 [MPL（Mozilla Public License）1.1 版](http://www.mozilla.org/MPL/) 许可。FreeSWITCH™ 本质上是一个库，而不是一个应用程序。默认情况下，提供了一个会加载该库并作为软交换机运行的程序。FreeSWITCH™ 能够处理来自 PSTN（公共交换电话网，或者常规电话公司）和 IP 网络（例如互联网或私有 IP 网络）的语音、视频和文本通信。

FreeSWITCH™ 能够像软电话（模拟电话的软件）、PBX（私有分支交换机，通常是办公电话系统）、软交换（操作类似于传统基于硬件的电话公司交换机）或介于两者之间的任何角色。

在一个电话环境中，通常将多个组件合并到一个应用程序中。其中包括：

* 应用服务器（执行诸如语音邮件等功能的地方）
* 注册服务器（用户宣布自己在线和位置的地方）
* 路由引擎（决定将呼叫发送到何处的地方）

FreeSWITCH™能够作为这些功能的全部或部分来运行。此外，FreeSWITCH™还提供了在OSX、Linux、Windows（2003、XP、Vista、CE/Pocket PC）、BSD和Solaris上本地执行的机会。

### FreeSWITCH™的历史

FreeSWITCH™始于2005年的关于如何编写更好的切换平台的讨论。在设计过程中，我们特别注重了可移植性、稳定性和性能。2006年初，我们将源代码开放给更多开发人员，以帮助增加更多的功能支持，并提高错误检测和修复能力。2006年春季，我们扩展了项目的文档。2006年秋季，FreeSWITCH™已在实际生产环境中使用。我们持续更新文档，同时编写新代码、修复错误、增加功能，并不断进行性能调优。
