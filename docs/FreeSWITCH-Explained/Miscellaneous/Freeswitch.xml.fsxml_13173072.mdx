# Freeswitch.xml.fsxml 

 

位于{freeswitch\_installation}/log/目录下。此文件是FreeSWITCH预处理器将{freeswitch\_installation}/conf/\*文件编译在一起的方式。在解决XML和/或配置问题时，这个文件是一个很好的起点。

如果需要进行测试和故障排除，您可以用这个文件替换您的freeswitch.xml文件。
