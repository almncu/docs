# 语音

## 默认语音 - Callie

### 安装和配置

从 FS 构建目录:

```xml
make sounds-install
```

FreeSWITCH 的默认语音是由 [GM Voices](http://www.gmvoices.com) 制作的 Callie 语音。

## 语音 - Sibylle

Sibylle 是为 FreeSwitch 授权的创作共用语音。

### 

### 安装和配置

在 FreeSwitch 中使其工作需要两个步骤:

* `mod_say_fr` 必须使用在 <http://jira.freeswitch.org/browse/FSCORE-528> 找到的补丁进行更新
* 您必须从 <http://www.archive.org/details/FrenchAudioFilesForFreeswitch> 下载适合您的网络的声音文件（大多数人会选择 8000 Hz 的文件）

这些文件由 Sibylle Luperce & Philippe Leroy 为 [Neuronnexion.coop](http://Neuronnexion.coop) 创建，并在创作共用许可下发布。

如需获取更多录音，请联系 Philippe Leroy。
