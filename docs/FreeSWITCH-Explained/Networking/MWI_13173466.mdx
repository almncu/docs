# MWI

## 关于

与广为人知的观念相反，FreeSWITCH 是支持订阅和MWI的。

点击此处展开目录

* 1 [Presence](#presence)
* 2 [发送事件](#发送事件)
* 3 [如果你将FreeSWITCH作为语音邮件服务器，但是向OpenSIPS等进行注册](#如果你将FreeSWITCH作为语音邮件服务器但是向OpenSIPS等进行注册)

### Presence

要启用订阅功能，请在配置文件中添加以下行。

```xml
<param name="manage-presence" value="true"/>
```

### 发送事件

可以向用户发送通知的方式与发送其他事件的方式相同。(比如，js、socket等)

### 如果你将FreeSWITCH作为语音邮件服务器，但是向OpenSIPS等进行注册

你只需要配置OpenSIPS将SUBSCRIBE消息转发到FreeSWITCH，它就可以正常工作。
