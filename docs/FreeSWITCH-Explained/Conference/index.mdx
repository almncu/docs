# 会议

## 示例

#### 会议示例

* 页面：
[confcall JS](/confluence/display/FREESWITCH/confcall+JS) — 使用mod_spidermonkey和mod_spidermonkey_odbc的FreeSwitch早期JavaScript示例。
* 页面：
[Conference Add Call Example](/confluence/display/FREESWITCH/Conference+Add+Call+Example) — 允许主持人将与会者添加到已建立的会议中。
* 页面：
[Conference Announce Count Inline](/confluence/display/FREESWITCH/Conference+Announce+Count+Inline) — 此示例是一个非常简单和粗略的拨号计划和会议配置，可以让您听到有多少电话呼入会议中。
* 页面：
[Conference Javascript Dialer](/confluence/display/FREESWITCH/Conference+Javascript+Dialer) — JavaScript：拨打与会者，提示拨打数字以进入会议。
* 页面：
[JavaScript Conference IVR](/confluence/display/FREESWITCH/JavaScript+Conference+IVR) — 示例JS IVR，为公开可用的会议室添加安全性。
* 页面：
[Lua Conference Room Management example](/confluence/display/FREESWITCH/Lua+Conference+Room+Management+example) — 此Lua示例添加了简单的会议室管理。
* 页面：
[Mumble Conference With ALSA](/confluence/display/FREESWITCH/Mumble+Conference+With+ALSA)
* 页面：
[Outbound Conference Calls](/confluence/display/FREESWITCH/Outbound+Conference+Calls) — 拨打目标电话号码列表并将它们添加到会议中。
