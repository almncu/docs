## 1.6.19 (发布日期：2017年7月13日)

我们非常高兴地宣布1.6.19版本的发布！该版本的一些出色改进包括：在mod\_av中添加对wav格式的支持，mod\_callcenter增加对更大拨号字符串的支持，以及在mod\_conference中新增一个视频盲目功能，可以启动/停止接收视频。

- **压缩包：**<http://files.freeswitch.org/releases/freeswitch/freeswitch-1.6.19.tar.gz>
- **软件打包：**<https://freeswitch.org/confluence/display/FREESWITCH/Installation>

## 1.6.18 (发布日期：2017年6月13日)

我们很高兴地宣布发布1.6.18版本！这是一次常规发布，但是您可以找到一些新功能，包括改进mod_av对核心文件参数的处理，对vpx进行的一些质量调整，以及更新WebRTC代码以匹配最新规范，同时还更新了Windows版本的mod_lua。此外，还对许多其他改进和错误修复进行了更新，请务必阅读列表以获取最新信息。

* **Tarball：** <http://files.freeswitch.org/releases/freeswitch/freeswitch-1.6.18.tar.gz> 
* **安装说明：** <https://freeswitch.org/confluence/display/FREESWITCH/Installation>
