# 如何创建SignalWire个人访问令牌（PAT）

SignalWire个人访问令牌（PAT）是访问FreeSWITCH安装包所必需的。要获取访问令牌，请在[signalwire.com](https://id.signalwire.com/signup/account/new)上注册SignalWire帐户。

1. 登录SignalWire帐户。
2. 从页面右上角的菜单中选择"个人资料"，然后点击"个人访问令牌"。

![Create-PAT-2.png](/attachments/67240087/67240092.png)

3. 点击"生成新令牌"并为其命名。然后点击"生成令牌"。

![Create-PAT-3.png](/attachments/67240087/67240091.png)

4. 复制个人访问令牌以保存它。这将不再显示给您。准备好后，点击"已保存"。

![Create-PAT-4.png](/attachments/67240087/67240090.png)

5. 您的令牌现在会出现在列表中。点击"撤销"可以删除它。

![Create-PAT-5.png](/attachments/67240087/67240089.png)
