## 安装方法

可使用以下方法在macOS上安装FreeSWITCH™：

* [Homebrew](#homebrew-installation)，最快的入门方法。**可用于Apple Silicon版本。**
* [macFI安装工具](#macos-freeswitch-installer-macfi)，与手动安装相同，但更快且支持安装日志。
* [手动安装](#manual-installation)，如果您想要从源代码构建，并了解每个安装细节。

## Homebrew 安装

这是在 macOS 上安装 FreeSWITCH 最简单和最快的方法。要进行 Homebrew 安装，请使用[macOS Homebrew 安装](./macOS-Homebrew-Installation_72253508.mdx#about)说明。

**优点**

* 安装 FreeSWITCH™ 的最简单方法。
* 支持 Apple Silicon 和 Intel 版本。
* 支持最近的三个 macOS 版本。
* 占用最少的磁盘空间。

**缺点**

* 仅提供当前的生产版本。
* 仅作为运行时版本提供，因此从源代码构建可选模块很困难，但通常不需要。
* 本维基上的信息仅限于自 Homebrew 版本独立维护以来的测试结果。

## macOS FreeSWITCH™ 安装程序 (macFI)

macOS FreeSWITCH™安装程序与[手动安装](./macOS-Manual-Installation_13174557.mdx#about)执行相同功能，还增加了日志记录等功能。

macOS FreeSWITCH™安装程序自动化了所有手动步骤，提供了更简单更快速的安装。macFI的信息和下载位置可在[macOS macFI安装](./macOS-macFI-Installation_1966731.mdx#about)中找到。

**优点**

- 与下面的手动安装相同。
- 比手动安装更快，执行相同的步骤。
- 在安装之前，自动重命名现有的FreeSWITCH™目录以保存它们。
- 为每个安装步骤创建日志。

**缺点**

- 与下面的手动安装相同，除了更快。

## 手动安装

手动安装需要使用macOS终端输入命令，设置目录，安装先决条件和FreeSWITCH™。要执行手动安装，请使用[macOS手册安装](./macOS-Manual-Installation_13174557.mdx#about)说明。

**优点**

- 提供对当前版本、主版本（下一个版本）和更新的当前FreeSWITCH版本的访问。
- 从源代码构建易于添加可选模块。
- 为本Wiki执行了测试，因此信息很全面。

**缺点**

- 安装FreeSWITCH™的最长方法。
- 使用的磁盘空间比Homebrew版本更多。
- **目前尚未进行Apple Silicon测试**。
