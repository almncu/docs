# macOS

## 关于

这是在 macOS 上安装 FreeSWITCH™ 及其先决条件的完整指南，并建立 24x7 运行的操作方法。

**什么是 FreeSWITCH™？**
**[FreeSWITCH™](http://en.wikipedia.org/wiki/FreeSWITCH)** 是一个管理 **[VOIP](http://en.wikipedia.org/wiki/Voip)** 电话和与 **[ITSP（Internet Telephony Service Providers）](http://en.wikipedia.org/wiki/ITSP)** 的连接的 **[应用程序](https://en.wikipedia.org/wiki/Application%5Fsoftware)**。

**利用 FreeSWITCH™ 可以做什么？**
它的强大定制功能使我们的来电可以这样处理：正常的响铃被替换为：“嗨，Briana，请等一下，我们的电话正在响铃”，电话响起并且Briana听到音乐，25秒后：“Briana，请稍等，我们的iPhone正在响铃”，内部电话继续响铃和多个iPhone正在响铃，如果没有回答：“很抱歉，我们错过了您，Briana，请留言”，Briana的语音邮件将被录制并发送到多个iPhone上！

**这些页面是为谁准备的？**
本文档适用于想要在 macOS 上安装 FreeSWITCH™ 但对 Linux/Unix 命令和目录不熟悉的人。

**这份文档的开发和维护过程**
FreeSWITCH™ 在一台无头（headless）的 Mac Mini 上经过测试和 24x7 运行。在 2010 年替换 VOIP PBX 时，几乎找不到有关 FreeSWITCH™/macOS 的信息。因此，我通过搜索、尝试和错误、[FreeSWITCH™ 邮件列表](http://lists.freeswitch.org/)等途径汇总了我所学到的知识，并创建了这份文档。
