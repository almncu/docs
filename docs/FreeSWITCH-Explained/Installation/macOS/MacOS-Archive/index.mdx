macOS FreeSWITCH™ 废弃版本

FreeSWITCH在废弃版本上未经过测试。Apple和Homebrew停止支持并不再发布安全补丁。如果你按照[手动安装](../macOS-Installation/macOS-Manual-Installation_13174557.mdx#installation-directories)说明进行自定义修改，可以解决一些问题。

| ------------- | --------------------- | ------------ | -------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Big Sur 11.7  | 1.10.8 发布版         | 2022年10月14日 | 良好     | 维基作者不支持                                                                                                                                                                                                                                             |
| Catalina 10.15.7    | 1.10.7 发布版 | 2022年5月14日 | 构建有问题 | [#1367](https://github.com/signalwire/freeswitch/issues/1367), [#1474](https://github.com/signalwire/freeswitch/issues/1474), [#1497](https://github.com/signalwire/freeswitch/issues/1497), [#1500](https://github.com/signalwire/freeswitch/issues/1500) |
| Mojave 10.14.6    | 1.10.7 发布版 | 2021年11月25日 | 构建有问题 | 不受Apple和Homebrew支持                                                                                                                                                                                                                                    |
| High Sierra 10.13.6    | 1.10.5 发布版 | 2020年12月26日 | 先决条件有问题 | Homebrew Lua无法构建                                                                                                                                                                                                                                       |
