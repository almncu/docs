# 在OS X 10.9 Mavericks上安装

### 存档页面

存档已过时-不再维护

从2016年12月31日起，FreeSWITCH不再在OS X Mavericks上进行测试。苹果停止支持并不再发布安全补丁。Homebrew也不再正式支持Mavericks。

在本通知发布时，FreeSWITCH仍可以在Mavericks上安装和正常运行。如果需要的话，您可以尝试使用下面的命令下载Xcode，并按照手动安装说明进行安装。

### 下载Xcode

Xcode下载需要一些时间，下载完成后会放置在应用程序文件夹（/Applications目录）下，命令行工具位于/Applications/Xcode.app/Contents/Developer目录下。

Mavericks 10.9：从[Xcode 6.2 下载](http://developer.apple.com/devcenter/download.action?path=/Developer%5FTools/Xcode%5F6.2/Xcode%5F6.2.dmg)安装Xcode 6.2，然后启动Xcode。

## 关于

这是安装和运行FreeSWITCH™的[macOS安装](../macOS-Installation/index.mdx#about)指南的一部分。本页面描述了可能与OS X 10.9有关的组件的安装过程。

## OS X 开发工具

OS X 开发工具由两个部分组成，[Xcode](http://developer.apple.com/technologies/tools/xcode.html) 是用于开发和测试 OS X 图形用户界面应用程序的 OS X 软件开发工具包，而**命令行工具**提供了生成非图形用户界面应用程序（如 FreeSWITCH™ 及其前置条件）所需的 LLVM、Clang、Git 和其他工具。命令行工具已包含在 Xcode 中，将自动安装。Apple Developer 网站提供了**独立的命令行工具**，但_**不能使用，因为 Homebrew 不会安装没有 Xcode/CLT 版本的情况下。**_

### 安装 Xcode 6

Xcode 6.2 不再在[OS X App Store](https://itunes.apple.com/us/app/xcode/id497799835?mt=12&ign-mpt=uo%3D4)提供，因此必须从[OS X Developer 网站](http://developer.apple.com/devcenter/mac/index.action)下载，要求至少免费会员。Xcode 下载需要一段时间，并放置在应用程序文件夹（/Applications 目录）中，命令行工具则位于 /Applications/Xcode.app/Contents/Developer 中。_**安装完成后启动 Xcode，同意许可协议。**_
