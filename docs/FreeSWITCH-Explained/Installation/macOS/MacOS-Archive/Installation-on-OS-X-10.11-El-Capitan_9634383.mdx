### 下载 Xcode（从2017年12月起不再需要）

Xcode下载需要一些时间，并放置在Applications文件夹（/Applications目录）中，带有**C**ommand **L**ine **T**ools在/Applications/Xcode.app/Contents/Developer中。

此Xcode下载要求您使用Apple ID登录并同意软件协议：  
El Capitan 10.11：从[Xcode 7.3.1 下载](https://developer.apple.com/services-account/download?path=/Developer%5FTools/Xcode%5F7.3.1/Xcode%5F7.3.1.dmg)并安装Xcode 7.3.1，然后启动Xcode。注意：不能使用Xcode 8。请参见[FS-9521](https://freeswitch.org/jira/browse/FS-9521)。

对于OS X 10.11和10.12：绝对不要删除/usr/local目录！它很难重新创建，必须从备份中恢复。这是由于[系统完整性保护（又称无根保护）](https://en.wikipedia.org/wiki/System%5FIntegrity%5FProtection)的安全性。

## OS X开发者工具

OS X开发者工具由两部分组成，一部分是用于开发和测试OS X图形界面应用程序的[Xcode 7](http://developer.apple.com/technologies/tools/xcode.html) OS X软件开发套件，另一部分是**C**ommand **L**ine **T**ools，提供生成非图形界面应用程序（如FreeSWITCH™及其先决条件）所需的[LLVM](http://en.wikipedia.org/wiki/LLVM)、[Clang](http://en.wikipedia.org/wiki/Clang)、[Git](http://en.wikipedia.org/wiki/Git%5F%28software%29)和其他工具。命令行工具已经包含并且会自动与Xcode一起安装。Apple开发者网站提供了**独立CLT**，但_**不能使用，因为Homebrew在没有Xcode/CLT的版本时无法安装** **。_

### 安装Xcode

使用[OS X App Store 应用程序](https://itunes.apple.com/us/app/xcode/id497799835?mt=12&ign-mpt=uo%3D4)安装Xcode，或者从[OS X 开发者网站](http://developer.apple.com/devcenter/mac/index.action)下载（需要至少一个免费会员资格）。Xcode 的下载需要一些时间，并会被放置在 Applications 文件夹（/Applications 目录）中，而 CLT 将位于 /Applications/Xcode.app/Contents/Developer 中。**安装完成后，请启动 Xcode 并接受许可协议。**
