# Linux

## 关于

各种 Linux 发行版的安装指南。FreeSWITCH™ 核心开发团队已经统一使用 Debian，并推荐其用于所有部署。

熟悉其他发行版的用户可以帮助项目 [维护这些安装说明](/confluence/pages/createpage.action?spaceKey=FREESWITCH&title=Corrections&linkCreation=true&fromPageId=1048805)。

## **Sofia-sip 和 SpanDSP 注意事项！**

Sofia-sip 和 SpanDSP

自 v1.10.4 版本（2020 年 8 月 5 日）起，FreeSWITCH™ 树中已删除了 Sofia-sip 和 SpanDSP 的依赖。

您可以在这里找到 Sofia-sip 的源文件：<https://github.com/freeswitch/sofia-sip>

SpanDSP 的源文件在这里：<https://github.com/freeswitch/spandsp>

Sofia-sip 和 SpanDSP 的软件包可在我们的打包存储库中获得，适用于所有支持的平台。

您可能需要访问每个目录，运行 `./bootstrap.sh`，然后运行 `./configure`，最后执行 `make install`。

我们不再支持包装任何旧于 Debian 9 和旧于 CentOS 7 的内容，因为在旧平台上存在一些依赖问题。

## Debian

### Debian 11 Bullseye

从 FreeSWITCH 版本 1.10 发布起，参考平台是 Debian 11 “Bullseye"。请使用[Debian 页面](./Debian_67240088.mdx#about)上的说明。

## RHEL、Fedora、CentOS

源自 Fedora 的 Red Hat Enterprise Linux 和 CentOS 可以使用与 [CentOS 页面](./CentOS-7-and-RHEL-7_10289546.mdx#about)上显示的 Makefile 和 init 指令相同的指令。

## Ubuntu

自 2018 年以来，FreeSWITCH 没有提供预编译的 Ubuntu 二进制文件，您需要从源代码进行编译。
