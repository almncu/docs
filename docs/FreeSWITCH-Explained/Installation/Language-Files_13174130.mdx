# 语言文件

## 关于

FreeSWITCH附带了许多英文声音文件。其中一些是独立的声音提示，而其他声音是在mod\_conference、mod\_voicemail和mod\_say\_en中使用的。所有英文声音都列在FreeSWITCH源代码目录（src/docs/phrase/phrase\_en.xml）中。

## 语言

有许多声音文件包可供选择。并非所有的文件都由FreeSWITCH托管。我们正在努力整理所有这些文件的位置。如果您知道不在此列表上的声音文件，请联系Andrey Volk (andrey@freeswitch.com)。

以下是可用的**Debian**包列表：

* freeswitch-sounds-en-us-allison
* freeswitch-sounds-es-ar-mario
* freeswitch-sounds-fr-ca-june
* freeswitch-sounds-pt-br-karina
* freeswitch-sounds-ru-ru-elena
* freeswitch-sounds-ru-ru-kirill
* freeswitch-sounds-ru-ru-vika
* freeswitch-sounds-sv-se-jacob
* freeswitch-sounds-zh-cn-sinmei
* freeswitch-sounds-zh-hk-sinmei

**在Debian上安装：**

```bash
# 假设已经设置了FreeSWITCH存储库。详见https://freeswitch.org/confluence/display/FREESWITCH/Debian+9+Stretch
apt-get update
# 从包中安装时使用小写字母！
apt-get install -y freeswitch-sounds-en-us-allison
```

**Windows声音包：** <https://freeswitch.signalwire.com/windows/installer/x64/sounds/>

### 英语

英语声音可在[freeswitch.signalwire.com](https://freeswitch.signalwire.com)上获取。请参阅freeswitch-sounds-en-us-callie-xxx。此外，您可以使用简单的Linux/Unix命令安装声音和音乐文件：

  
```bash
make cd-sounds install
make cd-moh install
```

### 法语

法语声音文件可以从[freeswitch.signalwire.com](https://freeswitch.signalwire.com)下载。请参阅freeswitch-sounds-fr-ca-june-xxx。

### 德语

德语声音和其他信息可以从[Xpirio](http://freeswitch.xpirio.com/)下载。

### 希伯来语

请参阅[mod\_say\_he](../Modules/mod_say_he_13173303.mdx#about)以及当前录音的链接。

这是一个独立的模块，用于说出消息，因为在希伯来语中，男性、女性、复数等问题比在英语中要复杂得多。

### 俄语

俄语声音可以从[freeswitch.signalwire.com](https://freeswitch.signalwire.com)下载。请参阅freeswitch-sounds-ru-RU-elena-xxx。

### 西班牙语

一套西班牙语声音可以从[freeswitch.signalwire.com](https://freeswitch.signalwire.com)下载。请参阅freeswitch-sounds-es-mx-maria-44100.tar.gz。

### 波兰语

波兰语言声音文件可以从[freeswitch.signalwire.com](https://freeswitch.signalwire.com)获取。请参阅freeswitch-sounds-pl-pl-espeak-8000-0.1.0.tar.gz和freeswitch-sounds-pl-pl-espeak-16000-0.1.0.tar.gz。存档仅包含mod\_say\_pl使用的文件（在源代码树中，默认情况下不构建），并由使用espeak TTS的脚本创建。
