# Solaris


## 关于

Solaris 11.1 的支持是为了维护传统的电信用户。


维护者需求

我们需要您来维护这个FreeSWITCH的移植版本。请联系FreeSWITCH核心开发者之一，通过Freenode IRC频道#freeswitch或[freeswitch-users@lists.freeswitch.org](mailto:freeswitch-users@lists.freeswitch.org)邮件列表，以帮助他们保持此操作系统的代码更新。


## 从源代码构建

FreeSWITCH™团队在源树中准备了一个Makefile，用于下载所有依赖项并构建FreeSWITCH。它在存储库中可用，位于

<https://github.com/signalwire/freeswitch/blob/master/build/Makefile.solaris11>

只需将上面链接的内容复制到Makefile中，然后在相同的目录中运行 'make'。完成后，FreeSWITCH应位于`/usr/local/freeswitch`下。

如果您发现任何错误，请注册一个帐户来帮助更新文档。
