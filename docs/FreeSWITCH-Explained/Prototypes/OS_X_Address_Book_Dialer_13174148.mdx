# OS_X_Address_Book_Dialer

## About

这个应用程序是OS X中通讯录的插件，允许拨打通讯录条目中列出的号码。安装完成后，编辑fsdial.py以反映您的配置的正确信息，点击电话号码（更准确地说是号码的标签），将会弹出一个名为"FS Dial"的菜单。点击这个菜单，您的电话将会响起，当您接听时，您将会接通您点击的号码。

该应用程序是用AppleScript和Python编写的，并使用[Event Socket Library](../Client-and-Developer-Interfaces/Event-Socket-Library/index.mdx#-new-)。

### 安装

* 将所有文件添加到\~/Library/Address Book Scripts 中。
* 编辑fsdial.py以设置您的服务器信息和电话号码。
* 启动通讯录。现在，您在电话号码上的弹出菜单上应该有一个FS Dial条目。

这些脚本依赖于加载ESL库的机器上的脚本运行。这些应该在开发机器上构建并在客户机上正确的位置加载（Python库）。
