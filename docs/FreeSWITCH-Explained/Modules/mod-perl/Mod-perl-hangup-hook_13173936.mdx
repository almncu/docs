# Mod perl hangup hook 

## 关于

使用简单的 perl 来进行挂机钩子测试，$env 对象中有可用的会话数据。

单击此处展开目录

* 1 [如何操作](#如何操作)

## 如何操作

无法在挂机钩子中使用 $session，因此需要从 $env 中提取数据。

```xml
# 简单的 perl 挂机钩子测试             
#                                          
# 在您的 dp 中确保有这行代码：    
# <action application="set" data="session_in_hangup_hook=true"/>
#

our $env;  ## 特殊的“event”对象，包含所有的变量...
my $info = $env->serialize();

freeswitch::consoleLog("WARNING","好吧，开始测试...\n$info\n\n");

1;
```
