# mod_celt

## 关于

mod\_celt 可以使你在 FreeSWITCH 中使用高清 CELT 音频编解码器。

### 设置

在 src/modules.conf 和 conf/autoload\_configs/modules.conf.xml 中取消注释 mod\_celt。

### 测试

Brian 设置了一个测试，播放一个 48kHz 的音频文件：sip:886@taz.bkw.org:5080
