### 函数定义

```xml
playAndGetDigits(int min_digits,
                 int max_digits,
                 int max_tries,
                 int timeout,
                 char *terminators,
                 char *audio_files,

                 char *digits_regex)
```

### 示例

```xml
tempfile = "/path/to/song.mp3"
in_rslt = self.session.playAndGetDigits(1,
                                        1,
                                        1,
                                        1000,
                                        "*#",
                                        tempfile,
                                        "",
                                        "")
                    
actual_result = in_rslt[1]
if not actual_result:
    actual_result = int(actual_result)
```
