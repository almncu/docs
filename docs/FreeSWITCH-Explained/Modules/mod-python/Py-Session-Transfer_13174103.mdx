# Py 会话转接

## 关于

将呼入的呼叫转接到不同的本地分机。

点击此处展开目录

* 1 [概述](#概述)
* 2 [示例](#示例)

## 概述

```xml
transfer(extension, dialplan, context)
```

* extension - 目标分机，例如 "411"
* dialplan - 拨号计划名称，例如 "XML"，或者留空以使用默认值
* dialplan - 上下文名称，例如 "default"，或者留空以使用默认值

## 示例

```xml
self.session.transfer("411", "XML", "default")

self.session.transfer("411", "", "")
```
