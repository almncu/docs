# 示例 Recipewizard

## 关于

Python中的示例语音识别应用程序。在recipewizard.py脚本文件本身中有许多说明和注释。

## 配置

本教程假设您已正确安装[mod_python](../mod_python_1048940.mdx#eol)。它还依赖于已安装并正常工作的mod_openmrcp与自动语音识别服务器配合使用。

* 打开 /usr/src/freeswitch/scripts/python/recipewizard.py 文件，并根据您的系统自定义GRAMMAR\_ROOT变量和TTS设置。
* 从mainmenu()注释中的内容创建 $GRAMMAR\_ROOT/mainmenu.xml。
* 将脚本目录添加到 PYTHONPATH。
```shell
export PYTHONPATH=$PYTHONPATH:/usr/src/freeswitch/scripts/python
```
* 在待测试的拨号计划中添加一个扩展

**RecipeWizard 拨号计划扩展**

```xml
<extension name="2001">
    <condition field="destination_number" expression="^2001$">
      <action application="python" data="recipewizard"/>
    </condition>
</extension>
```

* 重新启动 FreeSWITCH。
* 拨打该扩展号码。
