# mod_say_he

## 关于

mod_say_he 基于 mod_say_en 开发，并进行了修改以支持希伯来语。这个模块还附带了录音文件。

**希伯来语的音频文件尚未完全完成。所需的声音文件即将上线。**

单击此处展开目录

* 1 [配置](#配置)
* 2 [使用示例](#使用示例)

## 配置

首先在 _modules.conf_ 文件中启用或添加以下内容:

**modules.conf**

```xml
mod_say_he
```

如果您要在现有安装中添加希伯来语，您需要手动将所需更改添加到当前配置文件中。将以下内容添加到安装目录下的 _conf/autoload_configs/modules.conf.xml_ 文件中的 say 部分。

**conf/autoload_configs/modules.conf.xml**

```xml
<load module="mod_say_he">
```

在安装目录下的 _conf/freswitch.xml_ 文件中也是同样的步骤。添加以下内容：

**conf/freswitch.xml**

```xml
<X-PRE-PROCESS cmd="include" data="lang/he/*.xml"/> 
```

编译 mod_say_he

如果您想手动安装和编译 mod_say_he，请执行以下操作：

**编译 mod_say_he**

```bash
make mod_say_he-install
/usr/local/freeswitch/bin/fs_cli
reload mod_say_he
```

## 使用示例

为了使其生效，您需要在拨号计划中设置默认语言和录音的路径。下面是一个示例：

```xml
<action application="set" data="default_language=he"/>
```

还有一个关于发音数字的示例：

```xml
<action application="say" data="he number pronounced masculine 8744697"/>
```
