内置于GUI的FreeSWITCH可以直接使用。

它基于mod\_xml\_rpc模块，该模块默认构建但未加载，因此您只需要加载它（在conf/autoload\_configs/modules.conf.xml文件中取消注释）。

```xml
load mod_xml_rpc
```

打开您的浏览器（已在Chrome和FireFox上测试），并转到localhost:8080/portal，您将能够看到它。

您还可以在autoload\_configs/xml\_rpc.conf.xml文件中启用websocket支持。

```xml
<param name="enable-websocket" value="true"/>
```

## 常见问题

### 它能在Windows上运行吗？

是的，有人报告说它可以工作。

### 我在右上角看到红色的"Socket Disconnected"，这是个问题吗？

这意味着websocket未启用或您的浏览器不支持websocket。它将自动切换到事件轮询，所以一切都正常工作，只是页面更新速度可能会略慢于websocket。
