# mod_esl

## 关于

mod_esl提供与远程FreeSwitch通信的功能。当前实现了远程API执行的功能。

点击这里展开目录

* 1 [描述](#描述)
* 2 [single_esl API](#single_esl-api)
* 3 [简化](#简化)

## 描述

在多服务器设置中，查询远程FreeSWITCH™以收集信息可能会非常有趣。该模块将包括各种能力来实现这一点。

## single_esl API

此API将API请求传递到远程服务器并返回结果。

```xml
single_esl [<user>]|<password> <host>[:<port>] <timeout> <remote api> <arguments>
```

## 简化

为了使日常使用更简单，您可以使用来自mod_abstraction的特性。这里是一个例子：

```xml
<api name="remote_user_password" syntax="<id>@<domain>" parse="(.*)" destination="single_esl" argument="|ClueCon 192.168.1.2 5000 user_data $1 var effective_caller_id_name"/>
```

现在，您只需要调用**remote_user_password(1000@default)**就可以从远程FreeSWITCH™获取信息。
