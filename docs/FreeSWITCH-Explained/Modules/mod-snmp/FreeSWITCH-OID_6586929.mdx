# FreeSWITCH OID

## 关于

FreeSWITCH OID是一个已注册的OID，用于SNMP、LDAP或其他协议来识别对象。

定义：

src/include/switch.h:#define FREESWITCH_PEN "27880"
src/include/switch.h:#define FREESWITCH_OID_PREFIX ".1.3.6.1.4.1." FREESWITCH_PEN

## 目前使用的值

.1.3.6.1.4.1.27880 FreeSWITCH

* .1至.1000：保留给核心模块
  * _.1.3.6.1.4.1.27880.1_: FreeSWITCH核心
  * _.1.3.6.1.4.1.27880.1000_: [mod\_snmp](../mod_snmp_6586926.mdx#about)
* .1001至.2000：模块
  * _.1.3.6.1.4.1.27880.1001_: [mod\_sofia](../mod_sofia_1048707.mdx#about)
  * _.1.3.6.1.4.1.27880.1002_: [mod\_skinny](../mod_skinny_3375354.mdx#docs)
    * _.1_: [Skinny LDAP模式](https://wiki.freeswitch.org/wiki/Mod%5Fskinny/skinny.schema "Mod skinny/skinny.schema")
