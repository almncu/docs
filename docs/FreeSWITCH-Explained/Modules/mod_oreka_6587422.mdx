# mod_oreka

## 关于

一个用于与Oreka一起进行媒体录制的模块 - 该模块向频道添加了一个媒体bug，将媒体发送到一个监听oreka服务器进行录制。

单击此处展开目录

* 1 [使用方法](#usage)
* 2 [配置](#configuration)

## 使用方法

```xml
 <action application="oreka_record"/>
```

此外，如果您将标题以"oreka_"为前缀，则它们也将被发送到oreka服务器。例如:

```xml
<action application="set" data="oreka_sip_h_X-customer=123"/>
<action application="set" data="oreka_sip_h_X-extension=1001"/>
```

## 配置

* sip-server-addr: Oreka/Orkaudio 录音服务器地址
* sip-server-port: 向录音服务器发送信令的端口
* mux-all-streams: （true / false） - 在发送到oreka之前混合音频流 - 或者，您也可以设置频道变量oreka\_mux\_streams=true
