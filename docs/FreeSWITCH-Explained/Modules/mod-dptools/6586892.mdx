## 使用方法

该应用程序可以从[XML拨号计划](../../Dialplan/XML-Dialplan-archive_6586601.mdx#about)中内联运行。

此外，您还可以在autoload_configs/switch.conf.xml中设置以下内容，&lt;param name="verbose-channel-events" value="yes"/>

fsctl verbose\_events on|off 也可以使用

## 示例

```xml
<condition field="destination_number" expression="^Monroy$">
  <action application="verbose_events" data="true"/>
  <action application="answer"/>
</condition>
```
