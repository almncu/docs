# mod_dptools: endless_playback 

 

## 关于

这个应用程序用于无限播放文件，且播放[不能被外部停止](to_be_replaced1)。

## 用法

```xml
<action application="endless_playback" data="filename"/>
```

## 注意事项

呼叫控制事件，尤其是挂断，将终止无限播放。DTMF检测不会中断无限播放。

[暂停](../mod_commands_1966741.mdx#command-2)API命令将暂停由此应用程序生成的播放。

[uuid_break](../mod_commands_1966741.mdx#uuid_break)API命令将重新开始从文件开头的播放，但不会终止播放。
