# mod_dptools: set_user 

 

## 关于

这个命令用来获取用户身份认证后定义的所有通道变量。

点击这里展开目录

* 1 [使用方法](#usage)
* 2 [示例](#example)
* 3 [参考资料](#see-also)

## 使用方法

```xml
set_user <user@domain> [prefix]
```

这个应用程序可以从XML计划中[内嵌执行](./6586667.mdx#about)。

## 示例

```xml
<action application="set_user" data="5551212@${domain}"/> 
```

或者，如果要给所有变量的名称加上前缀"dest\_"，可以这样做：

```xml
<action application="set_user" data="5551212@${domain} dest_"/>
```
