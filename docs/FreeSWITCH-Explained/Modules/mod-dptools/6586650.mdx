# mod_dptools: system

## 关于

执行操作系统命令并等待结果（阻塞）。

单击此处展开目录

* 1 [使用方法](#usage)
* 2 [示例](#example)  
   * 2.1 [相关](#see-also)

## 使用方法

```bash
system <command>
```

由于此命令会阻塞拨号计划的进一步执行，因此不要使用此方法执行长时间运行的命令。另一种选择是使用“bgsystem”拨号计划应用程序在后台运行命令。

## 示例

```xml
<action application="system" data="myapp.sh"/>
```

将信息作为通道变量返回！

```xml
<action application="set" data="OUT=${system foo.sh}"/>
```

如果您需要删除末尾的换行符，可以使用以下方法：

```xml
<action application="set" data="OUT=${regex(${system foo.sh}|(.+)|%1)}"/>
```
