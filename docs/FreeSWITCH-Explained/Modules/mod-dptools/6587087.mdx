# mod_dptools: sleep

## 关于

暂停通道一定毫秒数，消耗该时间段内的音频。
调用 sleep 也将消耗操作系统输入队列中的任何未处理的 RTP 数据，这在音频积压的情况下非常有用。
要消耗 DTMF 信号，请使用 sleep_eat_digits 变量。

## 用法

```xml
<action application="sleep" data=<毫秒数>/>
```

## 变量

### sleep_eat_digits

当设置为 true 时，sleep 应用将会消耗 DTMF 信号，例如，防止呼叫者退出交互式语音应答 (IVR)。默认行为是不消耗 DTMF 信号。

这是默认行为的更改，因为之前的 sleep 应用始终会消耗 DTMF 信号。为了保持之前的行为，请确保将 sleep_eat_digits 设置为 true。

```xml
<action application="set" data="sleep_eat_digits=true"/>
```

## 示例

```xml
<action application="sleep" data="5000"/>
```
