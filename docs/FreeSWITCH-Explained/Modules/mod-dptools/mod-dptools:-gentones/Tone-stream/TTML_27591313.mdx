# TTML 

[定时文本标记语言](https://en.wikipedia.org/wiki/Timed%5FText%5FMarkup%5FLanguage)“被设计用于撰写、转码或交换目前主要用于[字幕和字幕](https://en.wikipedia.org/wiki/Subtitle%5F%28captioning%29 "Subtitle (captioning)")功能的定时文本信息”，但显然FreeSWITCH使用它们来组成一系列音调。

`fur_elise.ttml`随附在vanilla FreeSWITCH安装中，它是一系列FreeSWITCH音调的序列:
