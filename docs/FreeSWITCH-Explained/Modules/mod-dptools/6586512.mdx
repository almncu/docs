# mod_dptools：聊天

## 关于

将文本消息发送到即时通讯客户端。

## 使用方法

&lt;action application="chat" data="protocol | from_jid | to_jid | message | [&lt;content-type>]"/>

字段之间用管道符 | 分隔。管道符和数据字段之间允许有空格。

protocol — sip、verto、jingle中的一个；其他？在这里添加。

from_jid — 发送者的URI

to_jid — 接收消息的URI

message — 要发送的消息内容

content-type — 默认为text

### CLI

以下是一个示例，发送到SIP和Verto终端：

**Perl**

```perl
freeswitch@default>chat sip|Server@FS.local|1000@domain.local|Hello chat via SIP!
freeswitch@default>chat verto|Server@FS.local|1000@domain.local|Hello chat via Verto!
```

### Perl

**Perl**

```perl
$api = new freeswitch::API();
$foo = $api->execute("chat", "jingle|from_jid|to_jid|test Jingle message"); 
```
