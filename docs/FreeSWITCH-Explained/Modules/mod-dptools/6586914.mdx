## 描述

清除通道变量。

## 用法

```xml
<application="unset" data="variable_name">
```

此应用程序可以从 [XML 拨号计划](../../Dialplan/XML-Dialplan-archive_6586601.mdx#about) 中的内联运行。

## 示例

```xml
<action application="unset" data="foo"/>
```
