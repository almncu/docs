# mod_dptools: 发送显示

## 关于

发送包含sipfrag的信息数据包。如果电话支持，将在显示屏上显示消息。

单击此处展开目录

* 1 [用法](#-freeswitch--mod_dptools-send-display-)
* 2 [示例](#-freeswitch--mod_dptools-send-display-)
* 3 [兼容性列表](#compatibility-list)  
   * 3.1 [Polycom](#-freeswitch--mod_dptools-send-display-)  
   * 3.2 [Snom 3xx](#-freeswitch--mod_dptools-send-display-)  
   * 3.3 [Nokia](#-freeswitch--mod_dptools-send-display-)

## 用法

```xml
send_display <message>
```

## 示例

```xml
<action application="send_display" data="Hello world"/>
```

## 兼容性列表

### Polycom

在运行sip 3.0.4版本的soundpoint 670上经过测试。在Polycom上运行正常。显示文本出现在“已拨电话”中，似乎用于向呼叫方发送被叫方的姓名。  
在应答之前只能向显示屏发送一条消息。接听电话后，可以随意更新显示消息。通话记录将显示最后发送的消息。

### Snom 3xx

运行正常

### Nokia

接收到消息后电话会挂断。
