## 0. 关于

使用当前使用的终端配置的方法发送会话中的DTMF数字。使用字符w表示延迟0.5秒，字符W表示延迟1秒。

## 1. 使用方法

```xml
send_dtmf <dtmf digits>[@<tone_duration>]
```

## 2. 示例

```xml
<action application="send_dtmf" data="0123456789ABCD*#@100"/>
```

传输数字'0123456789ABCD\*#'，每个数字的持续时间为100毫秒。如果未指定持续时间，则默认使用DTMF长度为2000毫秒。

## 3. 选项

对于SIP呼叫，数字可以通过inband、RFC2833或INFO消息进行传输。有关在mod_sofia中配置DTMF传输方法的配置，请参阅[Sofia\_Configuration\_Files](https://wiki.freeswitch.org/wiki/Sofia%5FConfiguration%5FFiles)。
