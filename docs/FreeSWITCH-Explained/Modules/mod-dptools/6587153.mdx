# mod_dptools: 设置僵尸执行

## 关于

在当前通道上设置僵尸执行标志。如果通道被挂断，会话将继续执行命令。适用于额外的呼叫后处理（例如，在仍需要访问通道变量的情况下）。

单击此处展开目录

* 1 [用法](#-freeswitch--mod_dptools-set-zombie-exec-)
* 2 [示例](#-freeswitch--mod_dptools-set-zombie-exec-)

## 用法

```xml
set_zombie_exec 
```

## 示例

```xml
<action application="set_zombie_exec"/> 
```

在Lua脚本中：

```xml
session:execute("set_zombie_exec")
```
