## 0\. 关于

使用定义的文本到语音引擎将字符串或文本文件读出来给通道听。

TTS引擎

访问[TTS](../../Configuration/TTS_9634273.mdx#about)页面，了解可用的文本到语音引擎。

## 1\. 使用方法

```xml
<action application="speak" data="<engine>|<voice>|<text>|[timer_name]"/>
```
