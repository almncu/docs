## 0\. 关于

用于清除通道上接收到的DTMF信号。当呼叫者在一个对话中输入了多余的按键，而您希望在将其发送到另一个对话之前清除它们时，此功能非常实用。（参见 [mod\_dptools: play\_and\_get\_digits](./6586933.mdx#about) ）。

## 1\. 使用方法

```xml
<action application="flush_dtmf"/>
```

如果您在对话中遇到神秘的DTMF信号输入问题，请尝试在对话之前添加 ```flush_dtmf``` ，看看是否会有所帮助。
