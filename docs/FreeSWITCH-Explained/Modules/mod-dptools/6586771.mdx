# mod_dptools: digit_action_set_realm

## 0. 关于

与 [mod_dptools: bind_digit_action](./6586414.mdx#0-about) 一起使用，用于设置将应用绑定的数字动作的领域。

详细说明和示例请参见该处。

## 1. 使用方法

```xml
<action application="digit_action_set_realm" data="my_realm[,target]"/>
```

| 参数      | 描述                                 |
| --------- | ------------------------------------ |
| target    | 设置领域的腿。可能的取值：selfpeer |
