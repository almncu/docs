＃mod_dptools：multiset



## 关于

为呼叫该应用程序的通道设置多个通道变量。

### 用法

```bash
multiset [^^<delim>]<varname>=<value> <var2>=<val2>"
```

可以从XML拨号计划中作为内联操作运行此应用程序。

默认分隔符是空格。如果您想使用不同的字符，您可以以^^&lt;delim>开头，其中&lt;delim>是您想要用作分隔符的任何字符。

### 示例

```xml
<action application="multiset" data="effective_caller_id_name=FreeSwitch effective_caller_id_number=12345678"/>
<action application="multiset" data="^^:effective_caller_id_name=FreeSwitch:effective_caller_id_number=12345678"/>
```

### 警告

与[set](./6586661.mdx#about)一样，变量直到执行时才会被设置。如果您需要在条件中立即使用它，请在XML拨号计划中作为内联操作运行它。

或者，在[transfer](./6586616.mdx#about)或[execute_extension](./6586595.mdx#about)之后该变量将可用。
