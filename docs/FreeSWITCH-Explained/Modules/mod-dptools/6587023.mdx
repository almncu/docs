# mod_dptools: 信息

## 关于

将通道信息转储到控制台。

单击此处展开目录

* 1 [用法](#usage)
* 2 [另请参阅](#see-also)

## 用法

```xml
info 
```

```xml
<action application="info"/> 
```

在B连接上执行：

```xml
 <action application="set" data="bridge_pre_execute_bleg_app=info"/>
```

使用Lua脚本以不同的日志级别示例：

```xml
session1:execute("info","notice");
```
