## 说明

pre\_answer相当于带有SDP的SIP状态码183。（类似于Asterisk中的cmd来电中转）

pre\_answer建立媒体（早期媒体），但不会真正接听。例如，您可以使用它在断开通话之前向呼叫方发送一个带内错误消息（pre\_answer、playback、使用xxxx的拒绝原因码）。

话虽如此，大多数需要媒体（如音频播放）的应用程序会自动进行pre\_answer。如果您遇到设备设置媒体路径需要很长时间的问题，则可以在拨号计划中进行pre\_answer，并在此之后进行休眠。

## 使用方法

```xml
<action application="pre_answer"/>
```
