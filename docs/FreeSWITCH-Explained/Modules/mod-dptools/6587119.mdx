# mod_dptools: set_profile_var

## 关于

在CHANNEL事件中添加一个头部。

点击这里展开目录表

* 1 [用法](#-freeswitch--mod_dptools-set_profile_var-)
* 2 [示例](#-freeswitch--mod_dptools-set_profile_var-)

## 用法

```xml
set_profile_var <profile_variable>=<value>
```

可以从XML拨号计划中[内联运行](./6586667.mdx#about)此应用程序。

## 示例

```xml
<action application="set_profile_var" data="accountcode=${accountcode}"/> 
```

或者

```xml
<action application="set_profile_var" data="accountcode=14"/> 
```
