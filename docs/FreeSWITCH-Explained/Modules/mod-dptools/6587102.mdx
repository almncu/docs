# mod_dptools: 设置音频级别

## 关于

增加或减小通道的传入（读取）音频流或传出（写入）音频流的音量级别。

这样你就可以调整通话的音量，使其更大声或更轻柔。

## 使用方法

```xml
set_audio_level <方向> <级别>
```

## 示例

```xml
<action application="set_audio_level" data="read 1"/>
<action application="set_audio_level" data="write -1"/>
```

第一个示例增加了传入音频流的音量级别，而第二个示例降低了传出音频流的音量级别。

read 和 write 可以取从 4 到 -4 的整数值。

## 选项

这个拨号计划应用的音量级别对应于会议应用中的音量级别，即 -4 到 4。

## 注意事项

在数字音频中，修改服务器级别的音频流并不是最优解。通常最好在模拟点之一（如电话、耳机或模拟TDM卡）上调整音量。这个拨号计划应用可用于帮助那些无法或不愿调整音频的设备出现故障的用户。
