## 0\. 关于

在当前通道上生成DTMF信号。

  
## 1\. 使用方法

```xml
<action application="start_dtmf_generate"/>
```

## 2\. 示例

例如，在呼叫桥接之前将此代码放置在拨号计划中，允许一个设置为rfc2833或info的电话将DTMF音调传输到接收方（IVR）或自动接待员。从而将出局路由从info或rfc2833更改为inband。

TODO [RFC 2833](https://tools.ietf.org/html/rfc2833)已被[RFC 4733](https://tools.ietf.org/html/rfc4733)废弃。

## 3\. 注意事项

当以下方法使用时，此选项将无效：

```xml
<action application="export" data="nolocal:execute_on_media=start_dtmf_generate"/>
<action application="set" data="bridge_early_media=true"/>
<action application="bridge" data="sofia/..."/>
```
