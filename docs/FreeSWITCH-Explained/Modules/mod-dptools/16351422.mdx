# mod_dptools: loop_playback 

 

## 关于

循环播放文件到通道，但只限制播放次数。**但无法设置播放间隔时间**，类似于 [playback](./6586776.mdx#0-about)，[playback\_sleep\_val](../../Dialplan/Channel-Variables_16352493.mdx#channel-variable-scope-example)，[file\_string](./6586646.mdx#0-about)。

## 使用方法

```bash
loop_playback [+循环次数] <路径>
```

## 示例

同时播放文件两次。

```xml
<action application="loop_playback" data="+2 path.wav"/>
```
