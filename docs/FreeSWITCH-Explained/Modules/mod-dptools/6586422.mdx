# mod_dptools: clear_digit_action 

## 关于

清除所有数字绑定。

点击此处展开目录

* 1 [使用方法](#-freeswitch--mod_dptools-clear_digit_action-)
* 2 [示例](#-freeswitch--mod_dptools-clear_digit_action-)
* 3 [参考资料](#-freeswitch--mod_dptools-clear_digit_action-)

销毁以[mod\_dptools: bind\_digit\_action](./6586414.mdx#0-about)创建的绑定数字区域或所有区域。

## 使用方法

clear_digit_action &lt;realm>|all[,target]

如果未指定参数，则与指定`all`相同。

参数`target`可以取值"`peer`"或"`both`"

## 示例

```xml
<action application="clear_digit_action" data="realm"/>
```

`realm`是之前使用bind\_digit\_action指定的数字区域的名称或id。

```xml
<action application="clear_digit_action" data="all"/>
```

清除之前使用bind\_digit\_action指定的所有区域。
