## 关于

生成[TGML](./mod-dptools:-gentones/TGML_6586573.mdx#lua)音调。

点击这里展开目录

**未能渲染`toc`宏**

null

### 使用方法

```xml
gentones <TGML>[|循环次数]
```

### 示例

```xml
<pre>
<!-- 生成持续500ms、频率为800Hz的蜂鸣声 -->
<action application="gentones" data="%(500,0,800)"/>
<!-- 生成DTMF信号 -->
<action application="gentones" data="0800500005"/>
<!-- 持续生成法国铃声（循环次数：-1） -->
<action application="gentones" data="%(1500,3500,350,425)|-1"/>
</pre>
```
