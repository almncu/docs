# 沉默流

## 沉默流

沉默流是一种文件格式，可在任何需要文件的地方使用。沉默流由[mod_tone_stream](./Tone_stream_6586599.mdx#about)实现。

## 用法

silence_stream://[&lt;duration>[,divisor]]

持续时间以毫秒为单位指定

* 如果持续时间为-1，将生成无限的沉默。
* 如果持续时间大于0，则生成持续时间为ms的沉默。

除数是舒适噪音生成器的除数。

* 如果除数大于0，则生成舒适噪音。
* 如果除数为0，则生成完全的沉默。

### 示例

播放1000ms的沉默

&lt;action application="playback" data="silence_stream://1000"/>

播放2000ms的舒适噪音

&lt;action application="playback" data="silence_stream://2000,1400"/>

播放无限的舒适噪音

&lt;action application="playback" data="silence_stream://-1,1400"/>

播放无限的沉默

&lt;action application="playback" data="silence_stream://-1"/>
