# mod_dptools: soft_hold

## 关于

在FreeSWITCH™上将桥接的通道放置在保持状态。

仅限于两个通道建立桥接时才有效。

点击此处展开目录

* 1 [使用方法](#-freeswitch--mod_dptools-soft_hold-)

### 使用方法

```xml
<action application="soft_hold" data="<unhold_key> [moh_a] [moh_b]"/>
```

* unhold_key：解除保持的DTMF按键
* moh_a：在语音通道A上播放的音乐
* moh_b：在语音通道B上播放的音乐

如果没有设置音乐，将播放默认的音乐。
