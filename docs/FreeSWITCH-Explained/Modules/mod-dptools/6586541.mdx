# mod_dptools: cng_plc

## 关于

应用丢包隐藏和舒适噪声生成。cng_plc 对丢失的数据包执行丢包隐藏，且仅在 originate 过程中对外呼出进行执行，类似于 answer 过程等。舒适噪声是人工产生的噪音，用于向来电者确认通话未断开并正在处理中。

## 用法

```xml
<action application="cng_plc"/>
```

## 问题场景

在经历间歇性丢包时，可能是由于数据包晚到、丢失并被静音替代。

你可以尝试两件事情：

1. 在桥接前和 B-leg 的 execute_on_originate 中运行应用程序 cng_plc。  
```xml  
<action application="cng_plc"/>  
```
2. 设置小值的抖动缓冲区，将 jitterbuffer_msec 设置为 60。  
```xml  
<action application="set" data="jitterbuffer_msec=60"/>  
```
