# mod_dptools: mkdir 

## 关于

创建一个目录。如果父目录不存在，则默认也会创建父目录。

Freeswitch在给定路径上必须具有写权限 - 如果由于权限问题无法创建路径，则不会向日志发送错误消息。

### 用法

```bash
mkdir <路径>
```

### 示例

```xml
<action application="mkdir" data="/path/to/new/directory"/>
<action application="mkdir" data="${recordings_dir}/archive/${strftime(%Y)}/${strftime(%b)}/${strftime(%d)}/"/>
<action application="mkdir" data="/usr/local/freeswitch/sounds/custom/blabla"/>
```

### 权限

默认情况下，新建的文件夹的权限为u=rwx g=r-x a=---。如果要更改这个行为，您需要更改switch_types.h中的常量[SWITCH_DEFAULT_DIR_PERMS](https://freeswitch.org/stash/projects/FS/repos/freeswitch/browse/src/include/switch%5Ftypes.h#117)并重新编译。

### 替代方法

使用“system”工具创建目录：

```xml
<action application="system" data="/bin/mkdir -p /path/to/new/directory"/>
```
