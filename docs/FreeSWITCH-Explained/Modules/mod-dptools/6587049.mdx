## 0\. 关于

取消定时未来的广播/转移任务。

## 1\. 语法

```xml
<action application="sched_cancel" data="<task_id>" />
```

| 参数       | 描述                                                                                                         | 示例                                              |
| ---------- | ---------------------------------------------------------------------------------------------------------- | --------------------------------------------------- |
| &lt;task\_id> | 被取消的操作的任务ID（[sched\_transfer](./6587055.mdx#0-about) 或 [sched\_broadcast](./6587042.mdx#0-about)）。 | 123${last\_sched\_id} (请参考下方的 **2\. 示例**。) |

## 2\. 示例

```xml
<action application="sched_cancel" data="123"/> 
```

```xml
<action application="sched_cancel" data="${last_sched_id}"/> 
```

当一个预定的广播或转移任务被执行时，会将会话变量`**last_sched_id**`设置为广播/转移任务的任务ID。
