# mod_dptools: strepoch

## 关于

返回自Unix纪元（1970年1月1日午夜的UTC时间）以来的秒数。

## 示例

以下扩展将只在通过纪元之前的日期/时间之前调用时才开始播放保持音乐。

```xml
<extension name="moh">
  <condition field="destination_number" expression="^1234$"/>
  <condition field="${cond(${strepoch()} < 1273755900 ? true : false)}" expression="^true$">
    <action application="answer"/>
    <action application="playback" data="$${hold_music}"/>
    <anti-action application="log" data="NOTICE The string quartet has gone home."/>
  </condition>
</extension>
```

## 从CLI中使用

```xml
strepoch
```
