# mod_hash

## 关于

mod_hash 实现了一个操作哈希表的 API 和应用接口。它还提供了一个限制的后端。

## 概念

Realm 和 key 是任意字符串。可以将 realm 视为 key 的容器。

insert_ifempty 和 delete_ifmatch 在哈希表上执行线程安全的测试和设置操作。

## 拨号计划应用

将一个值插入哈希表：

&lt;action application="hash" data="insert/realm/key/value"/>

仅当键关联的值不存在时，将一个值插入哈希表：

&lt;action application="hash" data="insert_ifempty/realm/key/value"/>

从哈希表中删除一个条目：

&lt;action application="hash" data="delete/realm/key"/>

仅当键关联的值与给定的值相同时，从哈希表中删除一个条目：

&lt;action application="hash" data="delete_ifmatch/realm/key/value"/>

从哈希表中检索一个值：

&lt;action application="set" data="var=${hash(select/realm/key)}"/>

## API

hash insert/realm/key/value
hash insert_ifempty/realm/key/value
hash delete/realm/key
hash delete_ifmatch/realm/key/value
hash select/realm/key

## 限制

限制的哈希后端提供了一个非持久化数据存储，用于存储限制数据。这些数据特定于一个特定的 FreeSWITCH 实例。

注意：该后端支持基于速率的限制。
