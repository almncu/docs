## 关于

记录一个FSV文件。

点击这里展开目录

* 1 [用法](#usage)

### 用法

```xml
<action application="record_fsv" data="file.fsv"/>
```

在开始录制视频之前，您需要编辑vars.xml文件以包含视频编解码器

```xml
< X-PRE-PROCESS cmd="set" data="global_codec_prefs=G7221@32000h,G7221@16000h,G722,PCMU,PCMA,GSM,H263,H263-1998" / >
```
