# 播放 FSV

## 关于

播放 FSV 文件。

单击此处展开目录

* 1 [使用方法](#usage)

### 使用方法

```xml
<action application="play_fsv" data="file.fsv"/>
```

在实际开始播放视频之前，您需要对 vars.xml 进行配置，以配置视频编解码器，如 H263、H261 等。

```xml
<X-PRE-PROCESS cmd="set" data="global_codec_prefs=G7221@32000h,G7221@16000h,G722,PCMU,PCMA,GSM,H261,H263,H263-1998"/>
```
