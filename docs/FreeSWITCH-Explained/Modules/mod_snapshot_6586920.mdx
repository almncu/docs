# mod_snapshot

## 关于

mod_snapshot 允许录制一个滑动窗口的音频并将快照保存到磁盘上。

加载 mod_snapshot 后，fs_cli 命令 "uuid_snapshot" 就可用了。

```xml
freeswitch@default> uuid_snapshot
USAGE: <uuid> snap|start [<sec> read|write <base>]
read | write <base>
```

当在 "start" 命令中使用时，可以指定 "/usr/local/freeswitch/sounds/" 的子目录。

在以下命令执行之后，将会将一个 WAV 文件写入到 /usr/local/freeswitch/sounds 目录下。文件格式为 `mod_snapshot_<date>_.wav`。如果命令后没有输入时间，默认为 5 秒。

```xml
    uuid_snapshot dc763550-a459-11e1-a5d4-13b40a2cd4b8 start <sec>
```

写入文件。

```xml
    uuid_snapshot dc763550-a459-11e1-a5d4-13b40a2cd4b8 snap
```
