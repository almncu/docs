# mod_siren

## 关于

mod_siren允许您在FreeSWITCH中使用Polycom Siren™音频编解码器。

## 设置

在src/modules.conf和_conf/autoload_configs/modules.conf.xml文件中取消注释mod_siren。

## 测试

您需要支持这些编解码器的设备来进行测试。
