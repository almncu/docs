# Function db

## 关于

允许在运行时将事物插入和删除到数据库中。

单击此处展开目录

**错误渲染宏'toc'**

null

### 概要

```xml
[insert|select|delete]/<realm>/<key>[/<val>]


```

## 示例

### Select

```xml
<action application="playback" data="${db(select/music/${caller_id_number})}"/>
```

### Insert

```xml
<action application="db" data="insert/spymap/${caller_id_number}/${uuid}"/>
```
