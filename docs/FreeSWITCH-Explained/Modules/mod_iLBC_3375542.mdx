# mod_iLBC 

## 关于

iLBC 是一种良好的低比特率编解码器，与[G.729](https://wiki.freeswitch.org/wiki/Mod%5Fg729 "Mod g729") 相比具有类似的压缩效果，但在存在部分丢包时，其MOS（Mean Opinion Score）较高。

iLBC 是互联网低比特率编解码器的缩写。

### 动态载荷编号

iLBC 在 SIP SDP 中采用动态载荷编号。FreeSWITCH **支持**动态载荷。然而，一些客户端无法正确处理。在测试 iLBC 时请注意此问题。

请参阅：[动态载荷](../Codecs-and-Media/Dynamic-Payloads_3375552.mdx#about)
