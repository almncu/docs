# Markdown语法初探

Markdown是最流行的轻量级标记语言之一，被广泛使用于文档编写、网页编辑和博客发布等场景。它的语法简洁易懂，适合用于快速撰写文本内容。

## 标题

Markdown使用`#`符号表示标题，可根据需要添加1到6个`#`，表示不同级别的标题。例如：

```
# 一级标题
## 二级标题
### 三级标题
```

## 文本样式和链接

Markdown支持加粗、斜体、删除线等文本样式的标记。可以使用`**`将文本包围，表示加粗，使用`*`将文本包围，表示斜体。例如：

```
这是一个 **加粗** 的文本，这是一个 *斜体* 的文本。
```

在Markdown中，可以使用`[]`和`()`定义链接，方便快速添加超链接。例如：

```
这是一个[链接](http://example.com)，点击可以跳转。
```

## 列表和引用

Markdown支持有序列表和无序列表的标记。有序列表使用数字和`.`表示，无序列表使用`-`或`*`表示。例如：

```
1. 有序列表项一
2. 有序列表项二
- 无序列表项一
- 无序列表项二
```

引用文本使用`>`符号表示。例如：

```
> 这是一段引用的文本。
```

## 代码块

如果要插入代码块，可以使用三个反引号（```）将代码包围起来。例如：

```
​```
def hello_world():
    print("Hello, World!")

hello_world()
​```
```

## 图片

使用`![描述文本](图片链接)`的格式来插入图片。例如：

```
![示例图片](http://example.com/image.jpg)
```

以上是Markdown的一些基本语法。通过掌握这些语法规则，你可以快速编写美观、易读的文档。
