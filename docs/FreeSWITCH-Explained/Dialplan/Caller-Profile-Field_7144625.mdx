# Caller Profile Field 

## 关于

调用者配置字段可以作为呼叫计划中的条件使用。

单击此处展开目录

* 1 [字段](#字段)
* 2 [参考文献](#参考文献)

## 字段

以下字段可用：

```xml
username             呼叫者的用户名
dialplan             正在使用的呼叫计划模块的名称
caller_id_name       呼叫者ID名称
caller_id_number     呼叫者ID号码
network_addr         网络地址
ani                  ANI信息
aniii                ANI II信息
rdnis                RDNIS
destination_number   目标号码 
```
