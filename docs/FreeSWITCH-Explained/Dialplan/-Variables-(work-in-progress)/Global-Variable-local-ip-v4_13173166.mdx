### local_ip_v4

此全局变量由FreeSWITCH自动设置。您可以在[vars.xml](../../Configuration/Configuring-FreeSWITCH/vars.xml_7144079.mdx#about)中覆盖此全局变量。

**用法：**

```xml
<!-- 将local_ip_v4设置为硬编码IP -->
<X-PRE-PROCESS cmd="set" data="local_ip_v4=192.168.0.68"/>

<!-- 将local_ip_v4设置为eth1地址 -->
<X-PRE-PROCESS cmd="exec-set" data="local_ip_v4=ip addr show eth1 | awk '/inet /{print $2}' | head -n 1 | cut -d '/' -f 1"/>
```
