## fifo_member_wait

如果设置为'wait'，则当呼叫方挂断时，消费方的呼叫将不会挂断\[默认\]。  
如果设置为'nowait'，则当呼叫方挂断时，消费方的呼叫将挂断。

#### **用法：**

```xml
<action application="set" data="result=${fifo_member(add MyQueName {fifo_member_wait=nowait}user/1001@VoiceNetwork.ca )"/> 
```

#### **实现者：**

| **模块名称** | **源文件** | **最后修改** |
| --------------- | --------------- | ---------------- |
| mod_fifo       | mod_fifo.c     | 12782            |

## fifo_orbit_exten

设置目标分机和超时时间，将导致 FreeSWITCH 通过 FIFO 将呼叫者运行，直到达到指定的超时时间，然后呼叫者将被转移到目标分机。可在将呼叫者放入 FIFO 之前设置。

#### **用法：**

在 FIFO 语句之前使用：

```xml
<action application="set" data="fifo_orbit_exten=MyFIFOVoicemail:60"/> 
```

这将导致呼叫者在 FIFO 中等待 60 秒后，将呼叫转移到指定的语音信箱分机。

```xml
<action application="fifo" data="MyFIFO in"/> 
```

#### **实现者：**

| **模块名称**    | **源文件**       | **最后修订** |
| ------------- | ------------- | --------- |
| mod\_fifo     | mod\_fifo.c     | 8026      |
