# ARCHIVE_Sip_execute_on_image 

 

## 关于

在收到T.38邀请时立即执行一个应用程序。这个变量与execute\_on\_answer类似。当您收到T.38重新邀请但没有CNG音调时，或者希望忽略音调，只在收到T.38重新邀请时才做出反应，您可以运行t38\_gateway或rxfax等。
