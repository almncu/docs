## 配置文件

mod\_rtmp允许您监听多个配置文件。每个配置文件绑定到唯一的IP:端口组合（类似于[Sofia](../Configuration/Sofia-SIP-Stack/index.mdx#faq)配置文件）。每个配置文件可以配置不同的设置。

这样可以让您在多个地址/端口上监听呼叫，或者根据用户连接的配置文件为不同的用户提供不同的连接设置。

### 基本设置

#### bind-address

```xml
<param name="bind-address" value="0.0.0.0:1935" />
```

这将设置要绑定和监听连接的地址和端口。它可以绑定到0.0.0.0（即所有地址）。

#### context

```xml
<param name="context" value="public" />
```

这将设置要处理呼叫的拨号计划上下文。

#### dialplan

```xml
<param name="dialplan" value="XML" />
```

这将设置要使用的拨号计划提供程序，通常是XML。

#### auth-calls

```xml
<param name="auth-calls" value="true" />
```

这控制是否允许未经身份验证的呼叫。如果设置为true，则只能进行身份验证的呼叫。

#### buffer-len

```xml
<param name="buffer-len" value="50" />毫秒
```

这控制要为客户端缓冲多少时间的媒体数据，以毫秒为单位。

#### chunksize

```xml
<param name="chunksize" value="512" />
```

这设置了传出RTMP块的最大大小。
