## 基础的 XML 语法

如果你对 XML 不熟悉，它可能看起来有些奇怪。它与用于网站的 HTML 非常相似。在互联网上有许多精彩的网站可以为您提供更详细的 XML 教程，例如...

* <http://www.w3schools.com/xml/default.asp>
* <http://www.xml.com/pub/a/98/10/guide0.html>

XML 使用标签 "<something>" 和闭合标签 "</something>"。在这些开始标签和闭合标签之间，通常会有更多与配置部分相关的标签。您还会看到一些 "自闭合" 的单个标签，例如 "<something/>"。这些标签在一个标签中执行某些操作，不需要单独的开始标签和闭合标签（比如 &lt;action ...> 标签）。

将 XML 视为书籍的大纲是最简单的方式。在书籍大纲中，有节、章节和页。节中包含章节，章节中又包含这样的页面。

* 书籍
   * 第一节
      * 第一章
         * 1-1-1 页
            * 1-1-2 页
         * 第二章
            * 1-2-1 页
               * 1-2-2 页
   * 第二节
      * 第一章
         * 2-1-1 页
            * 2-1-2 页
         * 第二章
            * 2-2-1 页
               * 2-2-2 页

以 "&lt;!--" 开头，以 "-->" 结尾的标签是注释，用于添加进一步的文本解释或禁用标签（或标签），而不从文件中删除它们。

注意：为了帮助我们人类视觉上的理解，通常建议每个从属标签按照你在项目列表中的缩进方式进行缩进，但这对于解析器理解层次结构和处理它并非必需。
