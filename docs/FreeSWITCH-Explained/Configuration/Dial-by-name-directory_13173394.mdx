# 姓名拨号目录

## 关于

dial_by_name_directory.js 是原始的姓名拨号脚本 scripts/aadir.js的重构版本。

点击此处展开目录

* 2 [安装](#installation)
* 3 [使用](#usage)

### 概念

该脚本询问呼叫者输入他们要联系的用户的姓氏/名字。呼叫者输入字母，直到找到唯一匹配，然后将其连接到该用户。

### 安装

* 由于这是一个 javascript 应用程序，必须配置和启用 mod_spidermonkey。
* 将 dial_by_name_directory.js 复制到 ${freeswitch_base_dir}/scripts 目录下。
* 将 dial_by_name_directory.xml 复制到某个 ${freeswitch_base_dir}/conf/lang/en 目录下，并确保所在的目录被 en.xml 包含。

### 使用

在拨号计划中：

```xml
<extension name="dial_by_name">
	<condition field="destination_number" expression="^6000$">
		<action application="javascript" data="dial_by_name_directory.js" />
	</condition>
</extension>
```

或者，在 IVR 中：

```xml
<entry action="menu-exec-app" digits="9" param="javascript dial_by_name_directory.js"/>
```
