# min-idle-cpu

## 关于

min-idle-cpu指定了FreeSWITCH拒绝额外呼叫的最小CPU闲置时间。

## min-idle-cpu

用途：autoload\_configs/switch.conf.xml 

min-idle-cpu保留了指定百分比的CPU闲置时间给其他进程使用。此设置避免了除FreeSWITCH之外的进程耗尽CPU时间。如果min-idle-cpu设置为25，并且FreeSWITCH处理导致CPU闲置时间下降到25%以下，那么FreeSWITCH将拒绝处理额外的呼叫，以确保其他进程有足够的CPU时间可用。
