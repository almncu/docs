## 在 Screen 中运行 FreeSWITCH (Linux/Unix/BSD/OSX)

有人问如何以与 Asterisk 的 asterisk -r 命令类似的方式 "连接" 到正在运行的 FreeSWITCH 进程。对于在 Linux/Unix 环境下运行 FreeSWITCH 的用户，可以使用方便的 **screen** 实用程序，由 GNU 提供。（参见 <http://www.gnu.org/software/screen/>）

### 在 Screen 中启动 FreeSWITCH

确保 FreeSWITCH 没有运行，然后执行以下命令：

screen -d -m -S freeswitch /usr/local/freeswitch/bin/freeswitch

这将在可连接的 screen 中作为守护进程启动 FreeSWITCH。确认 FreeSWITCH 是否正在运行：

screen -ls

```xml
There is a screen on:
        2652.freeswitch (Attached)
1 Socket in /var/run/screen/S-root.
```

### 连接到正在运行的 FreeSWITCH 进程

这是一个酷炫的功能！从终端会话中简单地启动 screen：

screen -r

或者，如果有多个进程使用 screen 启动，请指定 FreeSWITCH 进程：

screen -r freeswitch

### 从正在运行的 FreeSWITCH 进程中分离

要从 FreeSWITCH "screen" 中分离，只需按下 Ctrl-A，然后按下 Ctrl-D，之后你会看到：

[detached]

... 然后您将返回到终端会话。

### 在 Screen 中终止 FreeSWITCH 进程

要终止在 **screen** 中运行的 FreeSWITCH 会话，只需在 Linux/Unix 命令提示符中使用 -stop 开关：

freeswitch -stop

您还可以在 [XML-RPC](../Modules/mod_xml_rpc_1048928.mdx#about) 会话或 fs\_cli 中发出 **fsctl shutdown** 命令。
