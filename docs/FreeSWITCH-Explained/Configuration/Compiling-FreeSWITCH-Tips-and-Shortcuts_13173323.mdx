## 介绍

本页面包含关于如何使用不同的Makefile选项来帮助您进行编译的有用提示。

## 讨论

编译FreeSWITCH需要编译许多库文件和各个子目录中的文件。有时您只需要编译FreeSWITCH的某些部分。其他时候，您可能希望从头开始编译。下面列出的选项将会有所帮助。

## 选项

* make clean - 清理编译环境
* make current - 清理编译环境，执行git更新，然后执行make install
* make core\_install (或 make install\_core) - 重新编译并安装核心文件。如果您正在处理一个核心文件并希望重新编译而不是整个过程，这个选项很方便。
* make mod\_XXXX-install - 重新编译并安装单个模块。以下是一些示例：
   * make mod\_openzap-install  
   * make mod\_sofia-install  
   * make mod\_lcr-install
* make samples - 这不会替换您的配置。相反，它会创建默认的扩展和拨号计划，以运行FreeSWITCH的基本配置。
