# 无线电通信

## 关于

业余无线电常用于灾难期间以及之后。FreeSWITCH能够实现用户之间的多种不同通信能力的桥接，是将移动电话、VoIP连接和传统电话线连接起来的理想中枢。将业余无线电连接到FreeSWITCH，使其成为搜救（SAR）、灾难响应等方面宝贵的工具。

卡特里娜飓风袭击后，新奥尔良的通信受到了限制，即使警察与其他参与救援的机构之间也存在沟通问题。当世界贸易中心大楼倒塌时，移动电话网络遭到严重破坏，难以处理其所面临的大量通信量。此外，拥有移动电话的个人很难与救援人员联系并协调救援行动。业余无线电操作员在这两种情况下提供了帮助，通过添加FreeSWITCH，即使在规模较小的行动中，也能增加通信效果。

这些技术中的许多将直接适用于商业无线电系统、无需许可的无线电系统（FRS/GMRS等）和其他无线电系统。在将无线电连接到FreeSWITCH之前，始终要检查相关法律法规。法律有所不同，无法一一列举并保持准确性。一般来说，无线电本身连接是合法的，但是，特别是在许可频段上进行传输是非法的。**您有责任确保遵守所在地的法律法规**。

无线电互连项目

- \[fs_rpt.pl\](../Client-and-Developer-Interfaces/fs_rpt.pl_13173092.mdx#about) - 这是一个Perl脚本，可以将无线电连接到会议中，允许无线电用户进行电话呼叫（自动连线），并将会议对话通过无线电端口中继出去。如果需要，这也可以是一个完整的中继器。
- \[mod_echolink\](https://wiki.freeswitch.org/wiki/Mod%5Fecholink "Mod echolink") - 这是一个与Echolink.org网络接口的模块。
- \[Ettus Radio\](http://ettus.com/) 是一个值得研究的软件定义无线电系统（开源）。
