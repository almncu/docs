# ODBC.close 

## 关于

关闭并释放ODBC资源。请注意，一旦脚本执行完毕，引擎将自动关闭任何ODBC资源。这个close方法旨在帮助您释放那些长时间运行脚本所占用的资源。

### 概述

```xml
db.close();
```

### 返回值

- bool - 每次都返回True。

### 示例

```xml
//简单示例
use("ODBC");
var DSN = "test_odbc";
var DB_USER = "abc";
var DB_PASS = "abc";
var db = new ODBC(DSN, DB_USER, DB_PASS);
db.connect();
//对数据库连接进行操作
db.close(); 
```

```xml
//示例: 在脚本中多次打开连接
var db = new ODBC(DSN, DB_USER, DB_PASS);
db.connect();
//做些什么...
db.close();
db = new ODBC(DSN, DB_USER, DB_PASS);
db.connect();
//做些什么...
db.close();
```
