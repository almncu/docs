# ODBC.execute 

## 关于

执行SQL语句。

此方法不会创建记录集，应用于INSERT、UPDATE和DELETE语句。

[点击此处](#synopsis)展开目录

* 1 [概述](#synopsis)
* 2 [返回值](#example)
* 3 [示例](#example)

### 概述

```xml
result = db.execute("sql语句");
```

###   
返回值

* 布尔值 - 如果成功调用语句则为True。

### 示例

```xml
use("ODBC");

var db = new ODBC("dsnName", "username", "password");
var sql="delete from users";
db.execute(sql);
```
