## 用法

连接到ODBC数据库。

### 返回值

* true - 成功连接到数据库
* false - 连接失败，请检查DSN设置。

### 示例

```bash
use("ODBC");
var db = new ODBC("dsnName", "username", "password");
db.connect();
```
