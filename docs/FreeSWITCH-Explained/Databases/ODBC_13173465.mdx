## ODBC

Freeswitch支持ODBC，可以将系统配置和状态的各个方面存储在数据库中。SIP注册、呼叫/通道状态、语音邮件消息和语音邮件偏好设置可以存储在数据库中。此外，还可以使用ODBC在数据库中存储_mod_limit_配置、_mod_nibblebill_和通话详单信息。

要了解更多信息，请参阅[在核心中使用ODBC](./ODBC-DSN/Using-ODBC-in-the-core_6586653.mdx)

## 在JavaScript中使用ODBC

JavaScript引擎可以通过ODBC模块与各种数据库进行通信。

请记住，您必须在`spidermonkey.conf.xml`文件的`conf/autoload_configs`目录中的配置文件中添加/取消注释以下内容：

```xml
<load module="mod_spidermonkey_odbc"/>
```

编译帮助：[Mod_spidermonkey_odbc](https://wiki.freeswitch.org/wiki/Mod%5Fspidermonkey%5Fodbc "Mod spidermonkey odbc")
