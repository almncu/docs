# 日志记录

## 拨号计划工具 - 日志

### 描述

将文本字符串记录到控制台。有关日志记录的更多信息，请参见 [mod\_logfile](../Modules/mod_logfile_1048990.mdx#about)。

### 使用方法

log [日志级别] `<日志消息>`

如果未指定日志级别，则默认为 DEBUG。

此应用程序可以从 XML 拨号计划中以 _[内联方式](https://wiki.freeswitch.org/wiki/Dialplan%5FXML#Inline%5FActions "Dialplan XML")_ 运行。

### 示例

```xml
<action application="log" data="DIALING Extension DialURI [${sip_uri_to_dial}]"/>
<action application="log" data="INFO DIALING Extension DialURI [${sip_uri_to_dial}]"/>
<action application="log" data="DEBUG DIALING Extension DialURI [${sip_uri_to_dial}]"/>
```
