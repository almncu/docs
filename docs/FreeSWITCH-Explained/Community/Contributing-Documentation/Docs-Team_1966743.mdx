## 关于

此页面跟踪 FreeSWITCH 文档团队的工作。任何 FreeSWITCH™ 社区用户都可以要求编辑页面；即使你不会编写代码，这也是回馈社区的绝佳方式。

如果你想要帮助完善 FreeSWITCH™ 文档，请按照[贡献文档](index.mdx#about)页面上的指南进行。谢谢！

文档团队会在[Confluence 上的博客文章](https://confluence.freeswitch.org/pages/viewrecentblogposts.action?key=FREESWITCH)上发布活动更新。

## 需要关注

### 文件存储

名为 [文件存储](../../Configuration/How-Tos/Holding-Bin_1966771.mdx#tls) 的页面是一个很好的地方，可以在找到最终目标之前存储有用的信息。这些注释和想法很多来自开发团队和 IRC 频道上的高级用户，它们过于珍贵，不能被遗失。

如果你在 [文件存储](../../Configuration/How-Tos/Holding-Bin_1966771.mdx#tls) 中看到了一个现在已经有一个适合存放的页面的部分，请将其移动到那里，并且增加描述，使其易读和有用。

[John Boteler](https://freeswitch.org/confluence/display/~boteman) 将文件存储移到 "Configuration -> How To's" 下，尽管在目录表中的实际位置只是视觉上的，但一切都可以通过搜索或标签标记找到。

## 我们的服务

我们提供以下翻译服务：

1. **文档翻译**：我们能够翻译各种类型的文件，包括商务文件、法律文件、技术文档等。我们会确保翻译的准确性和流畅性，使您的文档在目标语言中保持与原文一致的意思。

2. **口译服务**：我们的口译员拥有良好的语言表达能力和专业知识，能够在各种场合提供高质量的口译服务。无论是商务会议、学术研讨会还是国际会议，我们都能为您提供专业的口译员。

3. **网站本地化**：我们能够帮助您将网站内容本地化到目标语言，确保您的网站在不同语言环境下获得更好的用户体验。我们会保持原始页面的风格和布局，同时进行语言和文化上的适应。

### 我们的承诺

- 高质量翻译：我们承诺提供高质量的翻译，确保准确性和流畅性。

- 专业团队：我们拥有经验丰富的翻译团队，精通多种语言和领域。

- 保密性：我们会严格保护客户的文档和信息的机密性。

- 准时交付：我们会按时交付翻译项目，确保客户的时间计划得以遵守。

- 客户满意度：我们重视客户的需求和反馈，努力确保客户满意。
