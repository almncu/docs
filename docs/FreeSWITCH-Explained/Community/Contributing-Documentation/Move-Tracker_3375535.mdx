# 移动追踪器

## 关于

[Belaid Areski](https://freeswitch.org/confluence/display/~areski) 在Google Docs上建立了一个电子表格，供Docs团队成员跟踪将维基页面迁移到Confluence。

您需要具备正确的权限才能更新Google电子表格，请与[John Boteler](https://freeswitch.org/confluence/display/~boteman)或[Belaid Areski](https://freeswitch.org/confluence/display/~areski)联系，以了解有关此电子表格的问题：

<https://docs.google.com/spreadsheets/d/1qsG-kRymvKlNBapnBLw86W130VdbnK6naYapbR%5FUNds/edit?pli=1#gid=1187898333>

## 状态模式

待定 - 旧的维基页面尚未处理。

编辑中 - Docs团队成员正在编辑页面，此状态将“锁定”页面，请在执行任何操作之前与其咨询。

已移动 - 页面已从旧的维基复制到Confluence，但尚未进行格式化或有效性验证。

已完成 - 页面已复制、格式化并可供公众使用。

删除 - 页面已过时或在Confluence上不需要，请忽略。

重复 - 通常是重定向的维基链接，无需复制页面。

请求删除 - 如果不确定是否应该删除页面，请将其标记为请求删除，将会请求开发人员确认是否需要。
