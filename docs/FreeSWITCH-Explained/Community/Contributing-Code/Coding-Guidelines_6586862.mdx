## 结构体和大括号

请遵守以下约定：

* 优先使用制表符而不是空格，制表符宽度为四个字符
* 在条件语句中始终使用大括号，除非它们在同一行且非常短

```xml
  if (foo) printf("blah\n");
```

* **if** 后始终加上空格

```xml
if (foo) {

}
```

* 嵌套的 **else**

```xml
if (foo) {
...
} else if (bar) {

} else {

}
```

* 函数定义换行

```xml
int foo (int arg)
{
...
}
```

## 注释

请根据您使用的语言使用适当的注释。

### C

在 C 程序中添加注释时，请使用 /\* 和 \*/。请勿在 C 程序中使用 // 进行注释。

### C++

在 C++ 程序中，请使用 // 风格的注释。

### Lua

Lua 有两种注释风格：

* 单行注释

```xml
-- 这是一个单行注释
```

* 多行注释

```xml
--[[
所有这些
都是被注释的
]]
```
