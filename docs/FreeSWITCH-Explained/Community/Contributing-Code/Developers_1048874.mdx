## Github 仓库  

### 对于提交者

首先，创建一个 Github 账户

接下来，在 Github 中创建一个 github.com/signalwire/freeswitch 的 fork

最后，克隆仓库：

#### HTTPS  

git clone https://github.com/${YOUR_USERNAME}/freeswitch.git

YOUR\_USERNAME 是你的 Github 用户名。在需要时，Git 会提示你输入密码。

#### SSH

git clone ssh://git@github.com:${YOUR_USERNAME}/freeswitch.git

为需要的人提供的示例 ssh\_config。（大多数人不需要这个）

**示例 ssh\_config**

```xml
host github.*
     IdentityFile ~/.ssh/my_pubkey_file
```

### 对于非提交者

执行以下命令来克隆该仓库：

git clone https://github.com/signalwire/freeswitch.git
