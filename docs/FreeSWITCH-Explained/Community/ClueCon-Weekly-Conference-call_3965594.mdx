# ClueCon每周会议电话

## **暂停每周会议电话**

之前的每周会议电话是关于FreeSWITCH开发的。自2020年底以来，由于大多数社区成员无法参加这些电话，我们将暂停这些电话。由于目前没有进行每周会议电话，因此下面的会议链接等都无法使用。

## 关于

这是我们每周会议电话的主页，旨在帮助FreeSWITCH项目。将讨论如编程任务和目标、文档和[清洁工作](Contributing-Code/Janitorial-Tasks_3965753.mdx#-freeswitch--janitorial-tasks-)等子项目。

关注我们的Twitter账号[@freeswitch](https://twitter.com/freeswitch)或[Facebook](https://www.facebook.com/freeswitch)，获取最新更新。

## 日程安排

所有公开的会议电话都在FreeSWITCH™ 888会议桥上进行，如下所示。所列时间为美国中部时区的绝对时间，无论是标准时间还是夏令时。这意味着UTC时间在一年中的不同时间会有所变化。请使用[时区转换器](http://www.thetimezoneconverter.com/index.html)为您所在地区查询时间。

周二 12:00 CT（美国） — 每周Bug搜索和Bug维护，检查Bug是否真实存在，决定测试和排除故障的最佳方式，从而改善FreeSWITCH™。

待定 - 每周文档团队电话会议；向专家提问以改善文档，协调从旧的MediaWiki复制哪些页面，等等。

周三 12:00 CT — 每周ClueCon社区电话会议，邀请嘉宾发表演讲，并进行有关FreeSWITCH™的自由讨论。

周五 12:00 CT — 周五Free-For-All电话会议，紧随VoIP用户会议之后。

静音

加入会议时，您会被**静音**，如果您希望发言，请按0（这样可以减少会议上的背景噪音）。除非您在积极交谈，否则请保持静音。

在每周的ClueCon电话会议中，主讲人演讲期间可能会进行静音。按0将使您进入排队以提问，当轮到您发言时，主持人会取消静音。

一定要关注IRC频道 [#freeswitch](IRC_1970341.mdx#pb) 在 [irc.freenode.net](IRC_1970341.mdx#pb)！会议期间经常有相关的评论。

会议提示音

会议中的提示音是什么？

- 一个高音提示音 = 有人加入会议
- 两个高低音提示音 = 有人离开会议
