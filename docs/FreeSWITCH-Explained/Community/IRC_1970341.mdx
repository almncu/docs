## 网络礼仪

* 请不要使用一个通用的用户名，比如fsuser，请在登录后更改您的 /nick。
* 请不要在频道内频繁发送消息。**如果您需要复制和粘贴超过1-2行的内容，请使用我们的** [pastebin](http://pastebin.freeswitch.org)！
* 请不要要求提问的权利。直接提问即可。
* 请不要无礼、粗鲁或者言行不当。
* 请不要在频道内进行垃圾信息或者刷屏行为。
* 欢迎使用 \~take-a-number，这样之后加入频道的用户可以看到您的待解决问题。
* 如果没有人立即回答您的问题，可能是因为当前频道里没有人知道答案，或者他们正忙或者离开。请在重复您的问题之前稍等一下，使用 \~take-a-number，或者稍后再试（提示：主要的开发人员位于美国，如果当地是晚上时间，可能不会在线）。  
   * 如果长时间没有得到答案，可以尝试将问题升级到邮件列表中，那里有更多人关注。但是请不要在IRC和邮件列表上同时提问相同的问题；**请在两者之间选择一个**。
* FreeSWITCH是一个免费的产品，频道里的用户是志愿者为该项目贡献时间。请在提问时考虑到这一点，不要期望得到迅速的回复。如果您觉得这不足够，可以通过发送电子邮件至[consulting@freeswitch.org](mailto:consulting@freeswitch.org)获得商业支持。
* 另请参阅：<http://workaround.org/getting-help-on-irc>

如果您想要做出贡献，请随时留在频道内。特别对于初学者来说，您将学到很多关于FreeSWITCH的知识，并很快能够回答其他人的问题，继续推动循环。


## 机器人

我们有一个名为c888的机器人，协助运营该频道。但是他经常失联。

## 用户命令

命令可以在主频道或私聊给 c888 使用。

\~take-a-number $question

将你的问题存储在[等待列表](http://conference.freeswitch.org/waiting.cgi)中。这意味着其他人可以在以后上线时查看当前等待回答的问题。

```xml
 <newfsuser> ~take-a-number 如何配置FS在早上为我做咖啡
 <@c888> newfsuser，你是第5个
```

如果你找到答案，请记得将你的问题从列表中移除。如果开发团队的成员帮助你，他会将问题从堆栈中弹出。

你在列表中只能有一个问题。如果你想要提出另一个问题，请先弹出当前问题，再取一个新的编号。

如果你离开频道，你的问题将自动删除。如果你重新加入频道，你需要重新取一个编号。问题的回答不按照编号顺序进行，而是按照人们希望回答的顺序，根据他们的知识和经验。

\~popme

从[等待列表](http://conference.freeswitch.org/waiting.cgi)中移除你的问题。

```xml
<newfsuser> ~popme
<@c888> 5 newfsuser：(04/20/12 08:18:35) 如何配置FS在早上为我做咖啡
```

\~wiki $query

发布一个链接，以搜索FreeSWITCH百科。

```xml
<newfsuser> ~wiki mod_sofia
<c888> 试试这个链接：https://freeswitch.org/confluence/dosearchsite.action?queryString=mod_sofia&where=conf_all
```

\~bug FS-XXXX

显示在我们的[错误追踪器](http://jira.freeswitch.org/)上被提交的Jira票的信息。

```xml
<newfsuser> ~bug FS-3012
<c888> [FS-3012] fsctl暂停改进
<c888> 报告者：Bernhard Suttner [suttner]
<c888> 受理人：Anthony Minessale II [anthm]
<c888> 状态：已关闭
<c888> 链接：http://jira.freeswitch.org/browse/FS-3012
```

### \~rfc XXXX

显示 RFC 文档。

&lt;newfsuser> ~rfc 3261
&lt;@c888> RFC3261 - SIP: 会话初始化协议
&lt;@c888> <http://www.faqs.org/rfcs/rfc3261.html>
&lt;@c888> <http://rfc.net/rfc3261.html>

### s/$old_text/$new_text/\[g\]

机器人会将你输入的内容更正为你打算输入的内容：

```xml
<newfsuser> FreeSWITCH is awsome
<newfsuser> s/awsome/awesome/
<@c888> newfsuser 打算说的是... FreeSWITCH is awesome
```

可选的/g标志将替换所有出现的 $old\_text。

```xml
<newfsuser> FreeSWITCH is awsome awsome
<newfsuser> s/awsome/awesome/
<@c888> newfsuser 打算说的是... FreeSWITCH is awesome awsome
<newfsuser> FreeSWITCH is awsome awsome
<newfsuser> s/awsome/awesome/g
<@c888> newfsuser 打算说的是... FreeSWITCH is awesome awesome
```

### \~whatis $query

从机器人的数据库中显示定义。

```xml
<newfsuser> ~whatis pastebin
<c888> Pastebin 是一个很好的粘贴内容的地方，这样你就不会在频道里刷屏：http://pastebin:freeswitch@pastebin.freeswitch.org

<newfsuser> ~whatis jira
<@c888> Jira 是我们的错误追踪器：https://freeswitch.org/jira/
```

该数据库可以用于查找任何 SIP 响应代码的含义：

```xml
<newfsuser> ~whatis 100
<@c888> 尝试
<newfsuser> ~whatis 180
<@c888> 响铃
<newfsuser> ~whatis 200
<@c888> 确认
<newfsuser> ~whatis 401
<@c888> 未经授权
<newfsuser> ~whatis 401
<@c888> 未经授权
<newfsuser> ~whatis 403
<@c888> 禁止
<newfsuser> ~whatis 404
<@c888> 未找到
```

### \~pb

让机器人发布一个 Pastebin 链接。

```xml
<newfsuser> ~pb
<@c888> http://pastebin:freeswitch@pastebin.freeswitch.org/
```

### \~google $query

发布一个指向 [lmgtfy.com](http://lmgtfy.com/) 的链接。

```xml
<newfsuser> ~google 愚蠢问题
<c888> 看看这个网址 http://www.lmgtfy.com/?q=dumb%20question
```

### \~8ball

[魔法8号球](http://en.wikipedia.org/wiki/Magic%5F8-Ball)

```xml
<newfsuser> ~8ball
<c888> 最有可能。
<newfsuser> ~8ball
<c888> 如果我是你，我会跑。
<newfsuser> ~8ball
<c888> 前景不是很好。
```

### Cluecon

输入 **Clue** 或者 **Cluecon** 会让机器人发布到 Cluecon 网站的链接。

```xml
<c888> http://www.cluecon.com
```

## Admin commands

一些指令受限制为了防止滥用，防止机器人洪水般发消息至频道。

### \~waiting

展示[当前等待中的票](http://conference.freeswitch.org/waiting.cgi)。

```xml
<@fsadmin> ~waiting
<@c888> 11:22 等候无人！
<@c888> 11:22 http://conference.freeswitch.org/number.jpg
```

非管理员用户可以在浏览器中[查看列表](http://conference.freeswitch.org/waiting.cgi)。

###  \~head

展示最新的 Git HEAD。仅限管理员用户。

```xml
<@fsadmin> ~head
<@c888> 最新发布:
<@c888> commit 1c1720d37143ccb6a3a907dab320631476b302c6
<@c888> 作者: Anthony Minessale <anthm@freeswitch.org>
<@c888> 日期:   Wed Jan 11 12:34:57 2012 -0600
<@c888>     修复错别字
```

### +list

显示当前在[888](ClueCon-Weekly-Conference-call_3965594.mdx#about)会议中的参与者。

```xml
<@fsadmin> +list
<@c888> OK
* @c888 16:58 4833 John Smith 1002 (hear)
* @c888 16:58 4832 John Doe 10001 (hear)
* @c888 16:58 4831 Brian West 1000 (hear|floor)
* @c888 16:58 4824 Anthony Minessale 1010 (hear)
```
