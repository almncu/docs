## 英国

### O2/Be Unlimited

在BeBox II/III或O2 Wireless Box II/II（实际上是重贴标签的Thomson ST585）的默认配置中存在问题，该设备会将自身绑定到端口5060。这似乎影响一些设备尝试在本地网络之外的服务器上注册。例如，Aastra设备无法成功注册到远程FreeSWITCH服务器，但Siemens S685IP设备可以。您可以通过配置FreeSWITCH允许在5060端口之外的端口注册，或者使用此处的信息重新配置ST585路由器来解决此问题：<http://forum.o2.co.uk/viewtopic.php?t=1313>

要克服这些限制，必须通过telnet连接到路由器并运行以下命令来禁用SIP ALG：

```xml
 connection unbind application=SIP port=5060
```

有关更多信息，请参阅<http://beusergroup.co.uk/technotes/index.php?title=BEBox%5FCommands>。

目前提供的BeBox路由器型号是Technicolor TG582n。

### 英国电信/BT

BT HomeHub 2.0 包括 VoIP 功能，可能会干扰您的 VoIP 设备 / FreeSWITCH 注册。另请参阅：<http://www.the-scream.co.uk/forums/t29842.html>，以了解潜在的安全问题。由于过于简化的用户界面针对消费者设计，BT HomeHub 的配置选项有限。建议使用具备 QoS 功能的第三方路由器来克服这些限制。

BT HomeHub 3 不再包含 VoIP 功能，但仍提供非常基本的配置选项。
