# FSClient

## 关于

FSClient是一个完整的Windows sip客户端，使用[WPF/.NET 4.0](../../Client-and-Developer-Interfaces/Embedding-FreeSWITCH/index.mdx#php)编写，并使用[Embedded FreeSWITCH](../../Client-and-Developer-Interfaces/Embedding-FreeSWITCH/index.mdx#php)。它支持外部和内部联系人簿以及通过插件系统的完整耳机功能。FSClient旨在成为一个功能齐全的SIP客户端，包括标准的企业级客户端功能。它可以安装运行或独立运行，唯一要求是支持.NET 4.0。作为开源软件，它还可以作为使用嵌入式FreeSWITCH的示例。更多信息请参考其GitHub页面：<https://github.com/mitchcapper/FSClient>
