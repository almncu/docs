# Khomp/变量

点击此处展开目录

* **KDropCollectCall**: 在响铃或接听来电之前设置，启用（"yes"）或禁用（"no"，默认）基于来自公共中继的信令的代收呼叫降落，双重应答，音频音调识别（可以全局定义）；
* **KR2SendCondition**: 在接收到来电之前设置为数字值，调整此终端的条件为此值（仅在R2信令上可用）；
* **KR2GotCategory**: 当接收到来电时由终端调整，带有呼叫者的类别号码（仅在R2信令上可用）；
* **KR2GotCondition \***: 由终端调整，在由FreeSWITCH（**bridge** 应用程序）进行呼叫之后可用。返回时具有在进行呼叫时接收到的远程端条件（仅适用于R2信令）；
* **KISDNGotCause \***: 由终端调整，在由FreeSWITCH（**bridge** 应用程序）进行呼叫之后可用。返回时具有在进行呼叫时接收到的ISDN "cause"代码（仅适用于ISDN信令）；
* **KCallAnswerInfo \***: 由终端调整，在由FreeSWITCH（**bridge** 应用程序）进行呼叫之后可用。返回时包含标识进行呼叫的业务信息（仅适用于数字信令 - E1和GSM）；
* **KSmsDelivered**: 在使用应用程序**KSendSMS**发送短信消息时由终端调整，表示消息是否成功发送（"yes"）或未能发送（"no"）；
* **KSmsErrorCode**: 在使用应用程序**KSendSMS**发送短信消息时由终端调整，包含发送消息时发生的错误代码；
* **KSmsErrorName**: 在使用应用程序**KSendSMS**发送短信消息时由终端调整，包含错误的名称，如果没有错误，则为"None"；
* **KSmsType**: 在短信消息的上下文中为输入终端调整，定义接收到的消息类型（可以包含值"message"、"confirm"或"broadcast"）；
* **KSmsFrom**: 在短信消息的上下文中为输入终端调整，设置接收到的消息的来源号码（仅在"message"和"confirm"类型上可用）；
* **KSmsDate**: 在短信消息的上下文中为输入终端调整，设置消息的发送日期（仅在"message"和"confirm"类型上可用）；
* **KSmsSize**: 在短信消息的上下文中为输入终端调整，包含接收到的消息的大小（以字节为单位）（仅在"message"和"broadcast"类型上可用）；
* **KSmsMode**: 在短信消息的上下文中为输入终端调整，包含接收到的消息的编码类型（仅在"message"和"broadcast"类型上可用）；
* **KSmsBody**: 在短信消息的上下文中为输入终端调整，包含接收到的消息的文本内容（在"message"和"broadcast"类型上可用）；
* **KSmsDelivery**: 在短信消息的上下文中为输入终端调整，包含先前发送的消息的交付日期（仅在"confirm"类型上可用）；
* **KSmsStatus**: 在短信消息的上下文中为输入终端调整，包含先前发送的消息的状态（仅在"confirm"类型上可用）；
* **KSmsSerial**: 在短信消息的上下文中为输入终端调整，包含接收到的消息的序列号（仅在"broadcast"类型上可用）；
* **KSmsPage**: 在短信消息的上下文中为输入终端调整，包含接收到的消息的页码（仅在"broadcast"类型上可用）；
* **KSmsPages**: 在短信消息的上下文中为输入终端调整，包含要接收的总页面数（仅在"broadcast"类型上可用）；
* **KUserInfoDescriptor**: 设置/报告用户到用户信息消息的协议描述符（ISDN）；
* **KUserInfoData**: 设置/报告用户到用户信息消息中的数据（ISDN）；
* **KFaxSent**: 在使用**KSendFax**应用程序发送传真时由终端调整，确定传真是否成功发送（"yes"）或未能发送（"no"）；
* **KFaxReceived**: 在使用**KReceiveFax**应用程序接收传真时由通道调整，确定是否成功接收到传真（"yes"）或未能接收（"no"）；
* **KFaxResult**: 在使用**KSendFax**或**KReceiveFax**应用程序（分别）发送或接收传真时由通道调整，定义执行的结果。

#### Markdown格式转换器


此转换器可将Markdown文本转换为其他格式的文本，如HTML、PDF等。转换过程中保留文本中的所有标记和格式。


##### 使用方式


运行以下命令安装Markdown格式转换器：


```
npm install markdown-format-converter
```


在你的项目中导入转换器模块并调用适当的方法，例如：


```javascript
const converter = require('markdown-format-converter');

const input = '# 这是一个标题';
const output = converter.toHTML(input);

console.log(output);
```

以上代码将把输入的Markdown文本转换为HTML格式，并在控制台上打印输出结果。


##### 支持的格式


Markdown格式转换器支持以下输出格式：


- HTML
- PDF
- Word文档
- EPUB电子书
- 幻灯片（PPT）


##### 注意事项


请确保你的项目中已安装了相关的依赖包，并按照格式转换器文档中的说明正确调用相应的功能。
