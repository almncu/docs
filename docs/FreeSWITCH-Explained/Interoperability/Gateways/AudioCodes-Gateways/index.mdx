### AudioCodes

AudioCodes 有几个系列的设备：

* FXS/FXO 网关（MP 系列）。
* PRI 网关（Mediant 系列）。
* IP 电话（目前有两个型号：HD310、HD320）。

#### FXS/FXO 网关

* **AudioCodes MediaPack MP114 FXS SIP**
* **AudioCodes MediaPack MP118 FXS SIP**
* **AudioCodes MediaPack MP118 8-FXO SIP**

我们需要将 FreeSWITCH SIP 服务器的 IP 地址添加到 MediaPack 的 "Gateway Registration Name" 和 "Gateway Name" 中。在不使用 "suppress-cng"="true" 的默认内部 SIP 配置文件的情况下，与 MP118 FXO 配套使用会导致内部 SIP 电话出现可怕的背景噪音。

* **AudioCodes MediaPack MP118 4-FXS/4-FXO SIP 混合配置。具备呼入路由与呼出路由的来电显示功能，同时也支持内部分机。使用以下配置：_sip\_profiles/internal.xml: &lt;param name="suppress-cng" value="false"/>_

#### PRI 网关

Mediant 系列适用于 PRI 转 SIP/H.323 网关。有几个模块化和非模块化的配置，从 1 E1/T1 到 16xE1/T1 不等。

我用了一个星期的时间进行测试，看起来还可以，但没有机会在生产环境中进行全面评估。
