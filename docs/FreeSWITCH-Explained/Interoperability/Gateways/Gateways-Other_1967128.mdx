#### Multipoint MultiVoip 媒体网关

这是一个双 PRI 到 SIP 网关，旨在用于为具有 PRI 的传统 PBX 提供 SIP 网关。我在 FreeSWITCH 和 CopperCom CSX，以及 Lucent 5ESS 交换机之间进行了 SIP 到 PRI 网关测试。虽然在此配置中，Web UI 的一些标签被颠倒了，但其功能正常，并正确地传递呼叫。DNIS、CallerID 和 RDNIS 正确地从 PRI 传递到 SIP。

#### Patton M-ATA-1 微型模拟电话适配器

运行良好。文档简明，但功能齐全。带有 2 个 ISDN BRI 接口（一个 T，一个 S）的 Patton 4552 与 FreeSWITCH 配合非常好。已经测试过出厂时和最新的（5.3）SIP 固件。

#### Patton SmartNode 4524 FXS/FXO 网关

从 FXS 端口通过 T.38 发送传真到 FreeSWITCH（mod\_spandsp - rxfax）；已测试以下固件版本：

R4.2 2008-09-11：**失败** \- 结果（48）在允许的重试次数后断开连接

R5.2 2009-05-11：通过

R5.3 2009-11-17：通过

R5.4 2010-05-14：通过

R5.5 2010-05-18：通过

#### Quintum DX

良好运行。只支持 SIP。H323 存在数据包大小问题（奇怪！）（G729 20ms 为 72 字节数据包大小，G711 20ms 为 252 字节数据包大小）。

#### Goip GSM 网关（或 MadCom, Hybertone, Etross）

作为 SIP 网关正常工作。有关更详细的设置说明，请参阅 [Goip HowTo](./Goip-HowTo_1967130.mdx)。
