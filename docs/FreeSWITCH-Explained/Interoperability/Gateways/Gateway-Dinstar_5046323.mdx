# Dinstar 网关

## 关于

DAG1000-4S4O 是基于 SIP2.0 标准的 4 个 FXS 和 4 个 FXO 网关。提供一个 WAN 端口和三个 LAN 端口，可允许最多四台计算机在办公室或家中共享一条 ADSL 线路。

## Dinstar DAG1000-4S4O 网关

**特点**

 ● 主/备 SIP 帐号

 ● 灵活设置端口组

 ● IP 干线

 ● 灵活路由和操作

 ● 数据/语音/管理 VLAN

 ● 语音/FAX/调制解调器

 ● SNMP

 ● 固件自动更新/配置更新

**规格**

**物理接口**

 ● 电话接口：4 个 FXS、4 个 FXO、RJ-11

 ● 以太网接口：4 个 RJ-45 10/100Mbps、RJ-45

 ● LED 指示灯：电源、运行、网络和线路指示灯

**语音处理**

 ● 语音编解码器：G.711A/U law、G.723.1、G.729A/B

 ● 语音数据包功能：CNG、VAD、G.168、AJB

 ● 可编程增益控制

 ● DTMF 模式：信号/RFC2833/INBAND
