简称“Zapata 语音通信”

Zaptel 是指Jim Dixon的开放式计算机电话硬件驱动程序API。Zaptel驱动程序最初是为BSD和Jim的Tormenta系列DIY T1接口卡发布的。Digium后来根据Jim的设计生产了接口卡，并改进了Linux平台上的Zaptel驱动程序。然后，Digium还增加了其他遵循Zaptel API的电话硬件驱动程序。

Zapata 电话项目以墨西哥革命家Emiliano Zapata的名字命名，这表明了Zaptel项目具有革命性的特点。有关更多详细信息，请参见[http://www.zapatatelephony.org](http://www.zapatatelephony.org/)。
