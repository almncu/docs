### Gigaset S675 / S685 IP

最新测试版本：047.00

Gigaset S675 和S685 SIP DECT电话运行良好，支持带宽为G.722的编解码器。为了确保G.722宽带启用后正确支持DTMF，确保在基站的Telephony->Advanced Settings中选择[RFC 2833](https://tools.ietf.org/html/rfc2833)。为了配置正确的网络消息等待指示器支持，必须禁用内置答录机。您可以通过屏幕顶部的磁带图标的变化为网络图标来确定配置是否正确。语音邮件的等待指示器将显示屏幕左侧的FreeSWITCH语音邮件数量，并且更新非常快速。
可以在电话的Settings->Base->Add. Features->Eco Mode菜单中配置ECO DECT以实现显着的电话电源节约。唯一的限制是您需要使用同一系列的DECT电话连接到基站-不能混合使用不同型号的Gigaset电话。
