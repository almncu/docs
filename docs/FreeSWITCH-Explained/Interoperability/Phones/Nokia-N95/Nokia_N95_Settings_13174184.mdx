# Nokia_N95_Settings 

## 关于

本文档介绍了 Nokia N95 的设置信息。

点击此处展开目录

许多配备 Wi-Fi 的 Nokia 手机都内置了 SIP 客户端软件。以下示例描述了配置此软件以连接到 FreeSWITCH 的典型设置。该示例适用于 N95，但也适用于其他 Seris 60 手机。

假设 FreeSWITCH 是默认配置，并在 _192.168.0.7_ 上运行，我们要将 N95 注册为内部分机 _1000_。

首先确保您的手机上配置了正常运行的 Wi-Fi 连接，可以访问到您的 FreeSWITCH，同时确定您为接入点起的名称，本例中为 _MyAP_。

然后在 **工具/设置/连接/互联网电话** 中选择 **选项/新建配置文件**，将名称从 _默认_ 改为 _FreeSWITCH_（可以是任何您喜欢的名称，但以下示例假定为 FreeSWITCH）。

返回到 **工具/设置/连接/SIP 设置**，选择 **选项/新建 SIP 配置文件/使用默认配置文件**。

并按以下方式编辑设置。

```xml
配置文件名称：FreeSWITCH
服务配置文件：IETF
默认接入点：MyAP
公共用户名：sip:1000@192.168.0.7
使用压缩：否
注册：需要时
使用安全性：否
```

然后在 **代理服务器** 子菜单中按以下方式编辑设置。

```xml
代理服务器地址：sip:192.168.0.7
领域：192.168.0.7
用户名：1000
密码：1234
允许松散路由：是
传输类型：UDP
端口号：5060
```

然后返回到上一个屏幕并进入 **注册服务器** 子菜单，按以下方式编辑设置。

```xml
注册服务器地址：sip:192.168.0.7
领域：192.168.0.7
用户名：1000
密码：1234
传输类型：UDP
端口号：5060
```

然后返回主菜单，关闭手机然后重新打开（有时在编辑后设置可能无法正常工作，直到重新启动手机，尽管这可能是特定于我们手机上的软件版本）

手机重新启动后，找到**Internet tel.**图标（在不同版本的手机软件上似乎在不同屏幕上）

点击这个图标，你应该看到一个显示_Freeswitch_和_Not Connected_的屏幕，下面列出了_MyApp_接入点

高亮显示 _MyApp_ ，然后选择**选项/连接到服务**

几秒钟后，屏幕顶部的_Not Connected_应该变为_Connected_，屏幕右上方应该出现几个新的图标

检查手机是否在freeswitch上注册

现在尝试通过拨打**[1001@192.168.0.7](mailto:1001@192.168.0.7)**(需要打开字母模式)来呼叫另一个分机，或尝试将呼叫路由到分机1000。

就是这样！

进一步提高通话质量和AP的操作范围的方法是更改默认的Wi-Fi设置，进入**工具/设置/连接/无线局域网**，选择**选项/高级设置**，然后忽略警告并继续，现在您应该将**TX功率水平**调整到最高值，并禁用**节能**和**自动配置**。

注意-使用VoIP通过Wi-Fi时，N95会消耗电池，请随时准备好充电器。
