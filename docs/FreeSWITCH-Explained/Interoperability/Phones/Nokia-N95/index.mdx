## 连接 Nokia 设备到 FreeSWITCH

假设您已经定义了一个名为 'homeap' 的 Wi-Fi 接入点，并且您的 FreeSWITCH 盒子位于 192.168.0.9，并且默认配置可用。

* 进入主菜单，选择 "工具"，然后选择 "设置"，滚动到 "连接"，再滚动到 "SIP 设置"。
* 使用以下设置创建一个新的 SIP 配置文件;
  * **配置文件名称**: homefs
  * **服务配置文件**: IETF
  * **默认接入点**: homeap
  * **公共用户名**: sip:1000@192.168.0.9
  * **使用压缩**: 否
  * **注册**: 始终
  * **使用安全性**: 否
* 在代理服务器子菜单中;
  * **代理服务器地址**: 192.168.0.9
  * **域**: 192.168.0.9
  * **用户名**: 1000
  * **密码**: 1234
  * **允许宽松路由**: 是
* 在注册服务器子菜单中;
  * **注册服务器地址**: 192.168.0.9
  * **域**: 192.168.0.9
  * **用户名**: 1000
  * **密码**: 1234
  * **传输类型**: UDP
  * **端口**: 5060
* 按几次返回键，返回到 "SIP 设置" 菜单，在几秒钟后，您应该看到状态变为 "已注册"。选择刚刚创建的配置文件，并将其设置为默认配置文件。
* 返回到 "连接" 菜单，向下滚动到 "Internet 电话"。创建一个新配置文件，确保您刚刚创建的 SIP 配置文件 (homefs) 被选为与此配置文件一起使用。

就是这样。您可以在手机的 "设置"，"通话" 菜单中调整默认呼叫路由。

**请注意：**如果您不在与FreeSWITCH相同的局域网（LAN）上，您需要从诺基亚（Nokia）安装一个“高级SIP设置”应用，并设置STUN服务器。您可以从以下链接下载：[http://www.forum.nokia.com/info/sw.nokia.com/id/d476061e-90ca-42e9-b3ea-1a852f3808ec/SIP%5FVoIP%5FSettings.html](http://www.forum.nokia.com/info/sw.nokia.com/id/d476061e-90ca-42e9-b3ea-1a852f3808ec/SIP%5FVoIP%5FSettings.html)
