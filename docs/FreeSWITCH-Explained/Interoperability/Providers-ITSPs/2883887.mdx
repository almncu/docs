# Provider VoIPstreet (USA)

## [voipstreet.com](http://voipstreet.com)

最近更新：2008年6月18日

呼入：

```xml
<include>
    <gateway name="chi-in.voipstreet.com">
      <param name="username" value="用户名"/>
      <param name="password" value="密码"/>
      <param name="from-user" value="用户名"/>
      <param name="proxy" value="chi-in.voipstreet.com"/>
      <param name="expire-seconds" value="600"/>
      <param name="register" value="true"/>
      <param name="retry-seconds" value="30"/>
      <param name="context" value="公共"/>
    </gateway>
 </include>
```

呼出：

```xml
<include>
    <gateway name="chi-out.voipstreet.com">
      <param name="username" value="用户名"/>
      <param name="password" value="密码"/>
      <param name="from-user" value="用户名"/>
      <param name="proxy" value="chi-out.voipstreet.com"/>
      <param name="expire-seconds" value="600"/>
      <param name="register" value="true"/>
      <param name="retry-seconds" value="30"/>
      <param name="context" value="公共"/>
    </gateway>
 </include>>
```

  
01/15/09: 为了在呼出电话中设置呼叫者ID，我必须在我的呼出网关中添加以下内容：

```xml
<param name="caller-id-in-from" value="true"/>
```
