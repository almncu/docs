# Localphone 服务供应商（英国）

## [Localphone.com](http://Localphone.com)

最后更新：2010年6月20日

**注意：** localphone 不接受网关名称中的点号。否则会出现错误；在允许的联系人中注册失败，状态为无服务器地址 \[476\]

/conf/sip\_profiles/external/localphone.xml

```xml
<include>
<gateway name="localphone">
<param name="username" value="SIP ID"/>
<param name="from-domain" value="localphone.com"/>
<param name="password" value="SIP Password"/>

<param name="caller-id-in-from" value="false"/>

<param name="proxy" value="proxy.localphone.com"/>
<param name="expire-seconds" value="3600"/>
<param name="register" value="true"/>
<param name="retry_seconds" value="30"/>
</gateway>
</include> 
```
