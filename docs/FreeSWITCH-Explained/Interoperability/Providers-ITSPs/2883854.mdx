# 供应商 Teliax（美国）

## [www.teliax.com](https://www.teliax.com)

最后更新：2009年6月16日

适用于呼入和呼出 SIP。

conf/sip\_profiles/external/teliax.xml - 您可能需要将您的域改为除了 [atl.teliax.net](http://atl.teliax.net) 之外的其他 Teliax 服务器。

```xml
<include>
     <gateway name="teliax">
        <param name="username" value="您的用户名"/>
        <!-- 在此处添加您的 teliax 域名，按照 teliax 网站的说明进行操作 -->
        <param name="realm" value="atl.teliax.net"/>
        <param name="password" value="您的密码"/>
    </gateway>
</include>
```
