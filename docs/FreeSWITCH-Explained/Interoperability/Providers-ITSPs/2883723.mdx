# IPKall供应商（美国）

## [IPKall.com](http://IPKall.com)

最新更新：2009年6月18日

- 仅限呼入
- 我们不注册，他们直接发送到公共拨号方案
- 在他们的Web配置中，您必须指向您的服务器的5080端口
- 域名：[freeswitch.org](http://freeswitch.org):5080（如果您没有更改vars.xml，则为IP:PORT）
- 在Web配置中为SIP号码，您可以决定他人拨打的SIP号码
- 我将使用5150作为我的SIP号码
- 然后在dialplan/public.xml（或一个包含的文件）中：

```xml
<include>
  <extension name="IPKALL">
    <condition field="destination_number" expression="^5150$">
      <action application="set" data="domain_name=freeswitch.org"/>
      <action application="transfer" data="1001 XML default"/>
    </condition>
  </extension>
</include> 
```
