# CallWithUs 供应商（美国）

## [CallWithUs.com](http://CallWithUs.com)

最新更新日期：2009年11月8日

```xml
<include>
    <gateway name="callwithus-outbound">
        <param name="username" value="USERNAME"/>
        <param name="password" value="PASSWORD"/>
        <param name="realm" value="callwithus"/>
        <param name="proxy" value="sip.callwithus.com"/>
        <param name="register" value="true"/>
    </gateway>
</include> 
```

你可以在上述网站的账户管理门户的"VoIP 账户"部分检查 SIP 或 IAX 注册是否成功。请验证相应列中是否显示像"FreeSWITCH-mod_sofia/1.0.4-exported on sip"这样的文本。如果显示为"未注册"，请重新检查你的配置。如果你的 ISP 阻止了 5060 端口，并且你在 FreeSWITCH 配置中指定了备用端口，在此界面上连接的账户上选择"编辑"按钮将允许你指定备用端口。

为了防止呼叫计划错误导致费用昂贵的错误呼叫，你可以在网站的"用户设置"部分选择每分钟最高呼叫费率。

请参考 FreeSWITCH 文档以了解如何使用这个新配置的网关。
