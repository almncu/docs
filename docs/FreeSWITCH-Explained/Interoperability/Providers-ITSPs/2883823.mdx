# 服务提供商 PennyTel（澳大利亚）

## [www.pennytel.com.au](https://www.pennytel.com.au)

最后更新日期：2012年9月10日

呼入和呼出电话正常工作。在 conf/sip\_profiles/outbound/ 目录下创建一个名为 pennytel.xml 的文件，并将以下内容填入：

```xml
<include>
  <gateway name="PennyTel">
    <param name="username" value="61xxxxxxxxx"/>
    <param name="password" value="xxxxxxxx"/>
    <param name="realm" value="sip.pennytel.com"/>
    <param name="proxy" value="sip.pennytel.com"/>
    <param name="register" value="true"/>
  </gateway>
</include>
```
