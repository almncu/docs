# 服务提供商PeopleLine（加拿大）

## [www.peopleline.net](http://www.peopleline.net)

最近更新：2012年1月19日（已测试至2010年的最新更新）

Peopleline支持BYOD（自带设备），如果您告知他们您的情况，他们会提供以下信息：

显示名称：604XXXXXXX

用户名：604XXXXXXX

密码：password

授权用户名：604YYYYYYY（通常与用户名相同）

域名：[rmd.ucantalk.net](http://rmd.ucantalk.net)（根据您所在的大温地区不同，域名会有所变化，例如rmd、gs、gl、gk等）

出站代理：[op.ucantalk.net](http://op.ucantalk.net):5062

```xml
<include>
 <gateway name="peopleline">
 <param name="username" value="604XXXXXXX"/>
 <param name="from-user" value="604XXXXXXX"/>
 <param name="from-domain" value="rmd.ucantalk.net"/>
 <param name="extension" value="604XXXXXXX"/>
 <param name="auth-username" value="604YYYYYYY"/>
 <param name="password" value="password"/>
 <param name="proxy" value="rmd.ucantalk.net"/>
 <param name="register-proxy" value="rmd.ucantalk.net"/>
 <param name="outbound-proxy" value="op.ucantalk.net:5062"/>
 <param name="expire-seconds" value="1800"/>
 <param name="register" value="true"/>
 <param name="register-transport" value="udp"/>
 <param name="retry-seconds" value="60"/>
 <param name="caller-id-in-from" value="false"/>
 <param name="ping" value="60"/>
 </gateway>
</include>
```
