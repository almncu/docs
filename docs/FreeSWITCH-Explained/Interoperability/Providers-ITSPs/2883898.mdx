# Provider Vono (Portugal) 

## [www.vono.net.br](http://www.vono.net.br/)

最后更新日期：2011年1月11日

在external.xml文件中，插入以下行到设置中：

```xml
<param name="user-agent-string" value="FreeSWITCH Agent"/>
```

vono.xml文件：

```xml
<include>
  <gateway name="vono">
    <param name="username" value="[用户名]"/>
    <param name="realm" value="vono.net.br"/>
    <param name="password" value="[密码]"/>
    <param name="caller-id-in-from" value="false"/>
    <param name="extension" value="[用户名]"/>
    <param name="extension-in-contact" value="true"/>
    <param name="proxy" value="vono.net.br"/>
    <param name="register-proxy" value="201.86.87.2"/>
    <param name="expire-seconds" value="3600"/>
    <param name="register" value="true"/>
  </gateway>
</include>
```
