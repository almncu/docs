# 提供商VONAGE（美国）

## [www.vonage.com](http://www.vonage.com)

最新更新：2011年4月7日

* Vonage通常不提供SIP服务，但他们通过经销商提供一个名为"Vonage Business Plus"的产品，你可以很容易地通过谷歌搜索来找到一个经销商。
* 在_conf/sip\_profiles/external/vonage.xml_中添加以下内容：

```xml
 <include>
  <gateway name="sphone1.voncp.com">
    <param name="username" value="15555551212"/> 
    <param name="password" value="<Your Password>"/>
    <param name="extension-in-contact" value="true"/>
  </gateway>
</include>
```

请注意，**extension-in-contact**是必要的，以确保在Contact:头部中提供用户名。如果缺少这个参数，将会导致无法正确处理Vonage软电话代理发出的质询所带来的BYE消息，从而导致通话直到对方挂断才会结束。在一些SIP消息（比如BYE消息）中，用户名字段**必须**在Contact头部中存在。
