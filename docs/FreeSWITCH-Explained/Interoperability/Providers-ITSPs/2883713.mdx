# 供应商 iCall (美国)

## [iCall.com](http://iCall.com)

最后更新日期：2010年10月21日

- iCall运营商/批发类型帐户（非消费者）的配置
- \[[carriers.icall.com](http://carriers.icall.com/)\]
- 可与本地呼入DID号码、免费呼入号码以及呼出SIP服务一起使用
- 将以下内容放置在_conf/sip_profiles/external/icall.xml文件中：
- 使用您提供的帐户信息替换`USERNAME`和`PASSWORD`。将`cust_`作为用户名的前缀保留。
- **注意！** 自2009年6月起，iCall建议客户使用[sbc01-car.dal.us.icall.net](http://sbc01-car.dal.us.icall.net)进行**国内外拨**电话。

```xml
 <gateway name="icall">
   <param name="username" value="cust_USERNAME" />
   <param name="password" value="PASSWORD" />
   <param name="from-user" value="cust_USERNAME" />
   <param name="proxy" value="sbc01-car.dal.us.icall.net" />
   <param name="realm" value="sbc01-car.dal.us.icall.net" />
   <param name="register" value="false" />
 </gateway>
 <gateway name="icall_international">
   <param name="username" value="cust_USERNAME" />
   <param name="password" value="PASSWORD" />
   <param name="from-user" value="cust_USERNAME" />
   <param name="proxy" value="gw01-car.dal.us.icall.net" />
   <param name="realm" value="gw01-car.dal.us.icall.net" />
   <param name="register" value="false" />
 </gateway>
```
