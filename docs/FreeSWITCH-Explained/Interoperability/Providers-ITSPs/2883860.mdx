conf/sip\_profiles/external/thinktel.xml - 您应该在注册过程中（或通过客户代表或支持部门）获得SIP身份验证凭据

```xml
<include>
     <gateway name="Thinktel">
        <param name="username" value="<username>"/>
        <param name="password" value="<password>"/>
        <param name="proxy" value="<IP地址>"/>
        <param name="register" value="false"/>
        <param name="context" value="<您的上下文>"/>
    </gateway>
</include>
```
