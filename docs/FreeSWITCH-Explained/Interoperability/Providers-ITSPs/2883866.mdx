# 供应商 ViaTalk (美国)

## [www.viatalk.com](http://www.viatalk.com)

最后更新日期：2010年10月10日

**dialplan/default.xml** (以下代码段适用于11位和10位拨号)

```xml
 <extension name="长途">
   <condition field="destination_number" expression="^(1{0,1}\d{10})$">
   <action application="bridge" data="sofia/gateway/viatalk/$1"/>
   </condition>
 </extension>
```

**sip\_profiles/external/viatalk.xml**

```xml
 <include>
   <gateway name="Viatalk">
    <param name="username" value="1Xxxxxxxxxx" /> 
    <param name="password" value="XXXXXXXXX" /> 
    <param name="proxy" value="chicago-1a.vtnoc.net" /> 
    <param name="register" value="true" /> 
   </gateway>
 </include>
```

默认情况下，ViaTalk使用DTMF音频传输。要使DTMF与ViaTalk配合使用，请致电客户支持并要求将您的帐户切换到[RFC 2833](https://tools.ietf.org/html/rfc2833)。

请勿包括"realm"或"register-proxy"。
