# 提供商 Junctionnetworks (美国)

## [junctionnetworks.com](http://junctionnetworks.com)

最后更新日期：2009年6月18日

与传入和传出的 SIP（Junction的PSTN网关服务）一起使用。

_/conf/directory/default/junctionnetworks.xml_

```xml
<include>
         <gateway name="junctionnetworks">
           <param name="username" value="在此处输入您的JUNCTION用户名"/>
           <param name="realm" value="jnctn.net"/>
           <param name="password" value="在此处输入您的JUNCTION VOIP密码"/>
           <param name="extension" value="在此处输入您的JUNCTION DID"/>
           <param name="proxy" value="sip.jnctn.net"/>
           <param name="expire-seconds" value="3600"/>
         </gateway>
      </include> 
```
