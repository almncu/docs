# 提供商 iptel （德国&全球）



## [iptel.org](http://iptel.org)

最后更新：2009年10月16日

我已经对这种情景进行了基本测试。这与我撰写我的文章“使用FreeSWITCH进行VoIP”时使用的设置相同： <http://bit.ly/EpVrv>

```xml
<include>
  <gateway name="iptel"> <!--用于拨号字符串-->
    <param name="username" value="MY_IPTEL_USERNAME"/>
    <param name="realm" value="iptel.org"/>
    <param name="password" value="MY_IPTEL_PASSWORD"/>
    <param name="proxy" value="sip.iptel.org"/>
  </gateway>
</include> 
```
