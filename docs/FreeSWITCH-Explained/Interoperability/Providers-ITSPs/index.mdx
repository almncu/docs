# 供应商 ITSPs

## 关于

本章节提供了如何将FreeSWITCH连接到各种互联网电话服务提供商的示例。

如果你添加或更改某一页，请遵守以下规则！

* 如果你更新了某个供应商的页面，请删除页面顶部的更新说明。这是用来将旧维基中的最后更新日期带到新页面中的，当页面被更新时不再需要这个说明。
* 包括一个通用的配置。
* 不要包含市场营销信息。
* 在名称旁边包含总部所在国家。

xx

每个供应商的名称旁边都标明了其总部所在国家。在许多情况下，这些供应商涉及多个国家。
