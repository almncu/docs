# InPhonex服务提供商（美国，巴西，墨西哥）

## [Inphonex.com](http://Inphonex.com)

最近更新：2010年1月21日

在`conf/sip_profiles/external`目录下创建`inphonex.xml`文件，并添加以下内容：

```xml
<include>
  <gateway name="Inphonex">
    <param name="username" value="YOUR INPHONEX VIRTUAL NUMBER"/>
    <param name="password" value="YOUR INPHONEX VIRTUAL NUMBER PASSWORD"/>
    <param name="proxy" value="sip.inphonex.com"/>
    <param name="from-domain" value="sip.inphonex.com"/>
    <param name="expire-seconds" value="3600"/>
    <param name="register" value="true"/>
  </gateway>
</include> 
```
