# Provider: Halonet (Poland)

## [Halonet.pl](http://Halonet.pl)

最后更新日期：2009年6月18日

Halonet要求以以下格式使用Remote-Party-ID： _&lt;sip:account@[sip.halonet.pl](http://sip.halonet.pl)\>_ ，或者根本不使用此头部。要删除此头部，请在您的拨号计划中的_bridge_之前插入以下代码：

```xml
<action application="set" data="sip_cid_type=none"/> 
```

将USERNAME、PASSWORD和EXTENSION替换为提供的账户信息。

```xml
<include>
  <gateway name="halonet">
    <param name="username" value="USERNAME"/>
    <param name="extension" value="EXTENSION"/>
    <param name="password" value="PASSWORD"/>
    <param name="realm" value="sip.halonet.pl"/>
    <param name="proxy" value="sip.halonet.pl"/>
    <param name="expire-seconds" value="3600"/>
    <param name="register" value="true"/>
    <param name="register-transport" value="udp"/>
    <param name="retry-seconds" value="30"/>
  </gateway>
</include> 
```
