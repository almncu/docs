# DIDLogic供应商（全球）

## [DIDLogic.com](http://DIDLogic.com)

最后更新时间：2014年4月4日

Freeswitch配置以使用[DIDlogic.com](http://DIDlogic.com)发送SIP呼叫：<https://didlogic.com/support/setup-guides/freeswitch-config>

```xml
<include>
 <gateway name="didlogic-out">
  <param name="username" value="YOUR_SIP_USERNAME"/>  <!--使用DIDLogic提供的SIP用户名-->
  <param name="realm" value="sip.ca.didlogic.net"/>
  <param name="password" value="YOUR_SIP_PASSWORD"/>  <!--使用为SIP帐户设置的密码-->
  <param name="register" value="true"/>
  <param name="context" value="public"/>
 </gateway>
</include>
```
