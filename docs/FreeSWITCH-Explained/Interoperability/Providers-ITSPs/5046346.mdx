# 提供商 FPT 电信（越南）

请按照以下步骤连接到FPT SIP中继：

将以下列表添加到conf/autoload_configs/acl.conf.xml中的ACL配置中：

```xml
 <list name="fpt_sip_trunk" default="deny">
        <node type="allow" cidr="SIP_SERVER_IP/32"/>
        <node type="allow" cidr="MEDIA_SERVER_IP/32"/>
 </list>
```

FPT使用两个服务器进行信令和媒体传输：SIP_SERVER_IP是信令服务器的IP地址，MEDIA_SERVER_IP是媒体服务器的IP地址。

将通话桥接到FPT：

```xml
 
<action application="bridge" data="sofia/external/${dialed_number}@$${sip_trunk_ip_fpt}" />

```

${dialed_number}：越南目标号码

$${sip_trunk_ip_fpt}：上述ACL中的SIP_SERVER_IP地址。
