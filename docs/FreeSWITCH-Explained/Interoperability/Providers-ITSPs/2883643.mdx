# Provider Belcentrale (荷兰) 

 

## [Belcentrale.nl](http://Belcentrale.nl)

最后更新日期：2010年11月17日

在 _conf/directory/default/_ (或 default) 中创建一个名为 belcentrale.xml 的文件：

```xml
<include>
     <gateway name="belcentrale.nl">
        <param name="username" value="31XXXXXXXXX"/>
        <param name="realm" value="sip1.belcentrale.nl"/>
        <param name="password" value="your_password"/>
        <param name="register" value="true"/>
    </gateway>
</include> 
```

然后在 sip\_profiles/nat.xml (或 default.xml，不使用 NAT 时) 中编辑 auth\_calls 的设置，将其设置为 false：

```xml
<param name="auth-calls" value="false"/>
```

RFC2833 似乎无法使用 DTMF。必须将电话设置为使用音频中的 DTMF。
