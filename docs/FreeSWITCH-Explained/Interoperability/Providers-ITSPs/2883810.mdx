# Provider NetSIP（澳大利亚）

## [NetSIP.com.au](http://netsip.com.au)

最后更新日期：2010年6月7日

拨号格式可采用E.164和0NSN，来电号码可以以E.164格式呈现并根据用户需要进行筛选。

NetSIP支持以下功能：

- SIP中继。
- 私有线路。
- 当地号码携带。
- 1300号码托管。
- 呼叫方ID呈现。
- 将传真发送至任何分机号码的电子邮件。

呼入和呼出电话均可操作。在conf/sip_profiles/outbound/中创建一个名为netsip.xml的文件，内容如下：

```xml
<include>
  <gateway name="netsip">
    <param name="username" value="****SIP_Account_ID****"/>
    <param name="password" value="****SIP_Account_Password****"/>
    <param name="from-user" value="****SIP_Account_ID****"/>
    <param name="extension-in-contact" value="****SIP_Account_ID****"/>
    <param name="from-domain" value="sip.netsip.com.au"/>
    <param name="realm" value="sip.netsip.com.au"/>
    <param name="proxy" value="sip.netsip.com.au"/>
    <param name="register" value="true"/>
  </gateway>
</include>
```
