# 提供者 Gafachi (美国) 

 

## [Gafachi.com](http://Gafachi.com)

最近更新日期：2010年9月30日

身份验证ID和密钥提供在您的帐户内，登录到主菜单->配置信息（不是用于登录到您的帐户的那些）的选项下。

/conf/sip\_profiles/external/gafachi.xml

```xml
 <include>
  <gateway name="sip.gafachi.com">
    <param name="username" value="<Authentication ID>"/>
    <param name="password" value="<Secret>"/>
    <param name="register" value="true"/>
  </gateway>
</include>
```
您可能还需要提供领域和/或代理（ping [sip.gafachi.com](http://sip.gafachi.com) 以找出要使用的IP）

```xml
    <param name="realm" value="64.192.112.13"/>
    <param name="proxy" value="64.192.112.13:5060"/> 
```

Gafachi 要求您每次请求时都必须提供有效的呼叫者ID，否则通话将返回604错误。例如：

```xml
[effective_caller_id_number=YOUR_NUMBER,effective_caller_id_name=YOUR_NAME]/sofia/gateway/sip.gafachi.com 
```
