# 提供商 Phonzo（挪威）

## [www.phonzo.com](http://www.phonzo.com)

最后更新时间：2010年4月22日

phonzo 需要用户名和身份验证-用户名将是您的DID号码。

呼入和呼出电话可以正常工作。在conf/sip_profiles/external/下创建一个名为phonzo.xml的文件，并添加以下内容：

```xml
<include>
  <gateway name="Phonzo">
    <param name="username" value="10xxxxxxx"/>
    <param name="password" value="xxxxxxx"/>
    <param name="realm" value="sip.phonzo.com"/>
    <param name="proxy" value="sip.phonzo.com"/>
    <param name="auth-username" value="4610xxxxxxx"/>
    <param name="from-domain" value="sip.phonzo.com"/>
    <param name="register" value="true"/>
  </gateway>
</include>
```
