# Iristel 服务提供商（加拿大）

## [IrisTel.com](http://IrisTel.com)

最近更新：2010年11月17日

这是一个简单的配置，可以让你使用 Iristel Broadfone 服务与 FreeSWITCH 进行配合。

注意：Iristel 不接受任何来电显示号码，应使用 Iristel 提供的加拿大号码。

网关配置 /conf/sip\_profiles/external/iristel.xml

```xml
<include>
  <gateway name="iristel"> <!-- 在拨号字符串中使用 -->
    <param name="username" value="你的加拿大号码"/>
    <param name="auth-username" value="你的认证用户名，不是加拿大号码"/> 
    <param name="realm" value="iristel.net"/> <!-- iristel 域名 -->
    <param name="from-user" value="你的加拿大号码"/>
    <param name="from-domain" value="iristel.net"/> <!-- 来电显示的 iristel 域名 -->
    <param name="proxy" value="sbc1.iristel.net"/> <!-- Iristel SBC -->
    <param name="password" value="密码"/>
    <param name="extension-in-contact" value="true"/> <!-- 可以设置为 true，因为 iris 需要联系 yournumber@iristel.com -->
    <param name="expire-seconds" value="120"/>
  </gateway>
</include> 
```
