## Skype互连选项

- [Skype Connect](./Skype-Connect_13173678.mdx#about): 现在Skype通过Skype Connect提供Skype for SIP。这提供了与FreeSWITCH的最佳集成，但有一些限制：Skype Connect仅适用于企业，不允许转售；Skype Connect能处理来自Skype用户的呼入呼叫，但只能呼叫固定电话。
- [mod_skypopen](../../../Modules/mod_skypopen_3966336.mdx#-sk-): Skypopen（以前的skypiax）是一个端点（通道驱动程序），它使用Skype客户端作为与Skype网络的接口，并允许双向通话连接FreeSWITCH（可以像所有其他端点（例如sofia/SIP）一样进行呼入、呼出、应答等操作）。功能包括多条线路、多个并行呼入和呼出通话，适用于Windows和Linux系统。
- [SipTheeSkype](./SipTheeSkype-Skype-Adapter_1967135.mdx#usage): SipToSIS作为附加适配器与运行在同一台机器上的Skype客户端互连到FreeSWITCH。该适配器充当一个单线程的客户端用户代理，可以像其他用户代理一样发送和接收呼叫。SipTheeSkype还允许在两个方向上进行一些基本的切换功能。
