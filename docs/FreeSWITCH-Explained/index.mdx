## FreeSWITCH 是什么？

FreeSWITCH™ 是一个开源的运营级电话平台，它作为一个回路用户代理实现。因此，它可以执行许多不同的任务，从 PBX 到传输交换机，TTS（文本到语音）转换，音频和视频会议主机，甚至是 VoIP 电话等等。

## Confluence Wiki

这个在线文档 Wiki 将与 FreeSWITCH™ 核心开发团队的持续开发并行进行，同时也接受开源用户社区的贡献。这个在线参考文献将始终跟踪 Git 仓库中最新的可用代码版本（在您的帮助下）。

该维基提供了关于频道变量、拨号计划应用和API命令的用法原型和示例，可以通过使用Lua、Perl和其他语言编写的脚本通过E.S.L.（事件套接字层）访问。带有标签"[examples](https://freeswitch.org/confluence/label/FREESWITCH/examples)"的页面提供了配置和试验FreeSWITCH的参考信息。务必单击标签以查找与该主题相关的页面。

核心API函数的参考文档可以在[FreeSWITCH技术参考](http://docs.freeswitch.org/)上找到。
