# PDD（Post Dial Delay） 

 

## 关于

本文档介绍了PDD的相关信息。

点击此处展开目录

拨号后延迟（PDD）是指从呼叫开始（INVITE）到出现进展指示（183或早期媒体）之间的时间。

您可以找到设置为通道变量的延迟时间：

_**以秒为单位：**_

* progress\_mediasec
* progresssec

_**以毫秒为单位（1000 = 1秒）：**_ 

* 如果有媒体，则为**progress\_mediamsec**
* 如果没有媒体，则为**progressmsec**

_**以微秒为单位（1000 = 1毫秒）**_ 

* progress\_mediausec
* progressusec

其中一个将被设置为非零值，另一个将被设置为0。以秒为单位的PDD = (progress\_mediamsec + progressmsec) / 1000
