# 在Irssi中使用多个服务器

## 关于

对于那些想在离线时跟踪聊天记录的人，可以在Linux主机上的screen中运行irssi。然后您可以从任何地方断开和重新连接您的screen（其中包含Irssi）。

## Irssi

在Irssi中使用“正常”的IRC服务器并添加一个额外的服务器非常简单。只需输入命令`/server irc.freenode.org`，您就会有额外的连接。然后，您可以给您的加入标签：

```xml
/join -freenode #freeswitch
```

在您的.irssi目录中的启动配置文件中自动加入服务器和频道：

```xml
...
servers = (
  {
    address = "irc.your_normal.net";
    port = "6667";
    use_ssl = "no";
    ssl_verify = "no";
    autoconnect = "yes";
    chatnet = "IRCnet";
  }
  {
    address = "irc.freenode.net";
    port = "6667";
    use_ssl = "no";
    ssl_verify = "no";
    autoconnect = "yes";
    chatnet = "freenode";
  }
);

channels = (
  {
    name = "#freeswitch";
    chatnet = "freenode";
    autojoin = "yes";
  }
);
```
