## Nagios

kjhosein编写了一个插件（使用Perl语言），用于检查FreeSWITCH服务器上的各种健康参数。它利用了\*fs\_cli\*FreeSWITCH命令行工具。它可以扩展到几乎可以检查fs\_cli可以检查的任何内容。

它目前监控以下几个方面：

* sofia-status-internal - 寻找“internal”名称，并期望找到一个RUNNING状态。与上述“internal”测试格式相同。
* sofia-status-external - 寻找“external”名称，并期望找到一个RUNNING状态。与上述“internal”测试格式相同。
* show-calls-count - 报告当前呼叫总数。
* sofia-status-profile-internal-failed-calls-in - 报告“sofia status profile internal”查询中的FAILED-CALLS-IN参数。
* sofia-status-profile-internal-failed-calls-out - 报告“sofia status profile internal”查询中的FAILED-CALLS-OUT参数。

您可以从GitHub获取它和更多信息：[https://github.com/kjhosein/nagios-freeswitch-plugin](https://github.com/kjhosein/nagios-freeswitch-plugin)

它需要Nagios::Plugin Perl模块（由Ton Voon编写）。
