# FreeSWITCH GUI (图形用户界面）

## 关于

FreeSWITCH GUI 可以使用 XML-RPC、ESL、Erlang 和其他几种通信方法与 FreeSWITCH 进行通信。其中一个强大的方法是非常实用的 ESL 库，该库已翻译成多种语言，包括 Perl、Lua、Python、Ruby、.NET 等。

FreeSWITCH 有几个可用的图形用户界面。下面列出了其中几个。它们可以帮助管理 XML 配置文件和 FreeSWITCH 操作本身。所有这些界面都依赖于 FreeSWITCH 安装（因此您可能还是应该熟悉上面的说明）。

## FreeSWITCH 开源 GUI 解决方案
### Blue.box

blue.box 是一个模块化、高度可伸缩、面向对象、多平台、多操作系统的 Web 接口，用于管理 FreeSWITCH。它由 FreeSWITCH mod_nibblebill 模块的开发人员创建。该平台通过一个简单友好的图形用户界面，允许管理整个 FreeSWITCH 系统，并且不需要了解底层的 XML 配置结构。我们非常注重以模块和插件的形式提供未来扩展的可能性。

请注意，尽管我们支持多个交换引擎，但该项目重点关注 FreeSWITCH，并提供专门支持 FreeSWITCH 的模块，以实现高级功能。

状态: 废弃

* 网站: <https://github.com/2600hz/bluebox>
* IRC: #2600hz on freenode

### FusionPBX

FusionPBX 是一个功能丰富、多平台、高度可定制、可伸缩和快速的 Web 接口，用于将 FreeSWITCH 作为 PBX 或语音交换机进行管理。该项目最初是在 pfSense 上的 FreeSWITCH 包。为了使其适用于多个操作系统，我们进行了开发，包括各种版本的 Linux、BSD、Windows、Mac OS X 和其他操作系统。数据存储从 pfSense 的 XML 数据存储转移到了 PHP Data Objects（PDO），它提供了使用 SQLite、PostgreSQL、MySQL 和其他数据存储引擎的能力。它可以使用任何支持 PHP5 的 Web 服务器，包括 Apache、Lighttpd、nginx、IIS 等。

FusionPBX当前拥有48个额外的模块（也称为软件包），提供了很多功能。一些功能的示例包括但不限于活动呼叫、活动分机、交互式会议界面、交互式队列、具有XML CDR的非常详细的通话记录、自动电话配置、自动接听功能的IVR菜单、猎头组、呼叫转移、DISA、时间条件、拨号计划编辑器、SIP网关管理器、分机管理、用户管理、联系人管理、内容和菜单管理等。开发活跃。要了解更多信息，请使用下面的链接。

* 许可证：MPL 1.1
* 网站：[fusionpbx.com](http://fusionpbx.com)
* 截图：[fusionpbx.com/files/fusionpbx%5Fcom/screenshots/index.php](http://fusionpbx.com/files/fusionpbx%5Fcom/screenshots/index.php)
* 下载：[fusionpbx.com/download.php](http://fusionpbx.com/download.php)
* IRC: freenode上的#fusionpbx

### CDR-Stats

针对后付费计费应用程序的电信调解、评分引擎和呼叫数据记录分析，支持多种CDR数据格式的不同交换机类型的连接器。

* 许可证：MPL V2
* 网站：[cdr-stats.org](http://www.cdr-stats.org/)
* 代码：[github.com/cdr-stats/cdr-stats](https://github.com/cdr-stats/cdr-stats)

### FreePyBX

FreePyBX是一种基于MPL 2.0开源许可的功能丰富的GUI实现，用于FreeSWITCH。用户友好的直观界面使您的客户可以免费享受非常昂贵的专有软件所具有的所有功能和稳定性。它是用Python和Pylons框架编写的，呼叫控制主要使用Lua，但目录、配置和拨号方案也是XML cURL。它是一个ITSP多租户系统，支持许多功能，如呼叫中心队列、一键呼叫、文本转语音等等。它使用的一些技术包括dojo、ajax，并具有与JSON的接口，可以轻松用于第三方API、云API或遗留系统和关系数据库。代码易于理解，并采用了模块化的编写方式，以便与第三方模块和自定义开发轻松集成。这是ITSP电话框架的绝佳选择。

它利用单个配置文件和单个IP和端口，非常适合托管式PBX ITSP提供商。

开发非常活跃，目前仅支持FreeSWITCH，并没有计划将其移植到其他交换机。

* 网站：[http://www.freepybx.org/](http://www.freepybx.org/)
* 客户演示：[https://demo.freepybx.org/login](https://demo.freepybx.org/login)
* 管理员演示：[https://demo.freepybx.org/admin/login](https://demo.freepybx.org/admin/login)

### SimpleCOS

SimpleCOS 是一个基于 RoR 的GUI计费应用程序，用于管理多个freeswitch终端，通过 mod\_xml\_curl 和 mod\_nibble\_curl 进行维护。

* 网站： <https://github.com/bit4bit/simplecos>
* 许可证： GPL v3

### FSGui

FSGui 是一个基于 Qt 的FreeSWITCH界面，用于管理FreeSWITCH，并监视呼叫和通道。它使用ESL开发，并包含在freeswitch git仓库中。

在[Fs gui](./Fs-gui_13173087.mdx#fsgui)子页面上阅读更多内容。

### FreeSWITCHed

基于Web的PHP实用工具，用于查看分机、呼叫、会议和FreeTDM通道。

它还允许执行多个维护命令。

该项目基于ESL运行，似乎旨在作为管理员/ HUD界面，不应暴露给最终用户。

* 网站： <https://sites.google.com/site/freeswitched/home>
* 代码： <https://github.com/Slonik/FreeSWITCH-panel>
* 公告： [FreeSWITCH邮件列表 2012年3月19日](http://lists.freeswitch.org/pipermail/freeswitch-users/2012-March/081894.html)

### FreeSWITCHConfig

基于Web的.Net程序，用于配置/控制freeswitch服务器。

它允许创建分机，设置内部和外部SIP网关以及一些基本的路由。通过实现IModule类来创建基于站点的模块以实现新的路由方法、安全性和其他所需的功能。该项目正在寻找熟练的C#或javascript开发人员协助开发。

* 网站地址: <http://code.google.com/p/freeswitchconfig/>
* SVN源码: <http://freeswitchconfig.googlecode.com/svn/trunk/>
* 演示站点: <http://freeswitchconfig.dalnet.ca:8080> 用户名: admin, 密码: admin
