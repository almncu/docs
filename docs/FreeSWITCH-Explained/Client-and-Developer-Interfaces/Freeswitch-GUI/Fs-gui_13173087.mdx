## FsGui

### 描述

FsGui 是一个可扩展的通话控制应用平台。它广泛使用 [Qt](http://www.qtsoftware.com/) 插件框架和 lib [ESL](../Event-Socket-Library/index.mdx#-new-)，FsGui 的目的是为了让应用程序能够在桌面上使用通话控制功能，如会议管理、点击呼叫、Outlook 集成等。

目前正在开发的第一个应用程序是 [#Console - 一个人性化的 FreeSWITCH 控制台](Fs-gui_13173087.mdx#console---a-freeswitch-console-for-human-beings)。

### 应用

#### 控制台 - 一个人性化的 FreeSWITCH 控制台

控制台是在新的 FsGui 结构中开发的第一个应用程序。它将显示在 fs\_cli 和 cli 中显示的信息，界面类似于 Wireshark 对 IP 数据包转储的界面。

控制台具有服务器管理工具，允许您管理多个 FreeSWITCH 实例，并且拥有选项卡界面，可同时连接到多个实例。

控制台还允许您快速执行常见任务，如将日志粘贴到 pastebin、将日志保存到磁盘或搜索现有日志。

计划在接下来的几周内发布版本。

### 平台

FsGUI 基于 [Qt](http://www.qtsoftware.com/) 和 Tango 图标集，并且可在 Windows、Linux 和 macOS 上运行。其他平台需要进行测试。欢迎提供反馈。
