# Lua工具包Jester


## 关于

Jester是用Lua编写的FreeSWITCH脚本工具包。其功能范围位于XML拨号计划、IVR菜单和自定义脚本之间。Jester的目标是通过提供一种简单而统一的方式来实现更复杂的特性，以便轻松开发语音工作流程，而这些特性通常需要自定义脚本。

## 架构

Jester的设计理念是小巧、简单和可扩展的。核心代码不足800行！大部分用户任务都是通过可插拔的模块来完成的，熟悉Lua脚本的人会发现添加新模块以扩展功能非常容易。终端用户不需要编写完整的脚本，而是在称为“序列”的脚本模板中工作，通过向底层模块传递带参数的命令来处理所有繁琐的工作。

## 喜剧演员邮件副本

Jester的默认配置是Asterisk的喜剧演员邮件系统的复制版。那些从Asterisk转换到FreeSWITCH并且对两个语音邮件系统之间的差异有所担忧的人可以利用它来为终端用户提供无缝过渡。
