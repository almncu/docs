## Java连接选项

至少有三种选项可用于使用Java连接到FreeSWITCH[Event Socket](../Event-Socket-Library/index.mdx#-new-)。

#### esl.jar（freeswitch核心）

* 许可证：与FreeSWITCH相同
* 可用：esl.jar可以作为FreeSWITCH代码库的一部分进行构建
* 特点：  
   * 由SWIG从核心C ESL接口自动生成（因此根据定义，始终与核心ESL保持最新）。  
   * 运行时需要可用的本机esl\_lib依赖项。  
   * 此页面底部提供了一个示例[Event Socket Library](../Event-Socket-Library/index.mdx#-new-).

#### org.freeswitch.esl.client（freeswitch-contrib）

* 许可证：Apache v2
* 可用：在freeswitch-contrib git存储库中，有关详细信息和使用方式，请参阅[Java ESL Client](Java-ESL-Client_7144076.mdx#about)。
* 特点：  
   * 入站和出站  
   * 处理事件主体  
   * 使用maven构建，支持OSGi  
   * 仅依赖于出色的[Netty](http://www.jboss.org/netty)（Apache许可证）和[SLF4j](http://www.slf4j.org)（MIT许可证）库

现在Java ESL Client可以在Github上找到并进行更新 <https://github.com/esl-client/> ，欢迎您加入我们。 ^\_^

#### fs_eslib（Versafon）

* 许可证：GPL v3
* 可用于：<http://versafon.com/versafonweb/Software.jsp>
* 类型：入站
* 特点：请查看上述网页。
