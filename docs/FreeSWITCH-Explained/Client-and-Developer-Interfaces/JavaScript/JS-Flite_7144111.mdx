# JS Flite（JS飞飞）

## 关于

这是一个示例，展示如何调用Flite的文本到语音功能。

## 示例

**Flite示例**

```js
if (session.ready()) {
    session.answer();
    session.execute("set", "tts_engine=flite");
    session.execute("set", "tts_voice=kal");
    session.execute("speak", "这是Flite文本到语音引擎的示例。");
    session.execute("set", "playback_terminators=#");
    session.hangup();
}
```
