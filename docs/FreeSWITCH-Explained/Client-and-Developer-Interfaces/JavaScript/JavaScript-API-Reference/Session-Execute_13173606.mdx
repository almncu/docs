## 描述

**执行拨号计划应用程序。**  
(要执行API（控制台）命令，请使用[apiExecute](./index.mdx#file)。)

## 

## 概要

* execute(appname,arguments)
* appname是要执行的应用程序的名称
* arguments是传递给应用程序的参数

## 参数

参数变量是一个字符串，如果多个参数需要打包到这个字符串中，那么具体应用程序使用的分隔符由具体应用程序决定。然而，语法与在[XML拨号计划](../../../Dialplan/XML-Dialplan/index.mdx#about)中使用应用程序属性的动作标签相同。

## 示例

#### 设置主叫号码

```js
session.execute("set","effective_caller_id_number=4141231234");
```

**桥接**

```js
session.execute("bridge","sofia/mydomain.com/foo@bar");
```

**记录会话（通话）**

```js
session.execute("record_session", "/tmp/foo.wav");
```

**执行分机**

```js
session.execute("execute_extension", "100 XML default");
```

#### 延时

```js
session.execute("sleep", "2000");
```

**聊天权限：加入会议时静音**
