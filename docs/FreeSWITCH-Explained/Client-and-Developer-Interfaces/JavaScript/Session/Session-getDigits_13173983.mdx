# Session getDigits

关于

从指定通道获取DTMF。数字将排队等候，直到您调用此方法。如果您希望在调用此方法时确保没有数字在队列中，请调用flushDigits。

## 概览

```xml
getDigits(length, terminators, timeout, digit_timeout, abs_timeout)
```

* length - 要读取的数字个数
* terminators - 终止输入的数字字符串 (\*#)
* timeout - 第一个数字超时时间（以毫秒为单位）
* digit_timeout - 数字之间的超时时间（以毫秒为单位）
* abs_timeout - 绝对超时时间（以毫秒为单位）

如果只指定了第一个超时时间，它将被视为第一个数字超时时间和数字超时时间。或者，也可以明确指定数字超时时间。无论按键如何，绝对超时时间都会覆盖其他超时时间。

所有超时时间都以毫秒为单位指定。

## 示例

```xml
var dtmf;
dtmf = session.getDigits(5, "#", 40000, 20000, 20000);
```

```xml
var dtmf = '12';
dtmf += session.getDigits(3, "", 5000, 10000, 20000);
```

```xml
var dtmf;
dtmf = session.getDigits(1, "", 10000);
```
