# 销毁会话

Javascript 与 FreeSWITCH 会话绑定，在调用会话析构函数之前，会话线程将被阻塞。session.destroy() 方法会将当前会话与其Javascript对象分离，实际上使其消失，稍后会被回收。

## 概览

```xml
session.destroy([hangup_cause]);
```

###   
参数

* hangup\_cause（可选）

## 示例

```xml
session.destroy();
session = undefined;
```
