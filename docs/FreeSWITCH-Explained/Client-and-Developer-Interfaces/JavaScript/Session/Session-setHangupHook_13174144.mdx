# Session setHangupHook

## 关于

设置挂断回调处理函数，当呼叫终止时将调用该函数。

如果您希望在挂断回调处理程序中终止JavaScript会话，请记得调用exit()。

* 1 [用法](#-freeswitch--session-sethanguphook-)

* 3 [参见](#see-also)

## 用法

```xml
result = session.setHangupHook(hangup_function_name);
```

**注意:** 必须在调用上述代码行之前声明挂断函数，否则您的挂断钩子将不会被触发。
