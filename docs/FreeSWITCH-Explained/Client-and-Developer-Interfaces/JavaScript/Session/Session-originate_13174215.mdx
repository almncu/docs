＃会话发起

使用现有会话对象发起新呼叫。

警告

已废弃

确实！阿维·马库斯无法弄清楚如何让它工作，所以他使用以下方法访问API originate：

```xml
api = freeswitch.API()
api:execute("originate", "sofia/gateway/mygateway/123456789 9664")
```

注意：如果没有其他选项，可以使用“session.originate”发起呼叫。您可以使用会话构造函数发起呼叫：

```xml
session = freeswitch.Session("sofia/gateway/mygateway/123456789")
session.transfer(9664);
```

## 用法

**5389及以后版本**

```xml
result = new_session.originate(session, dest[, timeout]);
```

**5389之前的版本**

```xml
result = new_session.originate(session, dest[[[[[[[[[[, dialplan], context], cid_name], cid_num], network_addr], ani], aniii], rdnis], username], timeou
```

- session - 一个呼叫的A leg的会话
- dest - 符合Dialplan规范的目标字符串（“sofia/sip/1234@10.10.2.55”），包括“,”和“|”参数
- dialplan - 拨号计划模块的名称（默认值为“xml”），例如xml、enum等
- context - 拨号计划中的上下文（默认值为“default”）
- cid_name - 要使用的主叫方ID名称。
- cid_num - 要使用的主叫方号码。
- network_addr - 呼叫者配置文件网络地址
- ani - 呼叫者配置文件ANI
- aniii - 呼叫者配置文件ANI-II
- rdnis - 呼叫者配置文件RDNIS
- username - 呼叫者配置文件用户名
- timeout - 超时值（秒）
