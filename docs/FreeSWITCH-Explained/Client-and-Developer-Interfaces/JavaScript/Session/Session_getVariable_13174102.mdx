# Session_getVariable

## 关于

获取指定通道上的变量。

要获取可以使用的变量名称列表，您可以使用info app来显示：

```xml
<action application="info" />
```

单击此处展开目录

* 1 [概述](#see-also)  
   * 1.1 [示例](#example)  
   * 1.2 [另请参阅](#see-also)

## 概述

```xml
getVariable(variable)
```

* variable - 包含变量名称的字符串

### 示例

```xml
Lua:
```
```xml
local caller_id = session:getVariable("caller_id_number")
```

```xml
JavaScript:
```
```xml
var caller_id = session.getVariable("caller_id_number")
```
