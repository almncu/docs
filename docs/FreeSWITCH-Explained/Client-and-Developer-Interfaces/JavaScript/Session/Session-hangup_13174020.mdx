# 会话挂断

## 关于

挂断当前通道。请勿在挂断挂钩(hangup hook)内调用此方法。

挂断方法可以使用Q.931原因码（字符串或数字表示形式）来调用，并将传输到远程端。

单击此处展开目录

* 1 [概要](#概要)
* 2 [示例](#示例)
* 3 [参见](#参见)

### 概要

```xml
hangup(cause)
```

### 示例

```xml
session.hangup("NORMAL_CLEARING");
```

或者，

```xml
session.hangup(16);
```

在SIP会话中，原因码将根据[RFC4497](http://www.ietf.org/rfc/rfc4497.txt)映射到相应的SIP响应码。
