# Perl

## 关于

在 FreeSWITCH 中，有很多种使用 Perl 的方式，并不是所有方式都需要 mod\_perl 模块或 CPAN Perl 模块。您可以使用以下方式将 Perl 与 FreeSWITCH 结合使用：

- 通过 curl 等方式执行 Perl 脚本来返回 XML 配置（仅需要 curl 模块）
- 通过 [Perl ESL](./index.mdx#about) 进行套接字控制 / 事件套接字交互（发送和接收事件，执行命令以及通过套接字交互进行 IVR）；这不需要 mod\_perl，只需要 Perl ESL 模块。
- 应用动作 / 标准执行以执行拨号计划（需要 mod\_perl）
- 本地 XML 生成（需要 mod\_perl）

如果您想要或需要 mod\_perl 提供的高级控制，请参阅 [mod\_perl](../../Modules/mod_perl_1048930.mdx#about) 页面以获得完整的文档。

## 构建/安装

安装 Perl 支持没有一种标准的方式。如上所述，您可以以多种不同的方式使用 Perl，每种方式都有不同的要求。有关构建 mod\_perl 的信息，请参阅 [mod\_perl](../../Modules/mod_perl_1048930.mdx#about) 页面。

## 脚本的执行

已弃用

您可以通过以下方式直接从控制台运行脚本：

perlrun /path/to/your/script.pl

如果您只想在拨号计划中运行 Perl 脚本（或任何常规应用程序），请调用 system 应用：

&lt;action application="system" data="/path/to/app.pl"/>

注意：FreeSWITCH™ 有一个 "scripts" 目录，您可以通过以下命令从控制台确定该目录：

eval $${script_dir}

在从拨号计划调用脚本时，您必须指定：

$${script_dir}/my_script.pl

或者脚本的完整绝对路径。
