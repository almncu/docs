## 关于

这是关于[faxlib.jm](http://faxlib.jm/)的文档，它是一种基于对象的方案，用于在FreeSWITCH中使用JavaScript（[mod_spidermonkey](https://wiki.freeswitch.org/wiki/Javascript "Javascript")）。faxlib公开了一些可供调用程序使用的方法和属性。它们在此文档中有所记录。

##   
属性

* socketaddr - 字符串 主机地址（host:port）例如 "localhost:8084"
* session - 会话对象 当前会话
* faxDetect - 布尔值 传真CNG检测状态
* faxFilePrefix - 字符串 传真文件的目录前缀（用于发送或接收）
* faxFile - 字符串 传真的文件名（位于faxFilePrefix中）
* gotFax - 布尔值 是否已接收到传真

## 方法

* stopFaxDetect - 停止传真CNG检测，接收传真时会自动调用
* startFaxDetect - 启动传真CNG检测
* getFaxFile - 返回传真文件的完整路径字符串
* rxFax - 接收传真
* txFax - 发送传真
* fax2pdf - 将传真的tiff文件转换为pdf，传真对象将从此时起指向该pdf

## 构造函数

```xml
fax(session, socketaddr [, faxFilePrefix [, faxFile]])
```

示例:

```xml
var f = new fax(session, "localhost:8084", "/tmp", "myfax.tiff");
```

最后两个参数是可选的，如果不指定，则默认为"/tmp"和"&lt;uuid>.tiff"，其中&lt;uuid>将被当前呼叫中的UUID替换。
