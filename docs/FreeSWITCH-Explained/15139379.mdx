事件有许多“类型”或“类别”。您可以根据事件类型进行过滤，以便只获取FreeSWITCH事件系统中的子集事件。

## 事件类型（名称或类别）列表

#### 事件类型（类别）

大多数FreeSWITCH事件属于以下类型（从源代码中提取）：

* ADD\_SCHEDULE（添加计划）
* API
* BACKGROUND\_JOB（后台作业）
* CALL\_DETAIL（通话详情）
* CALL\_SECURE（安全通话）
* CALL\_SETUP\_REQ（呼叫设置请求）
* CALL\_UPDATE（通话更新）
* CDR（通话记录）
* CHANNEL\_ANSWER（通道接听）
* CHANNEL\_APPLICATION（通道应用）
* CHANNEL\_BRIDGE（通道桥接）
* CHANNEL\_CALLSTATE（通道呼叫状态）
* CHANNEL\_CREATE（通道创建）
* CHANNEL\_DATA（通道数据）
* CHANNEL\_DESTROY（通道销毁）
* CHANNEL\_EXECUTE（通道执行）
* CHANNEL\_EXECUTE\_COMPLETE（通道执行完成）
* CHANNEL\_GLOBAL（通道全局）
* CHANNEL\_HANGUP（通道挂断）
* CHANNEL\_HANGUP\_COMPLETE（通道挂断完成）
* CHANNEL\_HOLD（通道保持）
* CHANNEL\_ORIGINATE（通道发起呼叫）
* CHANNEL\_OUTGOING（通道正在呼出）
* CHANNEL\_PARK（通道停泊）
* CHANNEL\_PROGRESS（通道进展）
* CHANNEL\_PROGRESS\_MEDIA（通道进展媒体）
* CHANNEL\_STATE（通道状态）
* CHANNEL\_UNBRIDGE（通道解除桥接）
* CHANNEL\_UNHOLD（通道解除保持）
* CHANNEL\_UNPARK（通道解除停泊）
* CHANNEL\_UUID（通道UUID）
* CLONE（克隆）
* CODEC（编解码器）
* COMMAND（命令）
* CONFERENCE\_DATA（会议数据）
* CONFERENCE\_DATA\_QUERY（会议数据查询）
* CUSTOM（自定义）
* DEL\_SCHEDULE（删除计划）
* DETECTED\_SPEECH（检测到的语音）
* DETECTED\_TONE（检测到的音调）
* DEVICE\_STATE（设备状态）
* DTMF（双音多频）
* EXE\_SCHEDULE（执行计划）
* FAILURE（失败）
* GENERAL（常规）
* HEARTBEAT（心跳）
* LOG（日志）
* MEDIA\_BUG\_START（启动媒体窃听）
* MEDIA\_BUG\_STOP（停止媒体窃听）
* MESSAGE（消息）
* MESSAGE\_QUERY（消息查询）
* MESSAGE\_WAITING（消息等待）
* MODULE\_LOAD（加载模块）
* MODULE\_UNLOAD（卸载模块）
* NAT
* NOTALK（禁言）
* NOTIFY（通知）
* NOTIFY\_IN（通知输入）
* PHONE\_FEATURE（电话功能）
* PHONE\_FEATURE\_SUBSCRIBE（电话功能订阅）
* PLAYBACK\_START（开始播放）
* PLAYBACK\_STOP（停止播放）
* PRESENCE\_IN（在场）
* PRESENCE\_OUT（离场）
* PRESENCE\_PROBE（在场探测）
* PRIVATE\_COMMAND（私有命令）
* PUBLISH（发布）
* QUEUE\_LEN（排队长度）
* RECORD\_START（开始录音）
* RECORD\_STOP（停止录音）
* RECV\_INFO（接收信息）
* RECV\_MESSAGE（接收消息）
* RECV\_RTCP\_MESSAGE（接收RTCP消息）
* RECYCLE（回收）
* RELOADXML（重新加载XML）
* REQUEST\_PARAMS（请求参数）
* RE\_SCHEDULE（重新计划）
* ROSTER（名单）
* SEND\_INFO（发送信息）
* SEND\_MESSAGE（发送消息）
* SESSION\_HEARTBEAT（会话心跳）
* SHUTDOWN（关闭）
* STARTUP（启动）
* SUBCLASS\_ANY（任意子类）
* TALK（对话）
* TRAP（陷阱）
* UNPUBLISH（取消发布）
