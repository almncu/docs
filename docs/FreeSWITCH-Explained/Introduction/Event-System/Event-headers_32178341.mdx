# 事件头部 

## 0\. 关于

列出了事件可能具有的可用头部。

## 1\. 事件字段/头部

类型描述：

- **core**: 由FreeSWITCH核心定义的事件头部，与事件类一起使用（即由FreeSWITCH核心定义的事件）。
- **custom**: 一些头部与事件子类一起使用（即为某些模块或服务专门创建的自定义事件）。
- **common**: 一些头部包含在每个事件中，无论其类别如何。请参阅[Events](./Events_32178330.mdx#0-about)中的**1.1 头部部分**。

### 使用Markdown格式进行写作

Markdown是一种轻量级的标记语言，可以使文本的格式化更加简单和直观。它广泛用于编写文档、写作博客和编辑文本文件。在Markdown中，使用特殊的字符和语法来增强文本的可读性和结构。

以下是一些常用的Markdown语法：

1. 标题：使用`#`符号定义标题的级别，一级标题使用一个`#`，二级标题使用两个`#`，以此类推。
2. 强调：使用`*`或`_`将文本包含起来，可以实现斜体或加粗的效果。
3. 列表：使用`-`或`*`创建无序列表，使用数字和`.`创建有序列表。
4. 链接：使用`[文本](链接)`创建链接，将文本替换为链接显示的文字，将链接替换为实际的URL。
5. 图片：使用`![替代文本](图片链接)`插入图片，替代文本是图片无法显示时所显示的文本，图片链接是实际的图片URL。
6. 引用：使用`>`在文本前面创建引用块，可以嵌套多个引用。
7. 代码块：使用```
   这是一段代码
   ```插入代码块，可以指定代码的语言类型。

Markdown语法简洁明了，易于理解和使用。它可以让你的写作变得简单而规范，而且可以轻松地输出为HTML格式。无论你是在写作文档、编写投稿稿件，还是撰写博客，Markdown都是一个非常好的选择。所以现在就去尝试一下Markdown，体验它带来的便捷吧！
