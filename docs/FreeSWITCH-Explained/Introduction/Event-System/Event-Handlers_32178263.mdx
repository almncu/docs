## 0\. 关于

在FreeSWITCH的上下文中，事件处理器是[模块](../../Modules/index.mdx#about)，提供一种与[事件系统](./index.mdx#0-about)进行交互的方式。使用了"交互"一词，是因为

* 一些仅将[系统事件](./Event-List_7143557.mdx#nat)转发到外部（可能是远程）应用程序  
（例如，[mod_amqp](../../Modules/mod_amqp_9635078.mdx#about)，[mod_event_zmq](../../Modules/mod_event_zmq_3966418.mdx#about)，`mod_cdr_`\*），
* 但其他的还提供了除订阅事件外的能力，可以通过外部应用程序注入事件，并控制FreeSWITCH实例  
（例如[mod_event_socket](../../Modules/mod_event_socket_1048924.mdx#11-acl)，[mod_erlang_event](../../Modules/mod_erlang_event_1048926.mdx#api-)等）。
