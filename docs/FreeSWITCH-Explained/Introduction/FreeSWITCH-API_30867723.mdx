# FreeSWITCH API

公开的FreeSWITCH API包含可以通过以下途径向FreeSWITCH发送的所有命令：

* 通过控制台[fs_cli](../Client-and-Developer-Interfaces/1048948.mdx#or)
* 通过[event socket interface](Glossary_13173966.mdx#event-socket)
* 通过脚本接口

可用命令的集合取决于加载的模块。您安装的命令集是每个模块注册的命令集的并集。

要查看可用API命令的列表，只需在CLI中输入`**help**`或`**show api**`。
