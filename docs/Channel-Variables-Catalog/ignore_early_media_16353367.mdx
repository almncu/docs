# ignore_early_media

由Ryan Harris创建，于2018年02月07日修改

## ignore_early_media

字符串类型，控制呼叫是否在早期媒体上返回，默认值为`false`。

**用法**

```xml
<action application="set" data="ignore_early_media=true"/>
```

您还可以在`bridge`应用的参数中为`ignore_early_media`指定一个值，使用`{ }`语法。（由于`ignore_early_media`是用于发起会话的全局变量，因此不能使用`[ ]`语法在每个程中指定它）：

```xml
<action application="bridge" data="{ignore_early_media=true}sofia/test-int/1001@somebox,sofia/test-int/1000@somehost"/>
```

将该值设置为`ring_ready`将与`ignore_early_media=true`的效果相同，但也会在捕捉到第一个SIP 183时向入站程发送SIP 180。

```xml
<action application="set" data="ignore_early_media=ring_ready"/>
```
