# fifo_consumer_wrapup_key

由 Ryan Harris创建，最后修改日期是在2018.02.07

## fifo_consumer_wrapup_key

dtmf数字 如果设置了这个变量并且消费者处于"等待"模式，[mod_fifo](../FreeSWITCH-Explained/Modules/mod_fifo_3966031.mdx#-add)将会在消费者按下`fifo_consumer_wrapup_key`或者超过[fifo_consumer_wrapup_time](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#fifo_consumer_wrapup_time)设置的超时时间后，使其重新进入fifo。
