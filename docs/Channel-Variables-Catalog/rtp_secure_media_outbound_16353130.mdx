# rtp_secure_media_outbound 

由 Ryan Harris 创建，最近修改于 2018.02.07

## rtp_secure_media_outbound

字符串

与 [rtp\_secure\_media](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#rtp_secure_media) 语法相同：`rtp_secure_media_outbound=<权限>[:<加密套件列表>]`

示例：

`rtp_secure_media_outbound=true:AEAD_AES_128_GCM_8`

`rtp_secure_media_outbound=optional:AEAD_AES_128_GCM_8`
