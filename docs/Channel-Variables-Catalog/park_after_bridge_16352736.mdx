## park_after_bridge

boolean 如果设置为 `true`，则在桥接完成后将呼叫 [park](../FreeSWITCH-Explained/Modules/mod-dptools/6586687.mdx#0-about)。在检查 [transfer\_after\_bridge](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#transfer_after_bridge) 和 [hangup\_after\_bridge](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#transfer_after_bridge) 之前进行检查。

**用法**

```xml
<action application="set" data="park_after_bridge=true"/>
<action application="bridge" data="sofia/gateway/myprovider/5551231234"/>
```
