# spandsp_dtmf_rx_reverse_twist 

由Ryan Harris创建，最后修改于2018.02.08

## spandsp_dtmf_rx_reverse_twist

integer 用于设置spandsp DTMF检测器的反转扭曲设置。默认情况下，反转扭曲设置为4 dB。可以安全地将此值增加到6或7，而不会显著增加对话干扰，以允许检测到超过此阈值的DTMF信号。在执行[spandsp\_start\_dtmf](../FreeSWITCH-Explained/Modules/mod_spandsp_6587021.mdx#dialplan-applications)之前设置此变量。

**使用方法**

```xml
<action application="set" data="spandsp_dtmf_rx_reverse_twist=6"/>
<action application="spandsp_start_dtmf" />
```
