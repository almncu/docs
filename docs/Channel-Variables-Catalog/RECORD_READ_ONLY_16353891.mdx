# RECORD_READ_ONLY

由Ryan Harris创建，在2019.03.28日由Joshua Young最后修改。

## RECORD_READ_ONLY

布尔值，用于标识是否只读取记录流。

**用法**

##### 从FreeSWITCH的角度看向终端

当Aleg在录音时，只有从通道中读取的传入音频会被写入文件

```xml
<action application="set" data="RECORD_READ_ONLY=true"/>
```

或者当Bleg在录音时，只有从通道中读取的传入音频会被写入文件

```xml
<action application="bridge" data="{RECORD_READ_ONLY=true}user/1000"/>
```
