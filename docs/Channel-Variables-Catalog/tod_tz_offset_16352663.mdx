## tod_tz_offset

整数型，设置用于[时间条件](../FreeSWITCH-Explained/Dialplan/Time-of-Day-and-Holiday-Routing_6586868.mdx#about)的呼叫的GMT偏移量。

![(warning)](/images/icons/emoticons/warning.svg)在进行比较之前，必须实际设置该变量，因此可以将其内联设置、转接或在用户目录中设置。

**用法**

在`conf/vars.xml`中全局设置偏移量：

```xml
<X-PRE-PROCESS cmd="set" data="tod_tz_offset=5"/>
```

```xml
<action application="set" data="tod_tz_offset=5"/>
```
