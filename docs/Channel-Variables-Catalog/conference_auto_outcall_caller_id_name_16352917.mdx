## conference_auto_outcall_caller_id_name

在通过 [conference\_set\_auto\_outcall](../FreeSWITCH-Explained/Modules/mod_conference_3965534.mdx#outbound-conference) 应用程序拨打终端加入会议时使用的呼叫者ID名称。

**用法**

```xml
<action application="set" data="conference_auto_outcall_caller_id_name=$${effective_caller_id_name}"/>
```
