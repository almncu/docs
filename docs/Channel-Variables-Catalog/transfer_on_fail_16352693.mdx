## transfer_on_fail

字符串。当由于特定原因（如未分配号码等）无法联系到被呼叫方时，允许您转移呼叫流。

![(warning)](/images/icons/emoticons/warning.svg) 您还必须设置 [failure_causes](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#failure_causes)

参数语法：`hangupcauses 目标拨号计划 上下文`

**使用方法**

```xml
<action application="set" data="failure_causes=UNALLOCATED_NUMBER"/>
<action application="set" data="transfer_on_fail=UNALLOCATED_NUMBER"/>
```

```xml
<action application="set" data="transfer_on_fail=1"/>
```
