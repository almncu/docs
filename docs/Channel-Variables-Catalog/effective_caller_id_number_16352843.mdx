## 有效主叫号码

字符串，设置有效主叫号码。该号码会自动传递给B-leg，但在主叫字符串中无效。换句话说，在呼叫桥接之前设置此号码，否则请使用[origination\_caller\_id\_number](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#origination_caller_id_number)。

**用法**

```xml
<action application="set" data="effective_caller_id_number=9185551212"/>
```
