## voicemail_domain

string类型。当[mod\_voicemail](../FreeSWITCH-Explained/Modules/mod_voicemail_6587070.mdx#--)发送语音邮件时，设置发件人电子邮件地址中的域名。

**用法**

来自`conf/directory/default/1010.xml`：

```xml
<include>
  <user id="1010" mailbox="1010">
    <params>
      <param name="password" value="password"/>
      <param name="vm-password" value="1010"/>
    </params>
    <variables>
      <variable name="voicemail_domain" value="speakblast.com"/>
    </variables>
  </user>
</include>
```
