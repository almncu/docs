## sip_codec_negotiation

`sip_codec_negotiation`字符串基本上是入站编解码器协商的通道变量等效物。

选项：

* `generous`
* `scrooge`
* `greedy`

在[74a0cfd1e101413a3941c41d04ee01d8df2ae418](https://freeswitch.org/stash/projects/FS/repos/freeswitch/commits/74a0cfd1e101413a3941c41d04ee01d8df2ae418)之前，`sip_codec_negotiation`将始终被sofia配置参数`sip-codec-negotiation`覆盖。从这个版本开始，编解码器协商可以在每次呼叫时更改。

**使用方法**

```xml
<action application="set" data="sip_codec_negotiation=greedy"/>
```
