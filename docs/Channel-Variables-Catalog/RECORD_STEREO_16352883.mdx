由Ryan Harris创建，由livem Chan在2019年07月15日修改

## RECORD_STEREO

将A腿和B腿流（即，呼叫者的录音在左声道，接收者的录音在右声道）分别记录到立体声文件的不同声道。

**用法**

```xml
<action application="set" data="RECORD_STEREO=true"/>
```
