# 当空闲时发送静默

由Ryan Harris创建，由John Boteler在2021年04月10日修改

## send_silence_when_idle

字符串 适用于只有一个通话腿的呼叫，比如终止在IVR菜单或语音信箱中的呼叫。

当大于 `0` 时，此变量告知FreeSWITCH在空闲时传输舒适噪声。此变量的值被设置为生成静默的除数。`400` 或 `1400` 是常见的设置值，但您可以尝试其他选择以选择最佳的值。

当为 `true` 时，FreeSWITCH将选择一个默认的舒适噪声值。

当为 `-1` 时，FreeSWITCH将在传输静默时不加入舒适噪声。

**用法**

```xml
<action application="set" data="send_silence_when_idle=400"/>
```
