## session_in_hangup_hook

布尔值，允许通道变量在通道执行的 [api_hangup_hook](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#api_hangup_hook) 中访问。查看 [Lua env object](../FreeSWITCH-Explained/Client-and-Developer-Interfaces/Lua-API-Reference/index.mdx#special-case-env-object) 以获取示例。

**用法**

```xml
<action application="set" data="session_in_hangup_hook=true"/>
```
