# call_timeout (通话超时)

由Ryan Harris创建，最后一次修改由John Boteler于2022年06月14日完成。

## call_timeout（已废弃）

整数。控制使用bridge应用程序时，呼叫B leg（被叫方）的响铃时间（以秒为单位）。超时时间设置在A leg（主叫方），适用于通道中发生的任何桥接。

如果您需要为没有A leg的呼叫设置超时，请使用[originate_timeout](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#originate_timeout)。

如果您需要在进行企业桥接/发起呼叫时设置超时，请使用[originate_timeout](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#originate_timeout)。

如果您需要针对每个leg（即每个目的地）设置超时时间，请使用[leg_timeout](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#leg_timeout)变量。

请注意，如果您没有使用`{`[ignore_early_media](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#ignore_early_media)`=true}`，一旦收到早期媒体信号，`call_timeout`将不再适用。

  
**默认值:** 60

**用法**

```xml
<action application="set" data="call_timeout=20"/>
```

如果要为group_call()列表指定呼叫超时，请使用以下格式：

```xml
<action application="bridge" data="{originate_timeout=24}${group_call(sales@$${domain})}"/>
```
