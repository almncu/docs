## transfer_after_bridge

这个变量可以控制在通话挂断时发生的情况。它可以与[mod_fifo](../FreeSWITCH-Explained/Modules/mod_fifo_3966031.mdx#-add)一起使用，以控制“坐席”，可能将其发送回坐席队列。在[park_after_bridge](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#park_after_bridge)和[hangup_after_bridge](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#transfer_after_bridge)之后，会进行检查。

![(info)](/images/icons/emoticons/information.svg) 在桥接被销毁和传送到指定的拨号计划之后，变量会被清除。

**使用方法**

```xml
<action application="set" data="transfer_after_bridge=1000"/>
```

![(info)](/images/icons/emoticons/information.svg) 注意使用冒号分隔符

```xml
<action application="set" data="transfer_after_bridge=1000:XML:default"/>
```
