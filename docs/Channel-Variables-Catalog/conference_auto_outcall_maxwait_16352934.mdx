# conference_auto_outcall_maxwait

创建者：Ryan Harris，最后修改时间2018.02.07

## conference_auto_outcall_maxwait

整数。最长等待时间，以秒为单位，用于发起[conference\_set\_auto\_outcall](../FreeSWITCH-Explained/Modules/mod_conference_3965534.mdx#outbound-conference)的通道等待成员加入会议的时间。

**用法**

```xml
<action application="set" data="conference_auto_outcall_maxwait=10"/>
```
