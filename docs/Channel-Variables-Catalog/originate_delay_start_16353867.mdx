## originate_delay_start

integer（整数） 可以指定呼叫发起前的等待时间，单位为毫秒。此变量可用于企业呼叫发起，而[leg\_delay\_start](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#leg_delay_start)变量无法使用。

**用法**

```xml
<action application="bridge" data="sofia/profile/dest1:_:{originate_delay_start=10000}sofia/profile/dest2:_:{originate_delay_start=15000}sofia/profile/dest3"/>
```

一个带有详细说明和时序的更复杂的示例（括号内为秒）:

```xml
<action application="bridge","users/1000:_:{originate_delay_start=8000}user/2302:_:{originate_delay_start=14000}sofia/gateway/flowroute/1231231234"/>
```

假设所有用户都只是让它响铃:

[00] - 用户 1000 响铃
[10] - 用户 2302 响铃
[15] - 用户 1231231234 响铃

假设用户 1000 在 2 秒后拒绝呼叫，其他用户继续响铃:

[00] - 用户 1000 响铃
[08] - 用户 2302 响铃
[14] - 用户 1231231234 响铃
