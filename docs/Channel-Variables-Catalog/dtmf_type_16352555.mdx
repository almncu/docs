# dtmf_type

由Ryan Harris创建，由Attila Gulyas在2020年10月19日进行了修改。

A、B、C、D音调

确保预期（并处理）DTMF音调`A`、`B`、`C`和`D`；尽管现在很少见，但仍有可能发送（例如，某人有一台带有额外ABCD列的旧设备）。请参阅[DTMF](../FreeSWITCH-Explained/Configuration/DTMF_9634268.mdx#5-other)了解更多信息。

## dtmf_type

字符串 对于inband DTMF，必须在拨号计划中使用[start\_dtmf](../FreeSWITCH-Explained/Modules/mod-dptools/6587132.mdx#0-about)。

**用法**

```xml
<action application="set" data="dtmf_type=info"/>
```

```xml
<action application="set" data="dtmf_type=rfc2833"/>
```

TODO [RFC 2833](https://tools.ietf.org/html/rfc2833) 已被 [RFC 4733](https://tools.ietf.org/html/rfc4733)所废弃。

```xml
<action application="set" data="dtmf_type=none"/>
```
