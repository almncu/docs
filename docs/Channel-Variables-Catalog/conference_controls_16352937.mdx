## conference_controls

字符串 设置此变量以指定在将呼叫方转入会议时使用哪个会议控制集。这使您可以为会议主持人设置一个控制集，为普通会议成员设置另一个控制集。例如，主持人的控制集可以包括静音或踢出人员的能力。

![(info)](/images/icons/emoticons/information.svg) 您必须创建所需的会议控制集。此外，如果未设置此项，则会议成员将使用默认的会议控制集。

**用法**

```xml
<action application="set" data="conference_controls=moderator"/>
```
