## conference_auto_outcall_caller_id_number

当通过[conference\_set\_auto\_outcall](../FreeSWITCH-Explained/Modules/mod_conference_3965534.mdx#outbound-conference)应用程序拨打端点加入会议时要使用的呼叫者ID号码。

**用法**

```xml
<action application="set" data="conference_auto_outcall_caller_id_number=${effective_caller_id_number}"/>
```
