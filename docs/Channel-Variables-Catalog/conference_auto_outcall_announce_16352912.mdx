## conference_auto_outcall_announce

字符串 文件名 ，用于播放给通过 [conference\_set\_auto\_outcall](../FreeSWITCH-Explained/Modules/mod_conference_3965534.mdx#outbound-conference) 应用程序加入会议的会议成员的音频消息。因为会议将向成员发起出站呼叫，所以通常会播放一个问候词，解释收件人将参加一个会议电话。

**使用方法**

```xml
<action application="set" data="conference_auto_outcall_announce=sounds/soundfile.wav"/>
```
