## conference_auto_outcall_profile

用于通过 [conference\_set\_auto\_outcall](../FreeSWITCH-Explained/Modules/mod_conference_3965534.mdx#outbound-conference) 应用程序加入会议的成员使用的会议配置文件。

**用法**

```xml
<action application="set" data="conference_auto_outcall_profile=default"/>
```
