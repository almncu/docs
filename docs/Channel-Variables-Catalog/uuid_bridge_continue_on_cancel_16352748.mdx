## uuid_bridge_continue_on_cancel

当设置为`true`时，如果系统遇到错误的B-leg时会在拨号计划中继续执行。默认值为`false`，因为这种行为可能不被推荐。在实施[Dialplan FollowMe](../FreeSWITCH-Explained/Dialplan/Dialplan-FollowMe_9634428.mdx#about)时，您可能会发现这个变量很有用。

**用法**

```xml
<action application="set" data="uuid_bridge_continue_on_cancel=true"/>
```
