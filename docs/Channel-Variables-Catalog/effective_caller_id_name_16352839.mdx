# 有效主叫者名称

由 Ryan Harris 创建，上次修改于 2018.02.07

## 有效主叫者名称

字符串 设置有效的主叫者名称。它会自动导出到 B-leg；然而，在发起呼叫的字符串中无效。换句话说，在调用桥接之前设置此项，否则请使用[发起者主叫者名称](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#origination_caller_id_name)。

Snom 370/820

如果您希望在本地Snom响铃时尽快显示LEG A的名称（如果可用），则必须按照描述设置`origination_caller_id_name`或`effective_caller_id_name`。否则，在LEG B的显示中，在响铃时会看到LEG A的号码，并在LEG B接听通话后切换到LEG A的名称。要删除它，请将其设置为`_undef_`。

**使用方法**

```xml
<action application="set" data="effective_caller_id_name=Bob Smith"/>
```
