# `bridge_answer_timeout`  桥接应答超时

由 Ryan Harris 创建，Jurijs Ivolga 在 2020.04.10 进行了最后修改。

## `bridge_answer_timeout`  桥接应答超时

整数，以秒表示，用于容忍处于早期媒体状态但未被应答的桥接时间（可以在任一一端设置）。自 build 15057 版本引入。

另请参阅：[早期媒体](../FreeSWITCH-Explained/Codecs-and-Media/Early-Media/index.mdx#about)

**用法**

```xml
<action application="set" data="bridge_answer_timeout=20"/>
```
