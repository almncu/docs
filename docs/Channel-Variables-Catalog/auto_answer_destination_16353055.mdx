## auto_answer_destination

在呼入呼叫时，远程一方可能会发送一个带有事件类型为 `talk` 的 `SIP NOTIFY` 消息，告知 Sofia 终端应答该呼叫。在这种情况下，变量 `auto_answer_destination` 被设置为通道的 [destination\_number](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#destination_number)，并且呼叫被转接到当前拨号计划和上下文中的扩展号码 `auto_answer`。
