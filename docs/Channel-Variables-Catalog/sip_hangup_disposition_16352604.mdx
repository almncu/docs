# sip_hangup_disposition

由Ryan Harris创建，上次修改时间为2018年02月07日

## sip_hangup_disposition

string这个变量包含了发送SIP BYE消息的人的值。以下是一些来自XML CDRs的示例：

```xml
<sip_hangup_disposition>send_bye</sip_hangup_disposition>
<sip_hangup_disposition>recv_bye</sip_hangup_disposition>
<sip_hangup_disposition>send_refuse</sip_hangup_disposition>
<sip_hangup_disposition>send_cancel</sip_hangup_disposition>
```

对这些值的解释在呼入和呼出电话时有所不同，因为FreeSWITCH在会话的不同端点：

| 值           | 呼入                                                | 呼出                                               |
| ------------ | --------------------------------------------------- | -------------------------------------------------- |
| send\_bye    | FS向呼叫方发送BYE消息（我们挂断了电话）           | FS向目标终端发送BYE消息（我们挂断了电话）         |
| recv\_bye    | FS从呼叫方接收到BYE消息（他们挂断了电话）           | FS从目标终端接收到BYE消息（他们挂断了电话）       |
| send\_refuse | FS拒绝了呼叫（如4xx或5xx错误）                      | 终端拒绝了呼叫（如4xx或5xx错误）                   |
| send\_cancel | _n/a_                                               | FS取消了呼叫（我们发送了CANCEL消息）              |
