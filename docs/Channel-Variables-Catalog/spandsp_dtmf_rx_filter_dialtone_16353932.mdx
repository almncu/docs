# spandsp_dtmf_rx_filter_dialtone 

由Ryan Harris创建，上次修改于2018年2月8日

## spandsp_dtmf_rx_filter_dialtone

boolean类型，设置spandsp DTMF检测器中的拨号音过滤参数。默认情况下，拨号音过滤是禁用的。在执行[spandsp\_start\_dtmf](../FreeSWITCH-Explained/Modules/mod_spandsp_6587021.mdx#dialplan-applications)之前，请设置此变量。

**用法**

```xml
<action application="set" data="spandsp_dtmf_rx_filter_dialtone=true"/>
<action application="spandsp_start_dtmf" />
```
