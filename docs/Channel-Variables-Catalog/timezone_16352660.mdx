由Ryan Harris创建，最后修改于2018年2月10日。

## 时区

字符串，用于设置此特定呼叫的时区。可以用于设置检查语音信箱的呼叫者的时区。该值以Linux时区格式表示（例如America/New\_York -- 可参考`/usr/share/zoneinfo/zone.tab`获取Linux时区的标准列表）。

![(warning)](/images/icons/emoticons/warning.svg) 在比较之前，**必须**设置此变量，所以请在行内设置、传递或在用户目录中设置。

另请参见：[时间和假日路由](../FreeSWITCH-Explained/Dialplan/Time-of-Day-and-Holiday-Routing_6586868.mdx#about)。

**用法**

您可以在`/conf/vars.xml`文件中为FreeSWITCH全局设置时区：

```xml
<X-PRE-PROCESS cmd="set" data="timezone=America/Toronto"/>
```

在拨号计划中指定时区：

```xml
<action application="set" data="timezone=Asia/Seoul"/>
```

```xml
<action application="set" data="timezone=GMT0"/>
```

```xml
<action application="set" data="timezone=America/New_York"/>
```

在目录中：

```xml
<param name="timezone" value="America/New_York"/>
```
