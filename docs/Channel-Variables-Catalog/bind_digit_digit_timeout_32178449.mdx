# bind_digit_digit_timeout 绑定按键超时

由Attila Gulyas于2019.12.25创建

整数

按键超时，单位为毫秒。这个设置用于设置在拨号过程中每个按键之间等待的时间。

默认值为1500毫秒。

在拨号计划扩展中，通常与[mod_dptools: play_and_get_digits](../FreeSWITCH-Explained/Modules/mod-dptools/6586933.mdx#about)结合使用。

请参阅[mod_dptools: bind_digit_action](../FreeSWITCH-Explained/Modules/mod-dptools/6586414.mdx#0-about)中的3.1节。

用法

**示例**

```xml
<!--等待按键之间的时间为10秒：-->
<execute application="set" data="bind_digit_digit_timeout=10000" />
```
