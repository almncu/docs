# hold_music

由Ryan Harris创建，在2021年8月13日由Todd Rossi进行了最后修改

## hold_music

string 单通道保持音乐。支持所有音频格式和音频流。`hold_music`变量也可以在`vars.xml`全局设置。

**用法**

```xml
<action application="set" data="hold_music=/sounds/holdmusic.wav" />
```

```xml
<action application="set" data="hold_music=silence" />
```

您还可以使用`phrase:`来表示使用短语而不是特定文件：

```xml
<action application="set" data="hold_music=phrase:my_hold_music" />
```

您还可以将您的`hold_music`设置为特殊值`indicate_hold`而不是音乐源，它将通过SDP将等候信号传递到远端：

```xml
<action application="set" data="hold_music=indicate_hold" />
```

对于多租户环境，如果您想为带有保持按钮的电话（如Polycom）拥有单独的保持音乐（MOH），并且该音乐使用具有保持的RE-INVITE且没有媒体IP地址（0.0.0.0），您可以像下面的示例一样覆盖sip配置文件参数中的保持音乐值：

```xml
<action application="bridge_export" data="hold_music=$${sounds_dir}/music/company-a.mp3"/>
```
