## sip_looped_call

当呼叫通过ACL以外的方式进行验证，并且当前的请求IP/端口与配置文件中的IP/端口匹配时，返回布尔值`true`（参见[src/mod/endpoints/mod\_sofia/sofia.c](https://freeswitch.org/stash/projects/FS/repos/freeswitch/browse/src/mod/endpoints/mod%5Fsofia/sofia.c)）。

**使用方法**

在`conf/dialplan/public.xml`中：

```xml
<condition field="${sip_looped_call}" expression="^true$">
```

```xml
<extension name="unloop">
  <condition field="${unroll_loops}" expression="^true$"/>
  <condition field="${sip_looped_call}" expression="^true$">
    <action application="deflect" data="${destination_number}"/>
  </condition>
</extension>
```
