## recovery_profile_name

string 它在内部使用，以便核心恢复引擎知道哪个配置文件名与呼叫相对应。对于核心来说，配置文件名只是呼叫的任意子类别，在 [mod_sofia](../FreeSWITCH-Explained/Modules/mod_sofia_1048707.mdx#about) 中，它指的是 SIP 配置文件名。请参阅 [邮件列表讨论](http://lists.freeswitch.org/pipermail/freeswitch-users/2012-December/090291.html)。
