# intercept_unbridged_only

由Ryan Harris创建，最后修改于2018年2月7日

## intercept_unbridged_only

boolean 如果设置为true，则只有在通道未与任何人桥接时，该通道才能进行[mod\_dptools:intercept](../FreeSWITCH-Explained/Modules/mod_dptools_1970333.mdx#-c)操作。

**默认值：** `false`

**用法**

```xml
<action application="set" data="intercept_unbridged_only=true"/>
<action application="intercept" data="myUUID"/>
```
