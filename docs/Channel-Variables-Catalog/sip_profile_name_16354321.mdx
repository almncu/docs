# sip_profile_name

由Ryan Harris创建，上次修改日期为2018.02.08

## sip_profile_name

字符串类型。如果您的呼叫是外呼的，那么这个变量将包含用于外呼通道（Leg B）的配置文件名称。

如果外呼通道是一个网关，那么这个变量将被设置为`gateway`，您需要查看`sip_gateway_name`以获取网关的名称。

如果外呼通道是另一个配置文件，则这个变量将被设置为该配置文件的名称。

![(warning)](/images/icons/emoticons/warning.svg)请注意，这与[sofia\_profile\_name](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#sofia_profile_name)不可混淆，后者是Leg A的配置文件名称。

**使用示例**

从内部配置文件上的用户呼叫，通过网关"ntl"进行路由：

```xml
sip_profile_name=gateway
sip_gateway_name=ntl
sofia_profile_name=internal
```

从内部配置文件上的用户呼叫，通过内部配置文件进行路由：

```xml
sip_profile_name=internal
sip_gateway_name=
sofia_profile_name=internal
```

从外部配置文件上的用户呼叫，通过内部配置文件进行路由：

```xml
sip_profile_name=internal
sip_gateway_name=
sofia_profile_name=external
```
