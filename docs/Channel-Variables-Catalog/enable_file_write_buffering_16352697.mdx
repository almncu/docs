# enable_file_write_buffering 

由Ryan Harris在2018.02.07最后修改

## enable_file_write_buffering

string 记录文件时启用文件缓冲，默认为`true`。如果没有设置，则缓冲区大小默认为`SWITCH_DEFAULT_FILE_BUFFER_LEN`，但可以通过指定字节数来覆盖（请参见下面的示例）。[邮件列表讨论](http://lists.freeswitch.org/pipermail/freeswitch-users/2012-April/082835.html)

**用法**

```xml
<action application="set" data="enable_file_write_buffering=false"/>
```

```xml
<action application="set" data="enable_file_write_buffering=true"/>
```

```xml
<action application="set" data="enable_file_write_buffering=65535"/>
```
