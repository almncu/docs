## group_confirm_file

string 这个变量与`group_confirm_key`一起使用。在`group_confirm_file`中，您可以指定在被叫方接听通话时播放的wav文件。有关更多示例，请参见[回答确认](../FreeSWITCH-Explained/Dialplan/Freeswitch-IVR-Originate_9634753.mdx#answer-confirmation)。

**用法**

```xml
<action application="set" data="group_confirm_file=/usr/local/freeswitch/sounds/take_call_question.wav" />
```
