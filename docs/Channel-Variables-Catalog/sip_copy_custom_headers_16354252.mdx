## sip_copy_custom_headers

布尔型。如果要将一些自定义的X-头部从B-leg传递到A-leg，则需将 `{sip_copy_custom_headers=true}` 添加到B-leg的拨号字符串中。

**用法**

将其设置为 `false`，以禁止将自定义的X-头部发送到SIP网关提供商：

```xml
<action application="set" data="sip_copy_custom_headers=false"/>
```
