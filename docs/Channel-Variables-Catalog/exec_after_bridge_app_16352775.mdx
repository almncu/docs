## exec_after_bridge_app

字符串，在桥接终止后执行应用程序命令。与[exec_after_bridge_arg](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#exec_after_bridge_arg)一起使用。相比之下，要在建立桥接后执行，请使用[execute_on_answer](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#execute_on_answer)。

**用法**

```xml
<action application="set" data="exec_after_bridge_app=transfer"/>
<action application="set" data="exec_after_bridge_arg=2102"/>
```
