由Ryan Harris创建，上次修改于2018年02月07日

## switch_r_sdp

string readonly 该变量保存当前呼叫/信道的远程SDP。

**使用方法**

![(warning)](/images/icons/emoticons/warning.svg) 在`"set">`后面不要加换行符，否则你会写出一个不同名称的变量，从而导致`switch_r_sdp`保持相同的值。

```xml
<action application="set"><![CDATA[switch_r_sdp=(此处填写SDP)
]]>
</action>
```
