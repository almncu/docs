## presence_data_cols（通话状态数据列）

字符串 您可以通过使用全局变量`presence_data_cols`来自定义FreeSWITCH内部通道表中的字段。

在`presence_data_cols`中，使用冒号`:`来分隔列名。

在`channels`表中，您必须首先定义字段：

`ALTER TABLE channels ADD COLUMN accountcode VARCHAR(256)`

**用法**

在`conf/vars.xml`文件中：

```xml
<X-PRE-PROCESS cmd="set" data="presence_data_cols=accountcode:domain_name"/>
```
