## sip_execute_on_image

`Execute_on_image`是指在收到T.38邀请时立即执行一个应用程序。这个变量与[execute_on_answer](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#execute_on_answer)类似。当收到T.38重邀但没有CNG音调时，或者你希望忽略音调，在收到T.38重邀时才反应的时候，你可以运行t38_gateway或rxfax等应用程序。
