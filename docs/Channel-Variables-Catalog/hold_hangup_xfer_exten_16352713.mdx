# hold_hangup_xfer_exten

由Ryan Harris创建，最后修改于2018年2月7日

## hold_hangup_xfer_exten

字符串 用于控制当被呼叫方（B方）处于呼叫桥接状态并且桥接结束时，主叫方（A方）会发生什么情况。如果在B方（默认）未设置，则A方会被挂断。如果在B方设置了该字符串，则A方会被转接到指定的分机号码，参考[transfer\_after\_bridge](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#transfer_after_bridge)。

**使用方法**

```xml
<action application="set" data="hold_hangup_xfer_exten=1000:XML:default"/>
```
