# spandsp_dtmf_rx_twist 

由Ryan Harris创作，最后修改日期为2018.02.08

## spandsp_dtmf_rx_twist

dtmf digit 设置了spandsp DTMF检测器的扭曲参数。默认情况下，扭曲设置为8 dB。在执行 [spandsp\_start\_dtmf](../FreeSWITCH-Explained/Modules/mod_spandsp_6587021.mdx#dialplan-applications) 前设置此变量。

**用法**

```xml
<action application="set" data="spandsp_dtmf_rx_twist=8"/>
<action application="spandsp_start_dtmf" />
```
