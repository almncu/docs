## origination_uuid

字符串 使用 `origination_uuid`，您可以指定发起呼叫的UUID。这样，您可以在呼叫被接通之前挂断呼叫，因为您知道UUID。只需记住，您需要使用`create_uuid`命令生成UUID，因为具有相同UUID的2个呼叫==不好！

![(info)](/images/icons/emoticons/information.svg) Bridge也使用原始语法，因此您还可以通过使用`{}`/`[]`语法并在那里指定`origination_uuid`，为由桥接产生的新通道预先分配UUID。

**用法**

```xml
originate [origination_uuid=....]sofia/<profile>/<extension>
```
