# bind_digit_input_timeout 绑定数字输入超时

由 Attila Gulyas 在 2019.12.25 创建

整型

整体超时时间，以毫秒为单位。它设置了等待整个数字序列输入完成的总时间。

默认值为 0 毫秒。

主要与相同拨号计划扩展中的 [mod\_dptools: play\_and\_get\_digits](../FreeSWITCH-Explained/Modules/mod-dptools/6586933.mdx#about) 结合使用时才有用。

请参阅 [mod\_dptools: bind\_digit\_action](../FreeSWITCH-Explained/Modules/mod-dptools/6586414.mdx#0-about) 中的 **3.1** 小节。

## 用法

**示例**

```xml
<execute application="set" data="bind_digit_input_timeout=10000" />
```
