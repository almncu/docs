 创建者 Ryan Harris，在 2018.02.07 修改

## conference_auto_outcall_flags

字符串类型。设置成员通过 [conference\_set\_auto\_outcall](../FreeSWITCH-Explained/Modules/mod_conference_3965534.mdx#outbound-conference) 应用程序加入会议的标志。

**用法**

```xml
<action application="set" data="conference_auto_outcall_flags=mute"/>
```
