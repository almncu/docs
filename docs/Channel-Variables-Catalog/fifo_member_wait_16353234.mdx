# fifo_member_wait 

Created by Ryan Harris, last modified on 2018.02.07

## fifo_member_wait

string 如果这个变量被设置为 `wait`，当呼叫者挂断电话时，消费者的通话将不会挂断。这是默认的行为。

如果设置为 `nowait`，当呼叫者挂断电话时，消费者的通话将会挂断。

**使用方法**

```xml
<action application="set" data="result=${fifo_member(add MyQueueName {fifo_member_wait=nowait}user/1001@VoiceNetwork.ca )"/>
```
