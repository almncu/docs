## group_confirm_cancel_timeout

整数 假如设置了，当通话接通后取消[leg_timeout](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#leg_timeout)。 [originate_timeout](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#originate_timeout) 仍然有效。

使用组确认时，通话经历三个阶段：

1. 通话响铃。
2. 通话已接通，等待被确认。
3. 通话已确认并桥接。

然而，如果执行以下操作

`<action application="set" data="group_confirm_cancel_timeout=1"/>`

那么超时将只适用于第一阶段。因此，一旦通过了第一阶段，`leg_timeout`计数器将停止。
