## sip_network_destination

string 适用于在 NAT 后注册的设备，其中 `Request-URI` 应将在注册请求期间绑定到 AOR 的联系方式包含在内，而请求本身应发送到 NAT “针孔”的公共 IP 和端口号。它不像 `;fs_path=` 或 [sip\_route\_uri](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#sip_route_uri) 选项那样，在请求中添加一个 `Route` 头字段。

**用法**

```xml
<action application="bridge" data="{sip_network_destination=sip:5551234567@66.243.109.243:10005}sofia/external/5551234567@172.16.110.45:5060"/>
```
