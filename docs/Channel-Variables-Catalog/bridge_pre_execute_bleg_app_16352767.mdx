## bridge_pre_execute_bleg_app

在将两个通道进行桥接之前，在B路上执行的命令或API。在使用事件套接字、CLI或XML-RPC发起呼叫时非常有用。

例如，可以使用脚本或mod\_http执行HTTP GET请求，以增加Snom电话的响铃音量，以便进行唤醒呼叫。

也可以将DTMF绑定到呼叫的B路上的应用程序，以便在呼叫转移时保持不变。

可以使用[bridge\_pre\_execute\_bleg\_data](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#bridge_pre_execute_bleg_data)将参数传递给应用程序。

![(warning)](/images/icons/emoticons/warning.svg)这将在呼叫设置完成之后但媒体（音频）桥接之前执行。

**用法**

```xml
<action application="set" data="bridge_pre_execute_bleg_app=bind_meta_app"/>
<action application="set" data="bridge_pre_execute_bleg_data=1 a s att_xfer::sofia/profile/destination"/>
```
