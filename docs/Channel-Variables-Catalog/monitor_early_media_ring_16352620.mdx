## monitor_early_media_ring

字符串，用于监视特定用户的来电铃声。每次听到铃声时，交换机都会为该路径的内部计数器增加一个计数。一旦计数器达到`monitor_early_media_ring_total`（或未设置此变量），则会发送早期媒体。

语法是一系列以`!`分隔的早期媒体条件，格式如下：

`condition_name:number_of_hits:tone_detect_frequencies`

| condition\_name           | 用户定义的错误条件的名称                                                    |
| ------------------------- | ----------------------------------------------------------------------------- |
| number\_of\_hits          | 必须听到该铃声的次数，才会认为它是失败的                                      |
| tone\_detect\_frequencies | 要监听的频率（用`+`而不是`,`分隔）。参见[tone\_detect](../FreeSWITCH-Explained/Modules/mod-dptools/6586957.mdx#about) |

![(warning)](/images/icons/emoticons/warning.svg)只有在将`ignore_early_media`设置为true时，此变量才起作用。

**用法**

```xml
<action application="bridge" data="{monitor_early_media_ring_total=3,monitor_early_media_ring=usring:1:440.0+480.0!ukring:2:400+450}sofia/gateway/yourgateway/1239@conference.freeswitch.org"/>
```
