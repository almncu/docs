# spandsp_dtmf_rx_threshold

由Ryan Harris创建，最后修改日期为2018.02.08

## spandsp_dtmf_rx_threshold

整数类型，设置spandsp DTMF检测器中的阈值参数。默认阈值设置为-42 dBm0。在执行[spandsp\_start\_dtmf](../FreeSWITCH-Explained/Modules/mod_spandsp_6587021.mdx#dialplan-applications)之前，请先设置此变量。

**用法**

```xml
<action application="set" data="spandsp_dtmf_rx_threshold=-42"/>
<action application="spandsp_start_dtmf" />
```
