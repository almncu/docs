## ep_codec_string

字符串`ep_codec_string`包含了A路的"endpoint"编解码字符串。这个编解码列表仅包含A路和FreeSWITCH双方都同意的编解码器。当SIP配置文件启用了入站延迟协商([inbound-late-negotiation](../FreeSWITCH-Explained/Modules/mod_sofia_1048707.mdx#about))时，才会设置这个变量。

![(info)](/images/icons/emoticons/information.svg)当列表中有多个编解码器时，优先顺序由FreeSWITCH而非A路决定。

**使用方法**

```xml
<action application="bridge" data="{absolute_codec_string=${ep_codec_string}}sofia/foo/bar@baz"/>
```
