## dtmf_verbose

boolean值，用于启用Spandsp DTMF检测器的详细日志记录。默认值为`false`。在执行[spandsp\_start\_dtmf](../FreeSWITCH-Explained/Modules/mod_spandsp_6587021.mdx#dialplan-applications)之前设置此变量。

**用法**

```xml
<action application="set" data="dtmf_verbose=true"/>
<action application="spandsp_start_dtmf" />
```
