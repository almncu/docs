## bypass_media

boolean类型 当设置为真时，原始端点的媒体(RTP)直接发送到目标端点，反之亦然。两个端点的信令(SIP)仍然经过FreeSWITCH，但媒体是点对点的。参见: [绕过媒体概述](../FreeSWITCH-Explained/Codecs-and-Media/Bypass-Media-Overview/index.mdx#faq)

**用法**

```xml
<action application="set" data="bypass_media=true"/>
```
