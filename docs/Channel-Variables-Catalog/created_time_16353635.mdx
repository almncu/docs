## created_time

字符串，包含通话开始时间（以微秒为单位）。

如果通话被转移，此处将包含整个通话创建的时间，而不是特定的转移时间。如果您需要转移创建的时间，请参阅[profile\_created\_time](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#profile_created_time)和[variable\_progress\_time](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#leg_progress_timeout)。
