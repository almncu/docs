## playback_delimiter

字符串

设置后，允许按指定的分隔符播放多个文件的连续音频。

在内部使用 [mod_dptools: file_string](../FreeSWITCH-Explained/Modules/mod-dptools/6586646.mdx#0-about)。

### 用法

将 `playback_delimiter` 设置如下：

```xml
<action application="set" data="playback_delimiter=&"/>
```

允许连续播放文件 `foo.wav` 和 `bar.wav`：

```xml
<action application="playback" data="foo.wav&bar.wav"/>
```
