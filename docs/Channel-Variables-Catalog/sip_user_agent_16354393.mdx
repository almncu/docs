## sip_user_agent

字符串，SIP头中的用户代理部分。

**用法**

如果您的请求头包含：

```xml
User-Agent: FreeSWITCH-mod_sofia/1.2.0-rc2+git~20120713T162602Z~0afd7318bd+unclean~20120713T184029Z
```

那么该字段的值将是：

```xml
FreeSWITCH-mod_sofia/1.2.0-rc2+git~20120713T162602Z~0afd7318bd+unclean~20120713T184029Z
```
