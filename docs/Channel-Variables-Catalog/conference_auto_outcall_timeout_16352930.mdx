## conference_auto_outcall_timeout

当通过[conference\_set\_auto\_outcall](../FreeSWITCH-Explained/Modules/mod_conference_3965534.mdx#outbound-conference)将会员加入会议时，使用的发起超时时间。

该变量控制呼出电话的响铃时间。因此，如果不忽略早期媒体，它很可能永远都不会有机会增加。

**用法**

```xml
<action application="set" data="conference_auto_outcall_timeout=60"/>
```
