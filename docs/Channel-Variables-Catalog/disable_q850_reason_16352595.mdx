# disable_q850_reason

由Ryan Harris创建，最后修改日期为2018年2月7日。

## disable_q850_reason

布尔值。当设置为true时，禁止在响应和BYE（终止呼叫）中发送Reason头部，包括Q.850原因码。请参阅[挂机原因](hangup_cause_16352598.mdx#hangup_cause)获取挂机原因及其Q.850代码的列表。此功能在2009年12月8日提交的修订版本15850中可用。

**使用方法**

```xml
<action application="set" data="disable_q850_reason=true"/>
```
