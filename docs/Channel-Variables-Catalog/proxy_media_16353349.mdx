## proxy_media

布尔类型的Proxy Media模式将Freeswitch设置为RTP流的"透明代理模式"。RTP流仍然经过freeswitch（不像bypass media模式），但由于freeswitch根本不解析数据包或以任何方式处理它们，因此对CPU的负担较轻，只是简单地转发它们。

另请参阅：[Proxy Media](proxy_media_16353349.mdx#proxy_media)

**用法**

![(info)](/images/icons/emoticons/information.svg) [延迟协商](../FreeSWITCH-Explained/Codecs-and-Media/Codec-Negotiation_2883752.mdx#about)在sip配置文件中必须启用，以便在拨号计划中使用此功能。

```xml
<action application="set" data="proxy_media=true"/>
```
