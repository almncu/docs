## hangup_cause_q850

这个字段是设置为呼叫A路的挂断原因的Q850数值代码（请注意，在呼叫结束之前它本身并没有太多意义）。通常情况下，它会从B路的挂断原因中获取。关于挂断原因的列表，请参阅 [挂断原因](../FreeSWITCH-Explained/Troubleshooting-Debugging/Hangup-Cause-Code-Table_3964945.mdx#about)。

**用法**

```xml
<action application="log" data="1 A-leg挂断原因Q850：${hangup_cause_q850}"/>
```
