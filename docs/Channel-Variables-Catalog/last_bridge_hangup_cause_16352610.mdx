由Ryan Harris创建，最近修改于2018.02.07

## last_bridge_hangup_cause

字符串型。此字段用于设置通话中最近一次桥接的B路段的挂断原因。要查看挂断原因的列表，请参见[挂断原因](../FreeSWITCH-Explained/Troubleshooting-Debugging/Hangup-Cause-Code-Table_3964945.mdx#about)。

**用法**

```xml
<action application="log" data="B路段挂断原因：${last_bridge_hangup_cause}"/>
```
