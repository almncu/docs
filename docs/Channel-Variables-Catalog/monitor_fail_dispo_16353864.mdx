## monitor_fail_dispo

string 这个变量可以用来在早期媒体失败时使用 [monitor_early_media_fail](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#monitor_early_media_fail) 或 [monitor_early_media_ring](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#monitor_early_media_ring) ，作为自定义的 [originate_disposition](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#originate_disposition) 的结果。如果没有设置这个变量，将会在 `originate_disposition` 中放置一个默认值，即 `monitor_early_media_fail` 或 `monitor_early_media_ring`。

**用法**

```xml
<action application="bridge" data="{ignore_early_media=true,monitor_fail_dispo=strange_bleep_attack,monitor_early_media_fail=user_busy:2:1234}sofia/dial/string"/>
```
