## sip_to_user

`To` SIP标头中的用户名部分。

**用法**

要进行手动设置，请使用：

```xml
<action application="set" data="sip_to_user=whatevah" />
```

然后此字段的值将为：

```xml
1000
```

作为选项，您可以进行以下呼叫：

```xml
bridge(sofia/external/333@biloxi.com^1234)
```

或者进行以下呼叫：

```xml
bridge(sofia/external/333^1234@biloxi.com)
```
