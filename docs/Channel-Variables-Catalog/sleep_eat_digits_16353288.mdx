# sleep_eat_digits

由Ryan Harris创建，修改日期为2018.02.07

## sleep_eat_digits

布尔值。当设置为`true`时，睡眠应用程序将消耗DTMF信号，例如防止呼叫者退出IVR。默认情况下，不会消耗DTMF信号。该变量在SVN修订版14102中添加。

![(warning)](/images/icons/emoticons/warning.svg) 这是默认行为的变更，因为睡眠应用程序以前会无条件地消耗DTMF信号。请确保将`sleep_eat_digits`设置为`true`，以保持先前的行为。

**用法**

```xml
<action application="set" data="sleep_eat_digits=true"/>
```
