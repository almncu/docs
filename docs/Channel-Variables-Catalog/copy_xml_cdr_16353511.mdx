## copy_xml_cdr

复制对方通话腿的XML CDR到当前通话腿的CDR中。例如，A通话腿的CDR将包含一个名为`b_leg_cdr`的变量，其内容是从B通话腿解码的URL编码的XML CDR数据。此变量必须在B通话腿上设置，可以在拨号字符串中使用`{copy_xml_cdr=true}`或者使用`export`代替`set`。

**用法**

```xml
<action application="bridge" data="{copy_xml_cdr=true}user/${dialed_extension}@${domain_name}"/>
```
