## sip_to_port

字符串类型，代表`To` SIP头部的端口部分。

**使用方法**

如果你的请求头部包含以下内容：

```xml
To: <sip:1000@server.example.com:5661>;tag=0e70ba56
```

那么该字段的值将为：

```xml
5661
```

然而，如果你的请求头部不包含端口信息，则该字段的值将是默认SIP端口（通常为5060）：

```xml
To: <sip:1000@server.example.com>;tag=0e70ba56
```
