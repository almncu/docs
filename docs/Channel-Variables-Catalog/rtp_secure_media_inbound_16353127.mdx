# rtp_secure_media_inbound 

由Ryan Harris创建，最后更新于2018年02月07日

## rtp_secure_media_inbound

字符串

与[rtp\_secure\_media](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#rtp_secure_media)相同的语法：`rtp_secure_media_inbound=<permission>[:<list of encryption suites>]`

示例：

`rtp_secure_media_inbound=true:AEAD_AES_256_GCM_8,AES_CM_256_HMAC_SHA1_80,AES_CM_256_HMAC_SHA1_32`

`rtp_secure_media_inbound=mandatory:AEAD_AES_256_GCM_8`
