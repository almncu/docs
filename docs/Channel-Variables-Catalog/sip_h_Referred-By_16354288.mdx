# sip_h\_Referred-By（被引用者）

由Ryan Harris创建，最新修改于2018.02.08

## sip_h\_Referred-By（被引用者）

字符串 这是SIP消息中的`Referred-By（被引用者）`行。它可以在转接到一个号码时设置，然后将其连接到其他地方。

**用法**

```xml
<action application="set" data="sip_h_referred-by=000@domain.com"/>
```

如果您想取消设置，您可以使用：

```xml
<action application="unset" data="sip_h_referred-by"/>
```
