# hangup_after_bridge 

由Ryan Harris创建，最后修改者是Attila Gulyas于2020.08.04

boolean 控制了当调用方（A方）处于[桥接状态](../FreeSWITCH-Explained/Introduction/Life-Cycle-of-a-Call_1048888.mdx#0-about)，并且被呼叫方（B方）挂断时，对调用方（A方）会发生什么。如果设置为`true`，则[dialplan](../FreeSWITCH-Explained/Dialplan/index.mdx#0-about)将停止处理，并且当B方结束通话后，A方也会被终止。如果设置为`false`（默认值），则B方结束通话后，dialplan将继续处理。这个检查在[park\_after\_bridge](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#park_after_bridge)和[transfer\_after\_bridge](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#transfer_after_bridge)之后进行。

默认值为`false`

用法

```xml
<action application="set" data="hangup_after_bridge=true"/>
```
