由 Ryan Harris 创建，上次修改于2018.02.07

## origination_cancel_key

DTMF数字与attended transfer功能一起使用。允许设置一个DTMF按键来取消att\_xfer并重新连接保持的呼叫。它还可以取消尚未建立的桥接（因此无法使用[bridge\_terminate\_key](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#bridge_terminate_key)终止）。在修订版本14650中实施。

**用法**

```xml
<action application="set" data="origination_cancel_key=#"/>
```
