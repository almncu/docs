# max_session_transfers 

 由Tomas Bajarunas创建，最后修改日期为2019.04.23

## max_session_transfers

整数型，用于将传输次数与`max_forwards`分开计数。

通常，在调用`transfer`应用程序时，`max_forwards`会递减。

当在通道中设置了`max_session_transfers`变量时，调用`transfer`应用程序时不会影响`max_forwards`，而是将递减`max_session_transfers`的值。

这可以用于在预计会有许多传输时，阻止呼叫挂断在较大的IVR应用程序中。
