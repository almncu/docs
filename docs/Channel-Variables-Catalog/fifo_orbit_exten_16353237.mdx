## fifo_orbit_exten

字符串

语法：`exten:timeout`

设置目标分机和超时时间，这将导致FreeSWITCH在指定的超时时间内将呼叫者通过FIFO进行处理，然后将呼叫者转接到目标分机。可以在将呼叫者放入FIFO之前进行设置。

**用法**

在执行`fifo`应用程序之前：

```xml
<action application="set" data="fifo_orbit_exten=MyFIFOVoicemail:60"/>
```

现在，在FIFO中等待60秒后，呼叫者将被转接到`MyFIFOVoicemail`。

```xml
<action application="fifo" data="MyFIFO in"/>
```
