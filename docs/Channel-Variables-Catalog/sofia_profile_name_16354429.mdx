## sofia_profile_name

string 呼叫起始的配置文件名称（Leg A）。

![(warning)](/images/icons/emoticons/warning.svg) 请注意，这与[sip\_profile\_name](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#voicemail_profile_name)不同。

**用法**

从外部配置文件的用户呼叫

```xml
sofia_profile_name=external
```

从内部配置文件的用户呼叫

```xml
sofia_profile_name=internal
```
