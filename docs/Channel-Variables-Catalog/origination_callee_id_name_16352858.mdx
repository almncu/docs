# originator_callee_id_name

由 Ryan Harris 创建于2018.02.07

## originator_callee_id_name

string 在呼入呼叫中设置，用于控制呼叫方电话显示屏上的呼叫者 ID 名称。还请查看 `ignore_diplay_updates`，该变量会影响这些变量的处理方式。

**使用方法**

![(info)](/images/icons/emoticons/information.svg) 如果`set`无效，请尝试使用`export`。

```xml
<action application="bridge" data="{originator_callee_id_name=Reginald}sofia/gateway/provider/<Reginald的手机号码>" />
```
