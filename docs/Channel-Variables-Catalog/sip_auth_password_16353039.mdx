## sip_auth_password

用于[mod\_sofia](../FreeSWITCH-Explained/Modules/mod_sofia_1048707.mdx#about)与[sip\_auth\_username](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#sip_auth_username)一起使用，以应答验证挑战而无需定义完整的网关。

**使用方法**

```xml
originate {sip_auth_username=<你的用户名>,sip_auth_password=<你的密码>}sofia/external/1xxxxxxx@12.34.56.78 &echo
```
