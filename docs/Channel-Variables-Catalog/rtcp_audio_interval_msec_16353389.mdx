# rtcp_audio_interval_msec 

由 Ryan Harris 创建，最后修改日期为 2018.02.07

## rtcp_audio_interval_msec

此整数频道变量用于设置每个RTCP SR数据包之间的时间间隔（以毫秒为单位）。

![(warning)](/images/icons/emoticons/warning.svg) 将[rtp\_timer\_name](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#rtp_timer_name)设置为`none`将会产生各种不期望的副作用，例如RTCP数据包未发送、RTCP自动调整失败等。
