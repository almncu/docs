## sip_auth_username

字符串，用于在没有定义完整网关的情况下回答`mod_sofia`的认证挑战。与`sip_auth_password`配合使用。同时也指示了成功注册到FreeSWITCH的SIP用户名的设备。

**用法**

```xml
originate {sip_auth_username=<your_user_name>,sip_auth_password=<your_password>}sofia/external/1xxxxxxx@12.34.56.78 &echo
```
