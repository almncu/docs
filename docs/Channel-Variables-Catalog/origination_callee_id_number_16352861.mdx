## origination_callee_id_number

字符串，在呼入路由中设置，用于控制呼叫者电话显示的来电号码。同时请参考[ignore\_diplay\_updates](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#ignore_display_updates)，以了解这些变量的处理方式。

**用法**

```xml
<action application="bridge" data="{origination_callee_id_name=Reginald,origination_callee_id_number=2332}sofia/gateway/provider/<Reginald的手机号>" />
```
