## default_language

字符串，用于控制在API调用中未明确指定语言时，[Say Phrase engine](../FreeSWITCH-Explained/Configuration/Speech-Phrase-Management_9634492.mdx#0-about) 将使用的默认语言。这使您可以通过在调用时仅更改单个变量来轻松支持多种语言。

**用法**

```xml
<action application="set" data="default_language=fr"/>
```
