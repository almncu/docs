## presence_id

字符串类型。将指示[mod\_sofia](../FreeSWITCH-Explained/Modules/mod_sofia_1048707.mdx#about)调用`PRESENCE_IN`事件，以断定已订阅实体的BLF灯状态。这只在通道存在的时候持续存在，因此不应用于持续状态（例如，正在忙碌的用户或登录到队列的坐席）。

**用法**

```xml
<action application="set" data="presence_id=${dialed_extension}@${domain_name}"/>
```
