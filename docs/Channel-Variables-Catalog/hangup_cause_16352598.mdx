## hangup_cause

string 此字段设置为呼叫的A路断开原因（需要注意，在通话结束之前它并没有太多意义）。通常情况下，如果存在B路，会采用B路的断开原因。有关断开原因的列表，请参见[断开原因](../FreeSWITCH-Explained/Troubleshooting-Debugging/Hangup-Cause-Code-Table_3964945.mdx#about)。

**用法**

```xml
<action application="log" data="1 A路断开原因：${hangup_cause}"/>
```
