# min_dup_digit_spacing_ms

由Ryan Harris创建，最后修改于2018年2月8日

## min_dup_digit_spacing_ms

整数。如果两个数字之间的间隔小于此时间，则忽略在段内DTMF信号的重复。也就是说，这是第一个数字的结束和重复数字的开始之间的最小间隔，需要检测到两个数字。默认值为`0`。在执行[spandsp\_start\_dtmf](../FreeSWITCH-Explained/Modules/mod_spandsp_6587021.mdx#dialplan-applications)之前，请设置此变量。

**使用方法**

```xml
<action application="set" data="min_dup_digit_spacing_ms=40"/>
<action application="spandsp_start_dtmf" />
```
