# origination_caller_id_number 

由Ryan Harris创建，上次修改于2018.02.07

## origination_caller_id_number

string 设置呼出方主叫号码（LEG A）。

如果您想在呼出呼叫上设置呼叫方号码，应在拨号字符串之前的`{}`括号中添加此内容。

**使用方法**

![(warning)](/images/icons/emoticons/warning.svg) 如果您想通过FXO网关中继传递来自PSTN呼叫的呼叫方号码，请将此变量注释掉。

```xml
<action application="set" data="origination_caller_id_number=9185551212"/>
```
