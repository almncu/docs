# outbound_redirect_fatal 

 由Ryan Harris创建，上次修改时间为2018年2月7日

## outbound_redirect_fatal

boolean 当同时呼叫多个终端时，302重定向可能导致所有终端停止响铃并且呼叫将遵循重定向。当设置了该通道变量时，发送302重定向的终端将从呼叫列表中删除，而其他终端将继续响铃。

**用法**

```xml
<action application="bridge" data="{outbound_redirect_fatal=true}${group_call(sales@$${domain})}"/>
```
