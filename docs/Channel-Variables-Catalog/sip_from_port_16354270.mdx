## `sip_from_port`

整数型数据，代表`From` SIP头部的端口部分。

**用法**

如果您的请求头部包含以下内容：

```xml
From: <sip:1000@server.example.com:5661>;tag=0e70ba56
```

那么此字段的值将为：

```xml
5661
```

然而，如果您的请求头部不包含端口信息，则该字段的值将为默认SIP端口（通常为5060）例如：

```xml
From: <sip:1000@server.example.com>;tag=0e70ba56
```
