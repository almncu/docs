## transfer_ringback转接响铃

string 当通话已经被接听，并且将转接到另一个终端时，会播放该音频。语法与[ringback](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#transfer_ringback)相同。

**用法**

```xml
<action application="set" data="transfer_ringback=${fr-ring}"/>
```
