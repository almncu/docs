## park_timeout

字符串 当设置后，停泊的电话在超时后将会断开连接。超时时间以秒为单位进行指定。如果没有设置`park_timeout`的值，那么停泊的电话将会一直保持或直到通过[uuid\_transfer](../FreeSWITCH-Explained/Modules/mod_commands_1966741.mdx#uuid_transfer)将其移除。

## **用法**

```xml
<action application="set" data="park_timeout=30"/>
<action application="park"/>
```

您还可以在`park_timeout`超时后，指定需要哪个[hangup\_cause](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#bridge_hangup_cause)来断开通道。

```xml
<action application="set" data="park_timeout=30:MEDIA_TIMEOUT"/>
```
