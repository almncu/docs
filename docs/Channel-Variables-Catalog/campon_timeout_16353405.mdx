## campon_timeout

整数型变量，控制在超时之前尝试每个桥接的时间长度。它的工作方式与 [call\_timeout](../FreeSWITCH-Explained/Dialplan/Variables-Master-List_16352570.mdx#leg_timeout) 完全相同，只适用于露营。

默认值：`10`

**使用方法**

```xml
<action application="set" data="campon=true"/>
<action application="set" data="campon_timeout=20"/>
<action application="bridge" data="sofia/gateway/myprovider/5551231234"/>
```
