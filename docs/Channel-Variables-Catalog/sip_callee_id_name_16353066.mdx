# sip_callee_id_name 

由Ryan Harris创建，并于2018.02.07最后修改

## sip_callee_id_name

字符串 已弃用 在被叫端使用，用于控制呼叫方号码在呼叫电话的显示上。

**使用方法**

![(info)](/images/icons/emoticons/information.svg) 如果你发现 `set` 不起作用，尝试使用 `export`。

```xml
<action application="set" data="sip_callee_id_name=Reginald" />
<action application="set" data="sip_callee_id_number=2332" />
<action application="bridge" data="sofia/gateway/provider/<Reginald的手机号码>" />
```
