由Ryan Harris创建，最后修改于2018年02月08日

## DISPLACE_HANGUP_ON_ERROR

布尔值 当设置为`true`时，此通道变量将在尝试[mod\_commands#uuid\_displace](../FreeSWITCH-Explained/Modules/mod_commands_1966741.mdx#uuid_displace)呼叫时出现错误时导致呼叫挂断。默认值为`false`。2013年03月21日之前的默认行为是挂断。

**用法**

```xml
<action application="set" data="DISPLACE_HANGUP_ON_ERROR=true"/>
```
