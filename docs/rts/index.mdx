## 什么是RTS？

RTS是一个实时解决方案，也是一个实时SIP服务器。包含两个部分：rtss（RTS Server）和 rts （RTS客户端）。

- RTS 是一个FreeSWITCH 衍生版。
- 有稳定的发行版。
- 开源。
- 最大程度兼容FreeSWITCH。
- 有自己的Bug修复和扩展。
